## 4. ★가격영향 = 0 (zero) 입증

| **COMP_PRINT_DIGITAL_S1** | 디지털인쇄비(내지단면) | `["proc_cd", "plt_siz_cd", "print_opt_cd", "min_qty", "proc_grp:..."]` | ❌ (plt_siz_cd=국4절) |
| **COMP_PRINT_DIGITAL_S2** | 디지털인쇄비(내지양면) | `["proc_cd", "plt_siz_cd", "print_opt_cd", "min_qty", "proc_grp:..."]` | ❌ |

- ∴ **삭제 SIZ_000170(A5) → active SIZ_000007(A5) 교체의 가격영향 = 0.** 절가는 국4절 출력판수 기반이고, siz_cd는 UI 선택·임포지션 판수환산(`fn_calc_pansu(국4절, 작업siz)`) 입력일 뿐 단가 격자축이 아님.

- (margin/작업siz 차이는 판수환산에 영향 가능하나 — SIZ_000007 작업 150×212 ≈ 현 마스터 작업 150×214로 동일 4-up 국4절 판수[판걸이=4]. impos_yn은 6종 모두 N으로 동일.)

## 5. 광역 dedup 결함 flag (→ huni-basedata-dedup·여기서 마스터 수정 안 함)

삭제 SIZ_000170(A5)를 **del_yn=N으로 여전히 등록 중인 상품 20개** (라이브 실측):

```
           072 하드커버책자·077 레더 하드커버책자·082 하드커버 링책자
스티커(8): PRD_000052/053/054 반칼자유형·058 반칼원형·059 반칼정사각·
           060 반칼직사각·061 반칼띠지 (※스티커는 A5≠내지·별 용도 — 별건 검토)

           176 먼슬리플래너·177 스프링노트
```

- ★**삭제된 siz를 20개 상품이 활성 참조** = round-22/basedata-dedup 사이즈 표시중복 정리의 잔여 결함. 본 분석은 내지 승격 범위(책자 내지)만 정본 교체를 권고하고, **나머지(스티커·전단지 등)의 SIZ_000170 잔재는 huni-basedata-dedup 하네스에 광역 결함으로 라우팅**(여기서 siz 마스터/타 상품 수정 금지 — 범위 경계).
- 책자류 7개는 내지 승격(072→077→082)과 동시에 본체 A5 사이즈를 SIZ_000007로 정렬하면 잔재가 함께 해소되나, **본체 dims 변경은 별 승인**(inner-promotion-design.md §3.2 copy 정책과 충돌 주의 — 본 권고는 "내지 반제품에는 정본 siz를 등록"까지).

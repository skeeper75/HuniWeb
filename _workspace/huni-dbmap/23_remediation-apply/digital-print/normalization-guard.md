## 3. 오염 방지 — 무엇을 하지 않았나 (round-13 교훈 반영)

| 오염 패턴 | 본 정정이 회피한 방식 |
|-----------|----------------------|
| 색/형상/사이즈를 자재로 오적재 (MAT_TYPE.08~10 오염) | 형상 커팅은 자재 아닌 **공정(완칼)** + 형상=옵션으로 분리(축③) — 자재 테이블 미오염 |

| 신규 카테고리 코드 남발 | 정상 노드 273/274/275/283 **재사용**(search-before-mint), 신규 mint 0 |

| 적재처 없는 값 억지 삽입 | 형상 세부값·PP케이스는 적재처 부재로 **주석 보류**(컨펌) — NULL 강제/엉뚱한 칸 삽입 안 함 |

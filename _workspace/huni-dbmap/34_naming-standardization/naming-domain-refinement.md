### 0.3 후니 고유 상품·명칭 (타사에 없거나 후니 특유 — 우선 채택)
- **떡메모지** — 후니 핵심 상품(메모지 패드). 표준어 = **떡메모지**(레거시 카테고리명 그대로). RP 등 타사는 "메모지/노트패드"로 다르게 부름 → 후니 용어 우선.
- **합판도무송(합판)** — 합판(여러 도안 1판) + 도무송(완칼) 결합. 후니/국내 인쇄 고유. 형상=원형/정사각/직사각.
- **큐방** — 현수막 금속 아일렛 링의 후니/국내 현장 용어("QBANG"). RP도 `QBG_DFT`로 동일 개념. "구방/구멍"(1차 정리안 잠정) ❌ → **큐방** 확정.
- **오리지널박명함 / 트윙클** — 후니 박명함 고유 라인업·박종류명.
- **상상디자인** — 후니 디자인상품(에디터 템플릿) 카테고리 고유명.
- **전자액자** — 후니 인쇄상품 카테고리(디지털 액자).

> **★핵심 정정 3건(후니 실무 고유 용어):**
> - `QBANG` = **큐방**(금속 아일렛 링·국내 현장 용어·rpmeta QBG_DFT 일치) — "구방/구멍" 폐기.
> - `CUTEDGE` = **열재단**(라이브 proc PROC_000084/000096 "열재단" 실재) — "재단마감" 폐기.
> - `BONGSEW` = **봉미싱**(현수막 봉제 미싱) — note 권위가 "봉미싱"(라이브 proc는 "봉제"지만 현수막 add-on은 "봉미싱"이 후니 표기).

> **★빈 더미 comp 발견(컨펌):** `COMP_POPT_BNR_GAKMOK_STR_900_4`(base·접미사 없음)·`COMP_POSTEROPT_BANNER_MESH_PROC_OPT`(PROC_OPT)는 **가격행 0행**(use_yn=Y지만 단가 없음). `_GT`/`_LE`만 실제 가격행 보유. → PROC_OPT·GAKMOK base는 **미사용 잔재 가능성**(use_yn=N 검토 대상·인간 승인). PROC_OPT 한글명은 가격행 부재로 확정 불가 → 잔재면 정리, 활성이면 가격행 등록 필요.

### ★컨펌-2: "비" 부착 정책 (해소 — 도메인 일관 규칙)
- **가공·공정 단독 요소비** → "비" 부착(proc 권위 일관): 오시비·미싱비·귀돌이비·별색인쇄비·디지털인쇄비·코팅비·제본비·접지비·타공비·박비·형압비. → `오시 2줄`→**`오시비 2줄`**, `미싱 3줄`→**`미싱비 3줄`**.
- **가변데이터(텍스트/이미지)** → 후니/Figma 디자인시스템 v4.1.0 통용이 "가변텍스트"·"가변이미지"(비 미부착). → **현행 유지**(통용 우선). 다만 가격구성요소 화면 일관성을 원하면 "가변텍스트비"도 허용(선택·정책 결정).
- **add-on(완제 추가옵션)** → "추가가격" 접미(요소비 아님). → `{후니 옵션명} 추가가격`(예: `끈(4개) 추가가격`·`큐방(4개) 추가가격`).

### ★컨펌-3: `모서리 비` → `귀돌이비` (해소 — proc 권위)

## 3. 정책 결정 필요 (인간 승인 — 추정 0)

1. **PROC_OPT·GAKMOK base 빈 더미 comp 2건**: 가격행 0행·use_yn=Y. 미사용 잔재면 use_yn=N, 활성이면 가격행 등록 + 한글명 확정 필요. (현재 한글명 확정 불가 = 가격행 부재)
2. **use_yn=N 레거시 comp(별색 9·포스터 6)에 `[레거시]` 표기 여부**: use_yn=N이면 화면 미노출이라 정리 불요 가능. 표시 정책 결정.
3. **가변텍스트/이미지 "비" 부착**: 통용(미부착) 유지 vs 화면 일관(부착). 권장=현행 유지.
4. **기초코드 테스트 잔재 3건**(TEST/TESTTEST/TESTTESTTEST_TYPE): use_yn=N 또는 삭제.

- 코드노출 0: §component-naming-cleanup-v2 치환 후 `[COMP_*]` 잔존 0.
- rpmeta naming 유입 0: 후니 comp_nm에 RP 코드(QBG/LUM/CDL 등) 미혼입.

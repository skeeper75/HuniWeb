## 0. 한 줄 요약

합판도무송스티커 = **3개 논리 블록**(원형·정사각·직사각)을 수직 스택. 각 블록 = **밴드 단가표**(제작수량 5구간 × 형상사이즈 N × 소재 2축 복합). 형상사이즈 = `siz_cd`(라이브 37종 전부 등록 — BLOCKED siz **0**). 소재 = **2축 복합셀**(코팅 3종 + 데드롱 3종 = 개별 6소재). 단가 거동 = 장당가 체감 → **합가형(.02)** 의심(§3·decomposition §2). 라이브에 가격사슬 이미 적재(**소재 2종 collapse·prc_typ_cd=.01**) — round-16이 6소재 분해·합가형 정정 대상.

## 2. 매트릭스 셀 좌표 구조 (B1 원형 예시)

```
행2(병합 헤더, 사이즈) │ B2:C2 원형10mm │ D2:E2 원형15mm │ ... │ V2:W2 원형60mm  (11사이즈, 각 2컬럼)
행3(소재 2축)          │ B3 비코팅/무광/유광 │ C3 유포/투명데드롱/은데드롱 │ (사이즈마다 반복)
A열(수량 min_qty)      │ A4=1000 A5=2000 A6=3000 A7=4000 A8=5000
셀                     │ (수량 r, 사이즈 c, 소재축 m) → 단가
```

- **소재 2축 = 사이즈마다 2컬럼**: 홀수컬럼(B/D/F…) = `비코팅/무광코팅/유광코팅`(코팅축 3소재), 짝수컬럼(C/E/G…) = `유포/투명데드롱/은데드롱`(데드롱축 3소재).
- 셀 = 해당 (수량, 사이즈, 소재축) 조합의 **가격**.
- **셀 카운트(블록별)**: B1=11×2×5=**110** · B2=12×2×5=**120** · B3=14×2×5=**140** → **합 370셀**(= 라이브 370행과 일치, collapse 상태).

# 06 — 레이어 B: 상품 귀속 (prd_cd → cat_cd)

round-24 2단계 · dbm-category-mapper · **DB 미적재**.
**[round-24 검증 보정 반영]** D-1 PK dedupe·main_cat_yn 단일성 · D-2 del='Y' 가드 · D-3 partial 과매칭 철회 · D-4 명명변형 재분류 · D-5 카운트 정정.
산출 데이터: `product-cat.csv` (junction 적재 행 **248** + 미적재 노드예약 **11**) · 빌드 `scripts/build_product_cat.py`.

## 0. 산출 CSV 스키마 (junction 적재 명세)

`product-cat.csv` = `t_prd_product_categories(prd_cd, cat_cd, main_cat_yn)` 적재 명세. 컬럼:
`prd_cd · prd_nm · map_category · status · cat_cd · cat_nm · main_cat_yn · match_level · live_opt_n · live_priced · multi_class · note`.
- **main_cat_yn='Y'** = 대표 카테고리(prd_cd당 정확히 1행). **'N'** = 별칭 추가 노출.
- prd_cd 공백 행 = ❌미출시(노드 예약만·junction 적재 대상 아님).

## 1. 귀속 규칙

1. 각 MAP 상품 → 목표 cat_cd. 상품명이 **활성(del='N') L2/L3 노드**와 정확히 일치하면 그 노드, 아니면 **L1직속**.
2. **D-2 가드**: del='Y' 노드는 귀속 target 후보에서 제외(논리삭제 노드 귀속 = 조회 차단). 봉투제작(CAT_000066)이 이 케이스 → PRD_000050은 L1 CAT_000003 직속.
3. **별칭(다중분류)**: alias-dict — 본체 prd_cd를 추가 카테고리에 main_cat_yn='N'으로 노출. **중복 상품 생성 금지**. D-1: (prd_cd,cat_cd) 동일 별칭 중복(J28/J40 PRD_000157) dedupe → 19→18.
4. **D-1 PK 무결성**: (prd_cd,cat_cd) 중복 0 · prd_cd당 main='Y' 정확히 1행.
5. **출시상태 게이팅**: ✅GREEN 우선·🟡YELLOW 옵션보완 후·❌RED 보류(노드만 예약).

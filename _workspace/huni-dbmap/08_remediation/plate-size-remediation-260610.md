## 3. 트랙 C — 신규 전지 siz DDL 제안 (search-before-mint 통과)

기존 라이브 t_siz로 무손실 표현 불가 입증된 2종만 신규 제안(코드행 추가·테이블 무변경):

```sql

INSERT INTO t_siz_sizes (siz_cd, siz_nm, work_width, work_height, cut_width, cut_height, impos_yn, ...)
VALUES ('SIZ_PROPOSE_330x470','330x470', 330, 470, NULL, NULL, 'Y', ...);
-- 투명 소재 전지 (3상품 참조)
```

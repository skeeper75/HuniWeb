## 2. 속성별 정규화 결과 요약 (시트 단위)

### 2.1 사이즈 (size)
- 모든 시트 E컬럼. 실사·아크릴은 추가 F(가로)/G(세로) 비규격 입력.
- 한 상품당 다수 사이즈 세로 리스트. 예 `프리미엄엽서`=[73 x 98 mm, 98 x 98 mm, 100 x 150 mm, 135 x 135 mm, 95 x 210 mm, 110 x 170 mm, 148 x 210 mm].
- **고유 409종** 중 **133종이 비치수(모호, 4장 참조)**.

### 2.2 자재 (material)
- 단일자재(디지털/스티커/캘린더/실사/아크릴) vs 내지+표지 2축(책자/포토북/문구) vs 선택형(굿즈파우치 O컬럼).
- **고유 110종**. `*별도설정`(24회)=가격연동 플레이스홀더(실값 아님, 노트).

### 2.3 인쇄옵션 (print_option)
- **고유 21종**. 최빈 `단면`(83)/`양면`(50)/`배면양면`(11)/`화이트인쇄(단면)`(10).
- 별색 5컬럼(화이트/클리어/핑크/금색/은색)이 인쇄옵션 위치에 있으나 round-2 결정상 별색=공정(PROC_000007) → **이중성 분기 검증 필요**.

### 2.4 공정 (process)
- **명칭 고유 301종**(가격형 하위값 33종은 별도 분리). 코팅·박·제본·모서리·캘린더가공 등.
- 일부 값이 명칭 아닌 수치/안내(`*가격표참고`, `10 / 12 / 14 /16`) → 컬럼 의미 혼재, 모호 분류.

### 2.5 공정택일그룹 (process_select_group)
- 엑셀에 명시 컬럼 없음. **도출 규칙**: 공정의 각 옵션-컬럼 = 1 택일그룹(SEL_TYPE.01 단일 가정).
- DB `t_prd_product_process_excl_groups` 대조 시 "엑셀 구조 추론 vs DB 실제 그룹/sel_typ" 명시 대조 필요(추론값 — 검증 주의).

### 2.6 판형사이즈 (plate_size)
- 출력용지규격(`316x467` 등). 59상품. 권위 마스터=가격표 `판걸이수`.G(전지).

### 2.7 묶음수 (bundle_qty)
- 27상품. 조각수(아크릴/스티커)·장수(캘린더)·건수(디지털)·페이지묶음(문구/포토북 `4(8P) / 8(16P)...`).
- 값 형태 매우 비균일(`8조각`, `*최대40조각`, `12(24P) / 16(32P)`) → 코드化 곤란, 원천 보존+노트.

### 2.8 페이지룰 (page_rule)
- 16상품(책자·포토북·문구). 최소/최대/증가 3수치. DB `t_prd_product_page_rules` 대조용.

### 2.9 추가상품 (addl_product)
- 31상품. 봉투류 등 명칭+추가가격. **고유 64종**.

## 3. 마스터급 코드체계 고유집합 (Phase 3 마스터 정합용)

엑셀 등장 고유값 집합 + DB 마스터 대조 결과(정규화-매치 probe, **저장은 원천값**).

### 3.1 사이즈 마스터
- 엑셀 고유 **409종** vs DB `t_siz`(`ref-sizes.csv`) **497종**.
- 정규화 probe(공백·`mm` 제거 후 매치): **225/409 매치**. 나머지 184종 미스매치(표기차 + 비치수옵션).
- 브리지: 가격표 `판걸이수`.A(`73 x 98 mm`)↔.B(`73x98`)↔DB `siz_nm`.

### 3.2 자재 마스터
- 엑셀 고유 **110종** vs DB `t_mat`(`ref-materials.csv`) **333종(데이터)**.
- 정확매치: **74/110**. 미스매치 36종(`투명아크릴 3mm`, `아트250+무광코팅` 복합, `*별도설정` 등).
- 브리지: 가격표 `출력소재(IMPORT)`.C(종이명)↔DB `mat_nm`.

### 3.3 공정 마스터
- 엑셀 공정명 고유 **301종**(+가격형 33). vs DB `t_proc`(ref-product-processes는 매핑테이블, proc 마스터 별도 확인 요).
- 명칭 형태 혼재(enum + 안내문 + 수치) → 마스터 후보 정제 필요.

### 3.4 인쇄옵션 / 별색
- 인쇄옵션 enum **21종** — 소규모, 마스터화 용이. 단 별색은 공정 도메인과 중복(분기).

> **고유집합 규모 요약: 사이즈 409 · 자재 110 · 공정 301(명칭) · 인쇄옵션 21 · 추가상품 64 · 묶음수 23(distinct).**

## 5. 검증(DB 대조) 시 주의점

1. **JOIN KEY = `prd_nm`** (MES ITEM_CD 일부만, 신뢰 금지).
2. **사이즈 표기차**: 엑셀 `73 x 98 mm` ↔ DB `73x98`. `판걸이수` 브리지로 정규화 후 대조(매치 55%만 직접).
3. **세로 리스트 = 옵션 다중성**: 한 상품 size 7종 등. DB에서는 `t_prd_product_sizes` 다행으로 펼쳐짐 → 건수 대조.
4. **공정택일그룹/별색은 추론·이중성** → 단정 대조 금지, 분기 명시.
5. **상품악세사리**: 자재/인쇄/공정 전부 원천 부재 — DB에 값 있으면 그쪽이 추가, 없으면 정합.

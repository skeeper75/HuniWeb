# 후니 webadmin 실무진 표준 용어 사전 (라이브 권위) [v2 정교화]

> **산출자:** dbm-price-formula-auditor(v1) · dbm-domain-researcher(v2) · round-34 · 2026-06-18
> **목적:** 가격구성요소(comp_nm)·가격공식(frm_nm)·전 admin 페이지의 명칭을 실무진이 빠르게 식별하는 **표준 한글 용어**로 통일하기 위한 권위 사전.
> **v2 권위 순서(HARD·사용자 재정의):**
> - ① **후니 레거시 사이트** huniprinting.com·buysangsang.com(실제 상품명·후니 고유 상품 — 최우선)
> - ② 라이브 DB: `t_proc_processes`(proc_nm) · `t_cod_base_codes`(cod_nm) · `t_prc_component_prices.note`(후니 실제 종류/옵션명) · `t_prc_price_formulas`(frm_nm)
> - ③ huni-rpmeta(개념 흡수·교차검증만·naming/codes 유입 0)
> - ④ 국내외 인쇄표준(보조)
> **임의 창작 금지** — 모든 표준어는 위 권위에 근거. 미지는 가설+출처+컨펌.
> **v2 핵심:** 후니 고유 상품명 확정(떡메모지·합판도무송·큐방·오리지널박명함·상상디자인·전자액자) + add-on 용어 라이브 note 권위로 컨펌 3건 해소. 상세 §naming-domain-refinement.md.

## 0. 핵심 원칙

1. **comp_nm = 화면에서 빠르게 식별하는 한글 표준어.** 코드(`[COMP_xxx]`) 노출 금지.
2. **용어 권위 = proc_nm·cod_nm·frm_nm·기존 정리된 comp_nm.** 같은 요소는 페이지·comp·공식 간 같은 용어.
3. **note = 어떤 종류/값이 묶였는지·차원축·골든 — 빠른 확인용 일관 양식.**
4. **변별자(소재·면·종류)는 괄호로 부가.** 예: `별색인쇄비 금색(단면)`, `명함 완제품가(소재·면)`. 코드가 아니라 **사람이 읽는 변별자**.

## 3. 가격공식 frm_nm (권위 = 이미 정리됨·코드 노출 0)

48공식 모두 frm_nm에 코드 노출 없음. frm_nm 패턴이 comp_nm 정리의 모범:
- 합산형: `디지털인쇄 원자합산형A 엽서·상품권·슬로건` / `제본 합산형(제본비 구성요소)` / `접지 합산형(오시+접지 후가공 구성요소)`
- 완제품 단가형: `포스터사인 소재/사이즈/수량별 완제품가(포함항목 통가격)` / `명함 면/소재/수량별 단가(용지포함)`
- 실사: `일반현수막 완제품가(면적/규격 단가)`

> **frm_nm 점검 결과: 48/48 코드 노출 0 — 정리 불필요(GO).** 단 일부 note 빈값 3건(PRF_COROTTO_ACRYL·PRF_STK_PACK·PRF_STK_TATTOO)은 note 보강 권장(§component-naming-cleanup 참조).

## 4. comp_nm 표준 명명 패턴 (이미 정리된 사례 → 전수 적용 규칙)

라이브에서 **코드 노출 없이 잘 정리된 comp_nm**을 패턴 권위로 추출:

| 패턴 | 모범 사례(라이브 실재) | 규칙 |
|---|---|---|
| **요소비 + 변별자(괄호)** | `별색인쇄비 금색(단면)`, `유광코팅비`, `오시 2줄`, `가변텍스트 2개` | `{표준어}{변별자}`·코드 없음 |
| **실사 완제품가 결합본** | `실사 완제품가 (아트프린트포스터·접착방수포스터·…)` | `실사 완제품가 ({소재 한글나열})`·note에 [단독]/[동형결합]·골든 |
| **봉투/린넨 등** | `봉투제작 완제품가`, `린넨 마감가공비` | `{상품/요소} {표준어}` |

→ **나머지 102개(코드 노출분)도 이 패턴으로 통일.** §component-naming-cleanup.md 전수 정리표 참조.

## 0. 문제 (사용자 지적)

초판 `sticker-import.xlsx`는 가격표 헤더 3그룹을 3 mat_cd로 묶었다:
- `유포/비코팅/미색` → MAT_000153 (1개) ❌ 비코팅·미색 누락
- `무광코팅/유광코팅` → MAT_000155 (1개) ❌ 유광 누락
- `투명/홀로그램` → MAT_000170 (1개) ❌ **투명데드롱(합판도무송 전용)으로 오매핑** + 홀로그램 누락

→ 엔진이 소재로 매칭(`evaluate_price`)하려면 **각 소재가 개별 mat_cd 행**이어야 한다. 같은 가격이라도 소재는 별개.

## 2. 🔴 코팅 처리 결정 (BATCH-3 / Q-ST-A CONFLICT — 미해소)

`무광코팅·유광코팅`을 어느 축에:

| 안 | 처리 | 그릇 영향 | 근거 |
|----|------|----------|------|
| **(a) 자재 variant** | 무광/유광코팅스티커 = 개별 mat_cd(155/156) | 소재 7종(점착지) | round-11 product-bom §44("코팅된 점착지 완제 자재"·C23 빈값) |
| **(b) 공정 전환** | 비코팅(084) 자재 + 코팅 = `proc_cd`(PROC_000013 공정) | 소재 5종 + proc_cd 차원 | round-13 Q9★ 실무진("코팅=공정")·round-15 Wave-0 BATCH-3 |

- **(a)면**: 소재 7행(유포·비코팅·미색·무광코팅·유광코팅·투명·홀로그램). 가격=소재별(비코팅 6000 vs 코팅 7000).
- **(b)면**: 소재 5행(유포·비코팅·미색·투명·홀로그램) + 코팅 공정 proc_cd 차원. 비코팅 base 6000 + 코팅 공정 +1000 = 7000 합산. 가격모델 재작업.

> round-15 Wave-0 확정은 "출력소재=자재 / 입히는 코팅=공정"(b 방향)이나, round-11/가격표 3컬럼 구조는 (a). **전 가격표 횡단 영향**(코팅이 여러 시트에 등장) → 사용자 결정 필요.

## 3. 정리된 그릇 구조 (코팅 결정에 따라)

### (a) 자재 variant 채택 시 — 소재 7종
```
component_prices: comp_cd=COMP_STK_PRINT,
  mat_cd ∈ {153 유포, 084 비코팅, 242 미색, 155 무광코팅, 156 유광코팅, 162 투명, 163 홀로그램},
  siz_cd, min_qty, unit_price, (proc_cd=NULL)
→ B01: 6사이즈(매핑 4) × 7소재 × 36수량
```
- 가격: `유포/비코팅/미색`열 6000 → 유포·비코팅·미색 3행 각 6000 / `무광/유광`열 7000 → 무광·유광 2행 각 7000 / `투명/홀로그램`열 7000 → 투명·홀로그램 2행 각 7000.

### (b) 공정 전환 시 — 소재 5종 + proc_cd
```
component_prices(소재): mat_cd ∈ {153,084,242,162,163}, unit_price=비코팅 base가
component_prices(코팅공정): proc_cd=PROC_000013, unit_price=+코팅 추가가(7000−6000=1000)
formula_components: COMP_STK_PRINT(소재) + COMP_STK_COAT(코팅, addtn_yn=Y 합산)
```

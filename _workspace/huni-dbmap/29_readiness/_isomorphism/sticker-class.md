## 1. 동형 클래스 정의 (토대 §4·§5)

### 1-1. 공식유형 = MATRIX (상품자체 포함가)

스티커는 가격표 `스티커` 시트가 권위 — 상품자체 시트(토대 §4 "상품자체 포함가" 8종 중 하나). 구성:

- **포함가 구조**: 단가 셀 = `종이 + 인쇄 + 커팅` 통가격(T3 `종이+인쇄+커팅` 명시) = **소재+출력+가공 묶음 포함가**. 옵션(코팅 등)은 소재 variant로 포함가에 흡수, 별도 추가가 가공은 없음(아크릴 후가공 BUNDLE과 다름).
- **MATRIX 축**: B01 = `[수량(min_qty)] × [사이즈 6] × [소재그룹 3]` **3D 매트릭스**. B02~B04 = `[수량] × [사이즈]` 2D. B05 타투·B06 스티커팩 = 세트/번들 단가.
- **단가형(.01)**: 가격표 수량↑→장당단가↓ 거동(124x186 유포 1→6000·3→5900·100→5200). `단가 × 주문수량`. (B05/B06만 합가형 .02 = round-16 식별.)

→ 면적매트릭스형(아크릴 `[가로][세로]` 좌표 룩업)과 **다른 MATRIX** — 스티커는 **사이즈×소재 이산 격자 + 수량 구간**(off-grid ceiling 없음·이산 사이즈). 토대 §5 분류상 "MATRIX형 = 상품자체 포함가 다축"으로 별 유형. 파이프라인 4번째 공식유형 입증.

### 1-2. 옵션 구성 동형 (계산공식집초안·라이브 CPQ)

스티커 16상품의 옵션 골격 = **종이(소재 택1) + 인쇄(도수) + 커팅(반칼/완칼) + [투명군: 화이트별색]**. 라이브 CPQ 실측(052/053/055/066)이 이 골격을 확증:
- 종이 = `OPT_REF_DIM.03`(mat_cd + USAGE.07) 택1
- 인쇄 = `OPT_REF_DIM.06`(도수=단면) 택1
- 커팅 = `OPT_REF_DIM.04`(PROC_000054 반칼 공정) 택1
- 화이트별색(투명/홀로 군) = `OPT_REF_DIM.04`(PROC_000008 화이트 underbase) 택1, 화이트없음 분기 포함

→ 같은 골격 = 동형. 사이즈·소재·커팅방식만 상품별로 차감/치환되는 부분집합.

### 3-2. 대표 = 052(주축) + 053(보조)

- **052 반칼 자유형 스티커 = 주축 대표**: 토대 §6-1 "옵션·구성요소가 가장 완전하게 반영된 superset"을 **소재축**에서 충족 — 소재 option_items 5종(유포 153·비코팅 084·무광코팅 155·유광코팅 156·미색 242)으로 MATRIX 소재축 전체를 노출. MATRIX형의 핵심 = 소재×사이즈 격자이므로 소재 superset이 결정적.
- **053 = 보조 대표**: 052에 없는 **화이트별색 그룹**(투명소재 underbase 분기)을 보유 → 투명/홀로군(054·056) 부분집합의 superset 항. 052+053 합집합이 스티커 MATRIX 전 옵션 골격.
- 부분집합 전파: 054(투명 소재 차감+화이트)·055~057(소재1 차감)·058~062(규격 형상 GAP·소재5 동일)은 대표 매핑에서 "해당 항 빼기/형상 size 치환"으로 안전 전파.

> ★ 일반화 오류 회피: 엽서=프리미엄엽서·아크릴=키링처럼 상품군마다 대표가 다르다. 스티커는 **소재 MATRIX가 superset인 052**가 대표(052는 화이트별색은 없으나 소재축이 MATRIX 핵심). 화이트별색은 053 보조로 보완.

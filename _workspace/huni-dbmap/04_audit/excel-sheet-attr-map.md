## 1. 상품마스터 워크북 구조 (주 원천)

13개 시트: `계산공식집초안`, `MAP`, 그리고 상품 데이터 11개 시트.
9속성은 전부 **상품 데이터 11개 시트**에 있다(`계산공식집초안`=가격공식, `MAP`=참조표는 9속성 직접 보유 아님).

### 1.1 공통 레이아웃 (HARD — 파싱 핵심)

- **2행 헤더**: row1 = 속성 그룹명(예 `종이(필수)`), row2 = 하위 컬럼명(예 `최소`/`최대`/`증가`). 데이터는 **row3부터**.
- **상품 블록 구조**: 각 상품(`상품명` D컬럼)은 **여러 행에 걸친 블록**을 차지한다. 한 상품의 옵션값들은 각 컬럼에 **세로 리스트(ragged column)**로 쌓인다. 빈 `상품명` 행이 계속되면 같은 상품의 연속(continuation), 다음 비어있지않은 `상품명`이 나오면 새 상품 시작.
  - 예: `디지털인쇄` 시트 `프리미엄엽서`(r3) → 사이즈 7종(r3~r9), 인쇄 2종(단면/양면), 후가공 모서리 2종(직각/둥근)이 각각 세로로 쌓임.
- **공통 backbone 컬럼**: A=`구분`, B=`ID`, C=`MES ITEM_CD`(전부 NULL일 수 있음 — JOIN KEY는 `prd_nm`), D=`상품명`, E=`사이즈(필수)`.
- **제작수량 backbone**: 모든 시트가 `제작수량(필수)` 그룹(최소/최대/증가)을 가짐 → `excel-qty-range.csv`로 별도 보존.

### 2.1 매핑 주의 (HARD 노트)

- **공정택일그룹(process_select_group)**: 엑셀에 "택일그룹"이라는 명시 컬럼은 **없다**. 도출 규칙 = 공정(4번)의 **각 옵션-컬럼이 하나의 택일그룹**(SEL_TYPE). 예 `코팅`/`커팅`/`접지`는 각각 단일선택 그룹. DB `t_prd_product_process_excl_groups.sel_typ_cd`와 대조 시 이 도출을 검증해야 함. **엑셀 원천에서 직접 추출 불가 → 컬럼 구조에서 추론(노트로 표기)**.
- **묶음수 vs 사이즈 혼입(직전 SIZE_NAME_NOISE 맥락 인지)**: 본 파싱은 **엑셀 원천값 그대로**(정정본 아님). `장수`/`조각수`/`건수`는 묶음수 컬럼에서 추출. 단, 사이즈명 안에 장수가 혼입된 케이스는 여기서 보정하지 않고 size 원천값 그대로 둠(노트).
- **자재(material) 다컬럼**: 책자/포토북/문구는 내지+표지 2종 자재. 굿즈파우치는 `선택(옵션)`(O)을 자재로 매핑(파우치 색상/소재 선택). 모호 → 노트.
- **인쇄옵션 별색**: 디지털인쇄/스티커는 별색 5컬럼(화이트/클리어/핑크/금색/은색)을 인쇄옵션에 포함. 단 code-values.md상 별색은 **공정(PROC_000007)**으로도 해석됨(round-2 결정). 즉 별색은 인쇄옵션 컬럼에 있으나 DB상 공정일 수 있음 → **대조 시 분기 검증**(노트).
- **판형사이즈(plate_size)**: 상품마스터 E~L의 `출력용지규격`이 원천이나, 권위 마스터는 가격표 `판걸이수` 시트(아래 3장).
- **상품악세사리**: 9속성 중 사이즈·수량만 보유. 나머지 7속성 전부 빈값(원천 자체가 그러함).

## 4. 구조 노트 / 검증 시 주의점

1. **세로 리스트 파싱 정확성**: 블록 경계(다음 `상품명` 비어있지않은 행)를 잘못 잡으면 옆 상품 옵션을 흡수. 본 파서는 D컬럼 비어있지않은 행을 앵커로 사용 검증함.
2. **사이즈 컬럼(E) 의미 오버로드**: E=`사이즈(필수)`지만 폰케이스 모델명·색상·접지유형·"사용자입력"·팩수 등 **비치수 옵션**이 다수. DB siz 매핑 불가 항목 다수 예상 → `excel-attrs-normalized.md` 모호 분류 참조.
3. **MES ITEM_CD 신뢰 금지**: 일부만 채워짐. JOIN KEY = `prd_nm` (railway-db-access 메모와 정합).
4. **별색 인쇄 vs 공정 이중성**: round-2 결정(별색=PROC_000007 공정)과 본 시트의 인쇄옵션 컬럼 배치가 상충 가능 → 대조 시 분기.
5. **공정택일그룹은 추론값**: 엑셀 직접 컬럼 없음. DB excl_groups 대조 시 "엑셀 구조 추론 vs DB 실제"를 명시 대조.

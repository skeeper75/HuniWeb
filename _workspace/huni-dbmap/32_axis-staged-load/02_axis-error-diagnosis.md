## 3. 6축 staged 교정·적재 단계

**순서 = 기초코드 → 사이즈 → 도수 → 자재 → 공정 → 카테고리**(사용자 지정). 단 **FK 위상·논리삭제 마지막 원칙**상 실제 실행 시 마스터→연결→논리삭제로 재배열됨을 각 단계에 명시. 각 단계 = 멱등 UPSERT + 롤백전용 DRY-RUN **절차 명세**(실행 아님).

### [HARD] 전 단계 공통 가드 (P-TRUNCATE)
- 모든 교정 SQL은 멱등 `INSERT … ON CONFLICT DO UPDATE`(키=cod_cd/siz_cd/.../prd_cd+cat_cd)·`upd_dt` 변경로그.
- **load_master `--all` 재실행 금지 정책 또는 교정 컬럼 보존 로직** 없이는 교정 무효(TRUNCATE 소멸). round-22 1순위 선결.
- DRY-RUN = 단일 트랜잭션 BEGIN…ROLLBACK, 제약위반0·COMMIT0·2-pass delta 0 실증.

### 단계 1 — ① 기초코드 (마스터 선적재)
- **입력:** USAGE 본체/부속 코드 신설 여부(BATCH-9)·OPT_REF_DIM param(BATCH-13) 결정.
- **교정 액션:** 신규 코드그룹/코드 `INSERT … ON CONFLICT`(cod_cd). use_yn=N 1행 정체 확인.
- **검증 포인트:** 코드값이 하류 축 enum 참조와 일치(MAT_TYPE 12·USAGE 7 등). FK 무결성.
- **차단 분리:** 신설 코드는 **선적재 제안**(직접 INSERT 금지·인간 승인). 미결정은 BLOCKED.

### 단계 2 — ② 사이즈
- **입력:** size↔option 경계 결정(BATCH-6)·평면화 묶음 분리 대상.
- **교정 액션:** (a)치수 siz 신규(형상/용량 유입분)`INSERT … ON CONFLICT`(siz_cd) (b)siz_nm 평면화 정정 UPDATE.
- **검증 포인트:** **가격사슬 의존**(component_prices siz 차원) — 삭제·재키 전 사슬 영향 확인(기계삭제 금지). impos_yn 보존.
- **차단 분리:** size↔option 경계 미결정분 BLOCKED(BATCH-6). option화는 단계 6-CPQ 이후.

### 단계 3 — ③ 도수
- **입력:** ④에서 재귀속될 잉크색(만년스탬프/볼펜색).
- **교정 액션:** 도수축 자체 변경 거의 없음. 잉크색→도수/옵션 판정만(④ 단계와 연동).
- **검증 포인트:** 5행 유지·별색이 ⑤공정에 남아있는지(도수축 유입 금지).
- **차단 분리:** 잉크색 도수 vs 옵션 판정은 ④ BATCH-2에 종속.

### 단계 4 — ④ 자재 (최대 교정·핵심)
- **입력:** BATCH-2(색/부속 분리)·BATCH-4(레더·이미 부분완료)·추가-A(코팅 분해) 결정.
- **교정 액션(의사결정 트리 §2 적용):**
  1. mat_typ UPDATE(점착지 .01→.11·잔존 레더 .06 확인) — round-10.
  2. 비소재 자재행(색/형상/용량/구수/인쇄면)은 **정확 축 적재 검증 후** use_yn='N' 논리삭제.
  3. 코팅 평면화 자재명 정정 + ⑤공정 연결.
- **검증 포인트:** 자재행 논리삭제 **전** 색→option·형상→siz 적재 완료(먼저 삭제 금지). t_prd_product_materials 연결 영향.
- **차단 분리:** 색→option은 CPQ(단계 6) 선행 의존 → 자재행 논리삭제는 **단계 6 이후**. mat_typ UPDATE만 단계 4에서.

### 단계 5 — ⑤ 공정
- **입력:** BATCH-13(신설 공정)·Q9(코팅=공정)·BATCH-2 부속→공정.
- **교정 액션:** (a)신규 공정 mint(미싱제본·보드마운팅·삼각대거치 — 선적재 제안) (b)상품-공정 연결 INSERT(봉제/에폭시/맥세이프 등 자식0 공정) (c)mand_proc_yn 정정(PUR N→Y).
- **검증 포인트:** 신설 전 search-before-mint(기존 유사 공정 부재 확인). 부모공정 FK(upr_proc_cd).
- **차단 분리:** 신설 공정 미결정 BLOCKED(BATCH-13). 코팅 스티커 CONFLICT(BATCH-3).

### 단계 6 — ⑥ 카테고리 + CPQ 옵션(BATCH-6) + 논리삭제 일괄
- **입력:** BATCH-1(카테고리 재연결)·BATCH-6(CPQ option_items).
- **교정 액션:**
  1. 113상품 cat_cd UPDATE 의미매칭 정상노드 재연결(round-10·기계순 금지).
  2. CPQ option_groups→options→option_items 적재(④ 색·② 폰기종 유입분·round-6).
  3. **재연결·재적재 검증 후** 빈 고아 노드 14개 + 오염 자재행 + page 잡음 use_yn='N' 논리삭제(항상 마지막).
- **검증 포인트:** 정상노드 실재(search-before-mint)·fn_chk_opt_item_ref 무결성·CAT_000297(상품0) 선행 안전.
- **차단 분리:** 명함/상품권/단품형 정상노드 미확인분 BLOCKED(BATCH-1 Q).

### FK 적용순서(실행 시 재배열)
```
[0] 신규 마스터 mint(①코드·⑤공정·②siz) — 선적재 제안
[1] 마스터 UPDATE(④ mat_typ·①코드값)
[2] ⑥ 카테고리 재연결(cat_cd UPDATE)
[3] ⑤ 상품-공정 연결 INSERT
[4] CPQ option_items(④색·②폰기종 유입)
[5] 가격(범위 외·round-2)
[6] 논리삭제 일괄(고아노드·오염자재·page) — 검증 후 마지막
```

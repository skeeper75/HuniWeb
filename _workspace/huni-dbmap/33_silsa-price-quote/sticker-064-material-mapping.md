## 0. 핵심 5줄 (결론 먼저)

1. **064 소재 = B01 5소재(유포153·비코084·미색242·무광155·유광156) verbatim — 투명 없음.** 상품마스터·라이브 동일 실측. 사용자 규칙대로 매핑하면 **B01 col1(유포/비코/미색 6000·무광/유광 7000)** 단가 적용 대상(투명 col3 미해당).
2. **064 = 반칼 확정(사용자 규칙 + 마스터 verbatim).** 마스터 주문방법=`싱글(자유형)`·형상컬럼 X=`반칼(자유형)`·"(완칼)" 미명시 → **반칼 = B01 블록 귀속 정당**. 소재로 B01 가격열도 확정됨.
3. **🔴 그러나 064 사이즈(소형 50x70~94x94)가 B01 6사이즈에 부재 = 가격행 매핑 불가 → 정직 BLOCKED 유지.** B01 가격열 = A5/124x186·A4·A3·90x190·100x148·90x110뿐. 064의 소형 사이즈(50x70·70x50·50x94·94x50·65x65·80x80·94x94)는 **B01 단가표에 행 자체가 없음**. 판걸이수 시트는 이를 별 그룹 "**소형반칼스티커**"로 분류(반칼스티커와 구분)하나 = 임포지션뿐·단가 아님.
4. **★사용자 규칙은 "블록"을 확정(B01)·소재는 "가격열"을 확정 — 단 "사이즈 행"이 B01에 없음.** 반칼/소재 둘 다 맞아도 **B01 매트릭스에 064 소형 사이즈 단가 셀이 부재**하면 가격 산출 불가. 추측으로 소형에 B01 규격가(6000) 붙이면 오청구(돈 크리티컬·금지). → 소형반칼 전용 단가 출처 컨펌 필요.
5. **반칼/완칼 재분류 점검 = 오매핑 0(전건 정합).** 주문방법 컬럼이 사용자 규칙과 일치: 완칼=낱장055·낱장투명056·대형057(B02~B04)·나머지 반칼(B01)·타투/팩/합판 별. 058~063은 이미 B01 바인딩 COMMIT(정합). 064만 미바인딩(사이즈 BLOCKED).

### 2.3 판정 — **🔴 정직 BLOCKED (사이즈 행 부재)**
- 064는 반칼(블록=B01)·소재 매칭(가격열 확정)까지 풀렸으나 **사이즈 행(소형)이 B01 매트릭스에 없음** → 가격 셀 특정 불가.
- 판걸이수 시트가 064 사이즈를 "**소형반칼스티커**"(반칼스티커와 별 그룹)로 분류 = 소형은 별 임포지션·아마 별 단가 체계인데 **그 단가표가 가격표에 부재**.
- **돈 크리티컬: 소형(50x70)에 B01 규격가(A5 6000)를 추측 적용 금지** — 소형은 한 판에 더 많이 앉음(판걸이 32 vs A5 4)·단가 다를 개연 높음. 근거 없는 매핑은 오청구.
- → **소형반칼 단가 출처 컨펌 필요**(Q-064-1).

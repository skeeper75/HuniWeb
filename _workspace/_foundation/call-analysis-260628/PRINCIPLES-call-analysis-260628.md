## D1. 가격 계산 모델 = 3종 (단가/합가/고정가산)
- **단가형(.01)** = 장당단가 × 수량 (수량↑→장당↓·곱하기). 엔진 `pricing.py:193 .01=unit×qty`.
- **합가형(.02)** = 구간별 **합산 총액**이 표에 기재. 중간수량(300매)=**장당 환산 후 재곱**(2만/200×300). 엔진 `.02=unit÷min_qty×qty`.
  - ★미결: '장당환산 후 재곱'이 후니 레거시와 일치하는지 라이브 검증 필요. 합가형이 실존 모델인지 vs 사실은 고정가산형만 필요한지 판별.
- **제3형 고정가산형(가변텍스트형)** = 수량 무관 **구간 금액 단순 가산**(1~99매 15,000 고정·100매부터 상향). **곱하기 아님**(30매×15,000=450,000 폭증=오류). 엔진 `.03 고정=그대로`에 해당하나 '구간 가산' 의미 별도.
- **누적/구간(타투·스티커)** = 단가표에 구간별 **누적 금액 직접 기입**(1,2,4,6,9…)·별도 알고리즘 불필요.
- ★[연결] band-total `.01 ×qty` 과대청구 버그(`FINDING-bandtotal-x-qty-overcharge.md`)가 이 D1 미정의의 실증. .02 교정=합가형 보간.

## D2. 필수 공정 자동 가격 반영
- 공정에 **필수공정여부 플래그 이미 구현됨**(통화 확인). 필수=Y → 시뮬레이터 **무조건 자동 가산**(디폴트).
- 패키지 내재 공정(코팅명함의 코팅 등)=선택가능 공정에서 **제외**+패키지 단가에 포함.
- ★미결: 상품별 필수공정 지정 운영기준(상품마스터 빨강=필수/주황=옵션 색규칙 연결). 인쇄종류 누락 상품(프리미엄엽서) 데이터 보정.

## D3. 자재/소재 분류
- **자재 = 메인(주)자재 = 인쇄 매체로만 한정**(종이·아크릴). 부자재(고리·볼체인·구슬줄)=자재에서 제외→추가상품/옵션.
- **기본여부=Y 자재 → 시뮬레이터 자동선택**(시뮬레이터에 디폴트선택 기능 부재→기본여부 플래그로 대체). 서 대표 작업.
- 위젯: 데이터 1건=디폴트·다건=지정디폴트(명함 종이 기본 아트지).
- ★주의: '면지'=소재 아닌 **용도** → 자재로 직접 넣지 말 것(D6 책자 반제품에서 처리).

## D4. 분류 3원칙: 옵션 / 추가상품 / 공정 (엑셀 표기 기준)
- **추가상품** = 해도/안해도 됨 → 추가상품 템플릿 (OPP필름·볼체인·구슬줄). 엑셀 '추가상품 옵션' 표기.
- **옵션** = 반드시 택1 or 여러항목 묶여 단가 → 옵션그룹 (은/금 고리 택1·조각수·면지색상). 엑셀 '가공 옵션'=자재무관→옵션/공정.
- **공정** = 인쇄·후가공 가공행위·한 상품에 같은종류 다중추가 → 공정+공정상세.
- 아크릴 키링 고리=가공옵션→**우선 추가상품**(나중 택1강제 필요시 옵션 전환). 정답없음·운영결정.
- ★미결: 고리 가격포함 vs 별도가산(=별도). '고리없음' 선택시 고리옵션 비표시(위젯). 부착 vs 동봉 구분(상품 '개별상이' 표기).

## D5. 공정 상세 옵션 구조
- 공정은 한 상품에 **여러건 누적**(화이트+클리어·금박+은박 동시). 세부속성=공정상세옵션 분리.
- 별색·코팅·커팅·접지·박·후가공(모서리/오시/미싱/가변)=모두 공정. 양/단면·박크기='한 공정 내부 재입력'=공정상세.
- 박=색상(청박/녹박)별 공정 분리(기존합의 유지)·박크기=공정상세.
- 인쇄(양/단면): 양/단면=인쇄옵션·별색(화이트/클리어/금/은)=공정·그 양단면=공정상세.

## D6. 책자 반제품 구조 (표지/면지/내지)
- **반제품**: 표지·면지·내지 각각 반제품 등록(인쇄 별도). **자재(반제품하위)**: 면지반제품 아래 화이트/블랙/그레이 면지=자재(면지도 종이).
- **완제품(세트상품)**: '레더 하드커버 책자'=세트, 하위 prd_cd에 반제품(표지·면지·내지) 추가.
- 구성예시: 레더하드커버=표지+면지 / 링바인드=표지+면지 / 엽서북=표지+내지 / 포토북=표지+내지+면지 / 하드커버=표지+면지.
- **내지 개수제약**: 하위 반제품별 **최소/최대/증가단위**(min_cnt/max_cnt/cnt_incr). 표지=1/1·면지=1/1(시뮬엔 1~4 노출가능)·무선책자 내지=N종 추가.
- **제본=완제품(세트) 공정**(중철/무선/PUR/링·제본유형=공정). 제본세부(방향·면지옵션·링컬러·바인더링 D링31/42/56)=공정 하나에 묶임(분리 어려움).
- ★미결(내일 11시): 제본세부=공정상세 vs 옵션. 제본별 페이지분리 vs 제약조건 동적표시.

## D7. 조각수 / 스티커 (낱장·완칼)
- 옵션그룹 Min-Max 부재 → 조각수=**5/6/7/8/9/10 각각 개별 옵션 등록**('조각수' 옵션그룹). 현재 조각수별 가격 동일(추후 차등가능).

## D8. 사이즈 코드 / 비규격
- 아크릴 키링=사이즈코드 미관리·**비규격(가로/세로 직접입력) 디폴트**(사유 너무 많음).
- 사이즈코드 보유상품=코드선택+**비규격 체크시 직접입력 병행**.
- ★미결: 사이즈코드 부재시 비규격 입력박스 노출 트리거(**nonspec_yn 플래그**) DB·UI 위치 재확인(서 대표).

## D9. 버그 B1~B5 (회귀방지 대상)
- **B1**: 자재 저장 500에러(상품뷰 자재 2건 동시삭제+1건 변경 동시저장). 개별수정=정상.
- **B2**: OC 두줄+미싱 추가시 '두줄' 소실(프리미엄엽서). 서 대표 즉시수정.
- **B3**: 수량 증가분(Step) 미적용(증가분15인데 +클릭 1씩). Min15·Max1천만 무시.
- **B4**: 프리미엄엽서 '인쇄/인쇄옵션' 중복(공정 인쇄 + 단/양면 별도). 공정에 디지털인쇄 누락=데이터 보정.
- **B5**: 옵션·자재 중복노출(아크릴키링 고리 양쪽). D4 분류확정으로 정리.

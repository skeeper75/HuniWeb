## 2. 축별 diff (정답 vs 라이브)

### 2-1. size — ✅ CORRECT (전 상품)
- **규격 사이즈:** 라이브 SIZ 행이 L1 규격 프리셋과 field-for-field 일치(아트프린트=A3/A2/A1 SIZ_174/197/293·레더아트액자=5x5/5x7/8x8/8x10/A4/A3 SIZ_304~174·아크릴=290x90/290x190/390x290/590x390). `사용자입력`은 size 행 미생성(정당=자유입력).
- **비규격 범위:** 라이브 nonspec_*가 L1 비규격(C6/C7)과 정확 일치 —
  - 아트프린트: L1 200~1200·200~3000 = 라이브 200/1200/200/3000 ✓
  - 레더아트프린트: L1 200~600 = 라이브 200/600 ✓ (소형)
  - 메쉬프린트: L1 200~600 = 라이브 200/600 ✓
  - 일반현수막: L1 500~1750·500~5000 = 라이브 ✓
  - 메쉬현수막: L1 500~900·500~3000 = 라이브 ✓
- **판정:** **사이즈 면적매트릭스 적재 정확**. round-9에서 우려한 "비치수 연속범위 오판" **없음** — 라이브가 이산 규격 SIZ + nonspec 범위로 정확 분리. 입력 UX(혼합)와 사이즈 모델 올바름. → CORRECT(C2).

### 2-2. 자재 mat_typ — ❌ MIS-LOADED (24상품)
- **라이브:** 실사 소재 전부 `MAT_TYPE.08 실사소재`(인화지/매트지/PET/PVC/투명PVC/그래픽천/린넨/캔버스/레더/타이벡/메쉬/현수막천/화이트/블랙/홀로그램/골드/실버).
- **정답:** 패브릭(그래픽천/린넨/캔버스/타이벡/메쉬/현수막천)=`.05 원단`·레더(화이트)=`.06 가죽`(product-bom §0).
- **결정적 증거(SELECT):** 같은 "레더"가 책자 시트에선 `.06 가죽`(MAT_000008·173~175 레더하드커버)인데, 실사 레더(화이트)(MAT_000186)=`.08`. "린넨"도 책자 린넨커버=`.01 종이`, 실사 린넨(MAT_000184)=`.08`. → 실사 소재 일괄 .08 평면화 입증.
- **판정:** MIS-LOADED(자재유형). 소재명·usage(USAGE.07)는 정확. → C4.

### 2-3. 코팅 공정 — ✅ CORRECT (옵션 풀 정합)
- **라이브:** 코팅 옵션 상품에 유광(PROC_000014)+무광(PROC_000015) 2행(mand_proc_yn=N 선택).
- **검증:** 아트프린트포스터(L1 r3=코팅없음·r4=무광코팅·r5=유광코팅 옵션 나열)=라이브 유광+무광 2행(코팅없음=공정 부재). 아트페이퍼(L1 코팅 빈값)=라이브 공정 0행 ✓. 방수/접착방수/폼보드/포맥스/프레임리스/PET배너/미니배너=L1 무광+유광 옵션=라이브 2행 ✓.
- **판정:** 코팅은 옵션 선택 풀로 정당 적재. "코팅없음"을 행으로 안 만든 것 정당. → CORRECT(C5).

### 2-4. 봉제·족자 공정 param — 🟡 MISSING(variant/param) (~5상품)
- **라이브:** 린넨패브릭(124)·캔버스패브릭(125)·캔버스행잉(133)·린넨우드봉(134)=봉제(PROC_000080) **1행 param 없음**. 족자포스터(135)=족자제작(PROC_000082) 1행 param 없음.
- **정답:** L1 린넨패브릭=오버로크/오버로크+리본끈/말아박기/말아박기+면끈/봉미싱(7cm) **5종**·린넨우드봉=오버로크+봉미싱(4cm)·족자=사각족자/원형족자 **2종**. PROC_000080 param(유형/폭)·PROC_000082 param(모양) 마스터에 실재 → param 인스턴스 누락.
- **판정:** 공정 행은 존재하나 **옵션 variant(param) 미적재** = 변형 커버리지 결함(round-7 D-1). → C6.

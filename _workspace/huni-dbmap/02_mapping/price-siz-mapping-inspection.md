> **목적**: 구조분석(`00_schema/schema-relationship-analysis.md`)이 진단한 `component_prices.siz_cd = SIZ_PENDING_*`(2,697행 / 285 placeholder, 7군)을 **상세 매핑 점검**한다. 스키마로 확정 가능한 것은 **기존 라이브 siz_cd로 교정(RESOLVE)** 하고, 후니 모델링 판단이 필요한 것은 **결정하지 않고 정밀하게 프레이밍**한다.

> **권위**: 라이브 railway DB(PostgreSQL 18.4) read-only `SELECT` 1세션(2026-06-06) + `t_prc_component_prices.csv`(placeholder 원천) + round-2 가격 DDL/fit-gap. INSERT/UPDATE/DDL/COMMIT 없음. 비밀번호 stdout 미출력. 식별자/SQL 영어, 해석 한국어.

> **search-before-mint(HARD)**: 모든 RESOLVE는 라이브 `t_siz_sizes`에서 치수·이름으로 기존 행을 먼저 탐색해 증거로 뒷받침. 발명 0. 불확실은 🟡/🔴로 플래그.

### 1-1. GUK4 (870행) — **RESOLVE 🟢 : 단일 출력판형 SIZ_000499**

**현재 placeholder**: `SIZ_PENDING_GUK4` (단일, 870행 전부 동일).

**진단 (데이터 구조 근거 — 추측 아님)**:
- GUK4 870행은 **distinct siz_cd = 1**. 즉 추출 단계가 국4절 단가시트(디지털인쇄비·코팅·별색·커팅)를 **하나의 출력판형 축**으로 평면화했다.
- 이 구성요소들은 `t_prc_formula_components`에 **상품 공식 바인딩이 없다**(grep 결과 0건). 라이브 `price-engine-ddl.md` §316 "코팅·디지털·아크릴·후가공의 component_prices는 상위 상품 원자합산형 공식이 참조(개별 공식 미부여 — **시트단위 구성요소 단가**)"와 정합. → **국4절은 특정 상품의 판형이 아니라, 디지털인쇄 전 상품이 공유하는 출력판형 단가 인덱스**.
- 따라서 "국4절 = 제품별 다른 plate siz로 분기"는 **데이터가 지지하지 않는다**. 870행이 단일 placeholder인 것 자체가 **단일 공유 판형 의도**의 증거.

**search-before-mint 결과 (라이브 증거)**:
```
SIZ_000499 | 316x467 work / 306x457 cut | impos_yn=Y | del_yn=N
  → 국전(636x939)/4 = 국4절 표준 ~318x468 과 정합(bleed 포함). impos_yn=Y=조판(여러 장 앉힘)=출력판형 성격 확정.
  → output_paper_typ_cd='OUTPUT_PAPER_TYPE.01'(국전계열) plate 행은 라이브에 단 1건 = SIZ_000499 (PRD_000016).
  → siz_nm에 "국4/4절/국전" 명명행 0건 → 국4절은 치수행(316x467)으로만 존재.
```

**교정 (RESOLVE)**: 870행 전부 `siz_cd = SIZ_000499`.
- **무발명**: 기존 라이브 siz 재사용. FK(`component_prices.siz_cd → t_siz_sizes` CASCADE) PASS.
- **변환 규칙**: `SIZ_PENDING_GUK4 → SIZ_000499` (1:1 치환, 다른 차원 컬럼 불변).
- 산출: `02_mapping/load_price/_corrected_siz/t_prc_component_prices__guk4_corrected.csv` (870행, siz_cd 교체본).

**🟡 후니 확인 1건 (교정 자체는 RESOLVE, 의미만 확인)**: SIZ_000499는 라이브에서 ① 이름이 `316x467`(국4절 라벨 아님), ② 현재 PRD_000016(프리미엄엽서)에만 plate로 바인딩. 국4절 공유 단가를 이 siz_cd에 묶으면 **"316x467 = 국4절 공유 출력판형"** 이라는 해석이 전제된다. 치수·impos·국전계열분류 3중 증거로 **기술적으로는 동일 판형이 확실**하나, 후니가 "이 siz를 국4절 공유 판형으로 쓸지 vs siz_nm을 '국4절(316x467)'로 보정할지"는 운영 명명 결정. → **교정은 진행하되, siz_nm 명명 보정 여부만 후니 확인**(가격 적재를 막지 않음).

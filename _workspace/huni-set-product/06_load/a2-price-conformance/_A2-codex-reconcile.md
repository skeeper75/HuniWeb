## 확정 결함 3건 (codex 합의·교정 대상·인간 승인 후 dbmap)

### D-C1-02 디지털 흑백/칼라 축 오적재 (저청구·돈크리티컬)
codex가 권위 셀로 확증: `price-digital-print-price-l1.csv` B01 블록 = **흑백단면3000·흑백양면4000·칼라단면4000·칼라양면6000**(B02 블록 = 3500·5000·4500·7000). 라이브 `COMP_PRINT_DIGITAL_S1` 단면=3000(흑백단면)·양면=4000(흑백양면 또는 칼라단면) → 색상엽서(흑백 선택축 없음)는 **칼라단면4000·칼라양면6000이 정답인데 흑백 단가가 매칭**. 단면 −1,000/양면 −2,000원/매 저청구. 교정=색상엽서 단가행을 칼라 단가로 재적재 + S2 칼라양면 배선.

### D-C1-01 디지털 S2 배선 0
6공식 전부 S1만 배선·S2(칼라양면 6000·7000) 배선 0 → 진짜 양면 단가 도달 불가. D-C1-02와 한 쌍. 교정=S2 배선 또는 단면/양면 단가 모델 재설계.

### C4-D01 제본 다종-1배선 (셋트 책자 가격 근본·돈크리티컬)

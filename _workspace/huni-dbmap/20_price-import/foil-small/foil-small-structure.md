## 0. 전수 스캔 결과 (954행 가설 → 실측 반증)

- 시트 격자 = `954 × 26`(빈 격자 포함). **실 데이터 = 1~50행, A~M열뿐.**
- 954행/26열은 워크북 빈 셀 영역(엑셀 max_row 아티팩트). 추가 박 종류/면적구간 **없음**.
- 데이터 블록은 4개(수직 스택). 메인 실측 "A~D 등급" → 실제는 **A~E 5등급**(I~M열).

## 2. 블록별 정밀 구조

### B1. 동판비 (A1~B3)
```
A1: 동판비
A2: 수량        B2: 80x40mm
A3: 1           B3: 5000
A4: *아연판          ← 노트(자재 주석)
```
- 의미: **동판(부자재) 1개 = 5,000원**, 규격 80x40mm 기준. 박 가공 1회당 발생하는 **셋업/부자재 비용**(수량 무관 고정).
- 명함박 라이브 선례에 `COMP_NAMECARD_FOIL_SETUP_*`(박형압 동판셋업비)가 이미 존재 → **동판비 = 별 구성요소(셋업비)** 로 분해 정당.

### B2a / B3a. 면적 → 등급 정의 매트릭스 (5×5 비대칭)
```
        가로:  10mm  20mm  40mm  60mm  80mm     (열 헤더, B9~F9)
세로 10mm:      (공란)  A     A     B     C       ← B10 공란
세로 20mm:       A     A     B     C     D
세로 40mm:       A     B     D     E     E       ← E (F12 'E ' 공백 포함)
```
- **2단 룩업 1단계**: `(세로, 가로)` 작업사이즈 → 등급 A~E.
- 세로 라벨 3개(10/20/40mm)만 있고 가로 헤더 5개(10~80mm) → **비대칭 격자**(세로 60/80mm 미정의 = 더 큰 면적 → 런타임 ceiling 또는 미정의 컨펌).
- **B10 공란**: 10x10mm 등급 미정의(노트 보존). 일반박/특수박 매핑 **완전 동일**(가격만 다름).

### B2b / B3b. 등급별 수량 가격 (밴드 단가표)
```
분류/수량   A        B        C        D        E
200      12,200   15,000   16,400   17,800   19,200
300      14,300   18,500   ...
...
1000     29,000   ...
2000     50,000   ...
...
10000    218,000  358,000  428,000  498,000  568,000
```
- **2단 룩업 2단계**: `(등급, 수량구간)` → 가격.
- 수량구간 18개: 200·300·400·500·600·700·800·900·1000·2000·3000·4000·5000·6000·7000·8000·9000·10000.
- **단위 판별(핵심)**: 값 = 해당 수량의 **총액**(예: A등급 200매 = 12,200원 = 200매 전체 가격). 장당가 아님.
  - 근거: 헤더 `분류/수량`이 수량 자체이고, 200→12,200 / 2000→50,000의 증가율이 비례 미만(수량할인) → **구간 총액 = 합가형**.
  - → `prc_typ_cd = PRICE_TYPE.02 합가형` (엔진: 구간총액 ÷ min_qty = 장당가 환산).
  - 명함박 라이브 선례(`COMP_NAMECARD_FOIL_*`, "오리지널박 합가(완제품가)")도 **동일하게 수량구간 총액**(200매 24,800)이나 라이브에서 `PRICE_TYPE.01`로 등록됨 → **단가/합가 라이브 정합 컨펌 필요**(§decomposition P4 참조).

## 5. "박 = 세트/고정형 신규구조인가" 1차 평결 (구조 관점)

- 박은 **MATRIX도 합산형도 아닌 제3 구조 = "2단 룩업(면적→등급→가격)"**.
- 단, **2단 중 1단(면적→등급)은 앱 계산**이라 DB 그릇에는 **1단(등급×수량→총액)만** 들어감.
- DB 그릇 관점에서는 **밴드 단가표(등급축 × 수량축)** = 스티커 MATRIX의 변형(등급이 siz 대신 옵션축).
- → **진짜 신규구조 = "등급 차원"의 부재**(라이브 10차원에 grade 없음). 이게 round-5 DDL GAP "박 2단룩업" 미해소분의 본질.
- 최종 평결은 decomposition §6에서 등급 차원 처리 판정 후 확정.

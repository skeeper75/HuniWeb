1. 개요 및 기술 스택 요약

본 보고서는 레드프린팅(RedPrinting) 트윈링 책자(컬러) 제품 상세 주문 페이지에서 사용된 위젯(Widget)과 SDK를 상세히 분석한 결과입니다.

핵심 기술 스택

구분기술버전/비고
기반 라이브러리jQuery1.11.3

UI 확장jQuery UI포함 (Autocomplete, Datepicker 등)
슬라이더/캐러셀Swiper.jsBundle (다수 인스턴스)

2.3 의존 관계

productRedWidgetSDK.js는 통합(Integration) 레이어로, 다음 두 파일과 함께 동작합니다:

productRedWidgetSDK.js (브릿지 레이어, 자체 호스팅) ├── widget.js (실제 위젯 런타임, CloudFront CDN) │ ├── widget.css (위젯 스타일, CloudFront CDN) │ └── Shadow DOM 렌더링 → #redWidgetSdk 내부 └── RedEditorSDK.min.js (에디터 SDK, CloudFront CDN)

# 01 — MAP 시트 파싱 (round-24 1단계)

**입력**: `docs/huni/후니프린팅_상품마스터_260610.xlsx` → 2번 시트 `MAP` (A1:Z1055, 유효 12열).
**파서**: `scripts/parse_map.py` (openpyxl, data_only=True). 산출 `map-entries.csv`.

## 시트 구조 (실측 정정)

- **R1 = 1차 카테고리 헤더** (스킬 §2가 R3로 추정했으나 라이브 실측 R1). 12개:
  `01 엽서 / 02 스티커 / 03 인쇄홍보물 / 04 포스터 / 05 사인 / 06 책자 / 07 캘린더 / 08 문구 / 09 아크릴 / 10 라이프 / 11 에코백 / 12 포장`
- **R2 이후 = 콘텐츠**. 각 셀 1개 = 1 엔트리. 컬럼(A~L)이 카테고리.
- 유형 판정:
  - `▶︎` 접두 = **섹션 헤더**(하위분류, 상품 아님) → `section`
  - `→` 접두/포함 = **별칭/교차참조**(타 카테고리 본체) → `alias`
  - 그 외 = **상품 엔트리** → `product`

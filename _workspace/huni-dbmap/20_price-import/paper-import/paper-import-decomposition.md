## 0. 목적지 판정 (핵심 결론)

**질문**: 이 시트는 ⓐ `t_mat_materials`(소재 마스터) vs ⓑ `t_prc_component_prices`(COMP_PAPER 용지비) 중 어디로 가나?

**답: 둘 다 — 단, 분담이 명확하다.**

| 데이터 | 목적지 | 차원/컬럼 | 근거 |
|--------|--------|----------|------|
| 종이명·평량·약어·전지·종이사이즈 | **`t_mat_materials`** (mat_typ_cd=MAT_TYPE.01 용지) | mat_nm·weight·note | 소재 정체 = 자재 마스터. 라이브 107행 기적재 |
| **가격(국4절) I열** | **`t_prc_component_prices`** comp_cd=`COMP_PAPER` | **mat_cd × siz_cd → unit_price** | 라이브 49행 = 이 시트 source(unit_price=I 전건 일치) |
| 가격(3절) J열 | 〃 COMP_PAPER | mat_cd × siz_cd(=**SIZ_000077**) | 3절 판형 용지비(현재 GAP) |
| 연당가 H열 | **DB 미저장** | — | 전지 원가관리용. 국4절 환산가가 가격엔진 입력 |

> **[F-PAP-4 헤더 정정]** 가격표 실제 컬럼: H=연당가·**I=가격(국4절)**·J=가격(3절). 산문이 "H=국4절"로 한 칸 어긋났던 것 정정 — 분해 로직은 처음부터 I열(idx8)을 unit_price로 옮겨 결과는 정합.
| 상품 적용 ● (L~Y) | **가격 그릇 아님**(CPQ 제약) | — | 용지×상품 허용 매트릭스 → round-6 옵션 트랙 |

> **핵심**: 가격 그릇(COMP_PAPER)에는 **디지털인쇄 용지(가격 보유)만** 들어간다. 합판/실사/아크릴(가격 없음)은 소재 마스터 ⓐ 엔트리이고 그들의 단가는 각 트랙(포스터사인 면적매트릭스·아크릴 면적매트릭스)에서 별도. 본 분해는 **COMP_PAPER 단가 그릇 + 그 FK 전제인 소재 마스터**까지를 산출 범위로 한다.

## 1. 공식 매핑 — COMP_PAPER는 "공급 부품"이지 "상품 공식"이 아님

이 시트는 **공식(price_formulas)을 만들지 않는다.** COMP_PAPER는 이미 6개 디지털 합산형 공식의 부품으로 배선돼 있다(라이브 실측):

| frm_cd | 대상 상품군 | COMP_PAPER disp_seq | addtn_yn |
|--------|-------------|:------------------:|:--------:|
| PRF_DGP_A | 엽서·상품권·종이슬로건 | 15 | Y |
| PRF_DGP_B | 모양엽서·라벨택 | 3 | Y |
| PRF_DGP_C | 인쇄배경지·헤더택 | 3 | Y |
| PRF_DGP_D | 소량전단지 | 5 | Y |
| PRF_DGP_E | 접지카드·접지리플렛 | 5 | Y |
| PRF_DGP_F | 썬캡(미출시) | 1 | Y |

> **그릇 엑셀에 formula_components·price_formulas 시트는 RU(재현)만** — 신규 배선 0. 이 시트의 작업은 **COMP_PAPER 단가행(component_prices)을 채우는 것**(소재 선택 시 용지비가 합산되도록). 배선은 이미 완결(가격사슬 정상).

## 2. 🔴 단가형 / 합가형 판별 (Phase11 핵심)

| 구성요소 | prc_typ_cd | 근거 | 엔진 계산 |
|----------|-----------|------|----------|
| `COMP_PAPER` | **PRICE_TYPE.01 단가형** | 셀 = **국4절 1장당 용지비(절가)**. 수량구간 없음(min_qty NULL). 헤더 "가격(국4절)" = 장당 환산가. | `용지비단가 × 출력매수` (출력매수=주문수량/판걸이수, 앱 계산) |

- **합가형(.02) 없음** — 용지비는 절가(장당가)이지 구간총액이 아님. 라이브 `COMP_PAPER.prc_typ_cd=PRICE_TYPE.01` 실측 일치. 추정 아님 — 가격표 헤더가 "가격(국4절)"=장당 환산가이고 라이브가 min_qty NULL(수량구간 미사용)로 절가 곱셈 거동.
- **Roll 단가 2건(유포지+엠보·투명접착PVC)은 별 처리** — I="Roll"은 절가가 아니라 롤 단위 → 단가형/합가형 어느 쪽인지 가격표 단위 불명 → **`9b_SPECIAL_roll` 분리·컨펌**(추정 금지).

## 3. use_dims — COMP_PAPER가 쓰는 차원 (라이브 실측 재현)

```
COMP_PAPER use_dims (라이브 jsonb) = ["siz_cd", "mat_cd"]
```

| 차원 | 사용 | 값 |
|------|:----:|-----|
| `siz_cd` | ✅ | SIZ_000499(316x467 국4절) · SIZ_000077(300x625 3절, GAP) |
| `mat_cd` | ✅ | 종이별 자재코드(MAT_000072 등 49+) |
| `clr_cd` | ✗ NULL | 용지비는 색 무관 |
| `proc_cd`(신설) | ✗ NULL | 공정 무관 |
| `opt_cd`(신설) | ✗ NULL | 옵션 무관 |
| `coat_side_cnt` | ✗ NULL | 코팅 면수 무관 |
| `bdl_qty` | ✗ NULL | 묶음 무관 |
| `min_qty` | ✗ NULL | **용지비는 수량구간 없음**(절가 고정) |

- **과분할 금지**: 10차원 중 2개만 값, 8개 NULL 와일드카드. 같은 (COMP_PAPER, siz_cd, mat_cd)당 단가행 1개 → 동시매칭 0.
- **siz_cd = 출력판형(임포지션 규격)**, 상품 표시사이즈 아님([[dbmap-platesize-is-output-paper]]).

## 4. component_prices long-form 분해 규칙

```
디지털인쇄 용지 행 (종이명 nm, 평량 g, 국4절가 p4, 3절가 p3)
  소재 마스터: mat_cd=lookup(nm) or 신규채번,  mat_typ_cd=MAT_TYPE.01, weight=g, mat_nm=nm
  국4절 단가행: comp_cd=COMP_PAPER, siz_cd=SIZ_000499, mat_cd=위, unit_price=p4, 나머지 8차원 NULL
  (3절 보유 시) 3절 단가행: comp_cd=COMP_PAPER, siz_cd=SIZ_000077, mat_cd=위, unit_price=p3, 8차원 NULL
```

- **동시매칭 0 검증**: 같은 (COMP_PAPER, siz_cd, mat_cd)에 1행. 국4절과 3절은 siz_cd가 달라 충돌 없음.
- **무손실**: 디지털인쇄 85 종이명행 ↔ 단가행 round-trip. 47 RU(라이브 일치, 뉴크라프트/팬시크라프트 포함하면 49) + **15 GAP-price**(14+아이보리 1) + **8 BLOCKED** + 2 Roll + 잔여(3절 5·랑데뷰류 RU 포함은 §5 표로 명시).

## 8. 한 줄 현황

출력소재 분해 설계 완료 — **목적지 = 소재 마스터(ⓐ) + COMP_PAPER 용지비 단가(ⓑ) 둘 다·상품적용매트릭스(ⓒ)는 CPQ 별 트랙.** COMP_PAPER **단가형(.01)**·use_dims=`["siz_cd","mat_cd"]`(2차원, 8차원 NULL)·unit_price=I 국4절가(연당가 H는 DB 미저장·F-PAP-4 헤더 정정). 가격사슬 정상(PRF_DGP_A~F 6공식 배선 완결·아크릴 단절과 대조). 디지털 **85 종이명행**: **47 RU + 15 PRICE-GAP(채움후보·아이보리 1 재분류 포함·뉴크라프트/팬시크라프트는 RU로 정정) + 8 BLOCKED(자재정체 미해소) + 5 3절GAP + 2 Roll-special.** 합판/실사/아크릴 37행=소재 마스터만(가격 각 트랙). 컨펌 4건(GAP채움정책·BLOCKED정체·3절mat·Roll단위). **라이브 COMP_PAPER=49행(실측).** **다음 = 그릇 엑셀(.xlsx)·validator P1~P6.**

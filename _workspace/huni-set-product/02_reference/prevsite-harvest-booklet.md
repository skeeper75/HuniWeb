## 4. 조사신호 (자동 교정 금지·설계가/게이트 보고)

### S-A [높음·돈크리티컬] 표지 양면 가격축 누락 — 4종 확정
- **실측**: 068/069/070 표지 단면 10,000→양면 16,000(+60%)·071 6,000→10,000(+67%). 표지 단/양면이 실제 가격 구성요소.
- **설계**: 표지인쇄=S1(단면) 기본·"양면 시 S2 별행(부활 선결)". 표지 양면이 **선택지로 노출되는데 S2 미부활이면 양면 주문이 단면가로 청구**(60~67% 저청구) 또는 양면 미선택.
- **처리(보고만)**: 설계가 부모 셋트공식에 표지인쇄 **단/양면 분기**(S1/S2 둘 다 배선·표지 print_opt 면축) 반영 재검토. ★가격 값은 권위 가격표로(이 수치는 거동 신호). 영향 = 069·070·071·068 공통(072 제외).

### S-B [중] 071 표지×2 — 이전사이트는 ×1 산정
- **실측**: 071 표지인쇄 단일값(×2 흔적 없음). 이전사이트는 트윈링 표지를 한 번만 계산.
- **설계**: 071 부모공식 표지인쇄·코팅 ×2(앞뒤).
- **처리**: ×2 자동 적용 **보류**(이전사이트 거동=×1·권위 가격표 대조 전까지). 게이트에서 권위 verbatim 단가로 ×1/×2 확정. 잘못 ×2면 ~100% 과대.

### S-C [중] 070 PUR A5 내지 단/양면 (P-1 재판정)
- **정찰 §2 신호 정정**: 이전사이트 select는 PUR 내지=양면만 표시였으나 **AJAX는 단면(p09=1)도 0000 정상 계산**. → 070 단면 내지가 시스템상 가능. 권위 booklet-l1(A5 단면)이 절대 — 라이브로 무효화 말 것. 단 설계 "070 A5=단면"이 라이브와 모순 아님(정찰서 불일치 완화).

### S-D [중] 내지인쇄 면 무관·용지가 면 종속 (M-6)
- **실측**: 내지인쇄[7]는 단/양면 동일(42,000 등)·내지용지[8]만 면에 따라 ~2배. 설계는 내지인쇄를 S1/S2 면별 분기.
- **처리**: 면별 가격차가 **인쇄가 아니라 용지(출력매수)**에서 나는지 설계가 재확인. PRF_DGP_INNER 면 배선 위치 점검(게이트 골든).

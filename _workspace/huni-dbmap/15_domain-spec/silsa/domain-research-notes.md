## 1. 발견한 충돌 — 사이즈 모델 + 가공 택일그룹 (실사 핵심 2건)

엑셀 실측 × 07_domain × round-9 적재 이력 3자 대조.

### 1-1. 사이즈 모델 — 입력UX(혼합) ≠ 가격격자(이산 매트릭스) [핵심 재확인]
- 실사 사이즈 = **규격(A3/A2/A1)** + **고정(600x1800·5000x900)** + **비규격(가로/세로 연속범위)** + **사용자입력**(자유) 혼합.
- **round-9 일반현수막(PRD_000138) L2가 비규격 `500~1750/500~5000`만 보고 사이즈를 비치수 연속범위로 오판**([[dbmap-l2-requires-l1-price-table]]). 실제 권위=포스터사인 가격표 **이산 면적 매트릭스**.
- **올바른 모델:** 비규격 범위(C6/C7)=입력 UX 한계(`t_prd_product_sizes` min/max 제약) / 가격·유효사이즈=포스터사인 매트릭스 셀 / off-grid=각 축 한 단계 큰 규격 ceiling(앱). → **CONFIRM-SL-1.**

### 1-2. 가공 택일그룹 미적재 (G-SL-3)
- 실사 가공(C20)=열재단/재단만/각목/오버로크/4구타공/사각족자/보드 등 다종. UI상 상품별 택일.
- **라이브 excl_groups에 `GRP-BOOK-제본`·`GRP-CAL-가공` 2종뿐** — 현수막/실사 가공 택일그룹 부재(`db-domain-structure-live §330`). 캘린더(GRP-CAL-가공)·책자(GRP-BOOK-제본)와 달리 실사는 택일그룹 미적재.
- **세 후보:** ① GRP-SL-가공 신규 excl_group 제안(ddl-proposer) ② 단순 다중공정(비택일·각 상품 1가공 고정) ③ round-6 옵션그룹으로. → **CONFIRM-SL-3.**

**검증된 도메인 정합(07_domain 예측 ↔ 엑셀 실측 일치):**
- 패브릭 오버로크/봉미싱 = PROC_000080 봉제(유형/폭) ✅
- 배너 4구타공 = PROC_000079 타공(구수=4) ✅
- 족자포스터 사각족자 = PROC_000082 족자제작(모양=사각) ✅
- 일반현수막 열재단 = PROC_000084(round-9 mint) ✅
- 접착투명 화이트별색 = PROC_000008 underbase(G-SL-2) ✅
- 코팅=공정(Q9) ✅

> **포토북 G-PB-2(표지타입 평면)·스티커 G-SK-2(형상 size 흡수) 같은 오모델은 실사에 없음.** 실사 고유 미확정=사이즈 모델(SL-1)·가공 택일그룹(SL-3·G-SL-3)·화이트별색 범위(SL-2·G-SL-2).

## 2. 🔴/🟡 컨펌 질문 (인간 결정 대기)

### CONFIRM-SL-1 [🟡·핵심] 사이즈 모델 — 비규격 범위 vs 포스터사인 매트릭스?
- **상황:** 실사 사이즈=규격+고정+비규격(연속범위)+자유입력. round-9 일반현수막이 비규격 범위만 보고 비치수 연속범위로 오판.
- **가설:** 비규격 범위(C6/C7)=입력 UX 한계(min/max 제약), 가격·유효사이즈=포스터사인 가격표 이산 면적매트릭스 셀, off-grid=ceiling(앱).
- **질문:** 실사 사이즈를 (a) 가격표 매트릭스 셀=유효사이즈 권위 + 비규격은 입력제약만, (b) 비규격 연속범위 자체를 사이즈로, 중 무엇으로 둘까요? (가격은 별도지만 사이즈 모델 결정에 필수)

### CONFIRM-SL-2 [🟡] 화이트별색(C18) 적용 범위 — 도메인 강제 vs 엑셀 명시분만? (G-SL-2)
- **상황:** 화이트별색=투명/홀로그램 소재 underbase(도메인 필수). 엑셀엔 접착투명포스터=`단면`만 명시, 투명포스터★·홀로그램시트커팅엔 빈값.
- **가설:** 투명/홀로그램 전 상품에 PROC_000008 화이트 underbase가 도메인상 필수(엑셀 빈값=누락). G-SL-2 미적재가 결함인 이유.
- **질문:** 화이트별색을 투명/홀로그램 전 상품에 적용(도메인 강제)할까요, 엑셀 명시(접착투명)분만 적재할까요?

### CONFIRM-SL-3 [🟡·핵심] 실사 가공 택일그룹 — GRP-SL-가공 신규 vs 단순공정? (G-SL-3)
- **상황:** 실사 가공(열재단/재단만/각목/타공 등) UI 택일이나 라이브 excl_group 부재(GRP-BOOK/CAL만).
- **가설:** ① GRP-SL-가공 신규 excl_group(캘린더 GRP-CAL-가공 동형) ② 상품별 1가공 고정(비택일) ③ round-6 옵션그룹.
- **질문:** 실사 가공을 신규 택일그룹(GRP-SL-가공)으로 둘까요, 상품별 단순공정으로 둘까요? (현수막류 택일 = 라이브 미존재 이슈)

### CONFIRM-SL-4 [🟡] 거치대·우드봉·액자 = 자재 vs 공정 vs 추가상품?
- **상황:** 거치대(배너/스탠딩)·우드봉(족자)·액자프레임(액자) = 거치/완성 부속. "출력만" 옵션 존재(부속 미선택).
- **가설:** 거치대/우드봉=추가상품(별매 SKU tmpl_cd), 액자가공=공정. booklet BK-3(링)·calendar CL-5(삼각대/트윈링)와 동류.
- **질문:** 거치대·우드봉을 추가상품(별매)으로, 액자를 공정으로 둘까요? "출력만"=addon 미선택 기본값 맞나요?

### CONFIRM-SL-5 [🟡] 본체 usage 슬롯 — USAGE.01 vs .07 vs 실사 전용?
- **상황:** 실사=낱장 단일(내지/표지 없음). 자재=소재 1슬롯(캘린더 USAGE.01 본체와 동형).
- **가설:** USAGE.01 본체(낱장 단일, 캘린더 정합) 또는 .07 공통.
- **질문:** 실사 소재 본체 usage를 USAGE.01 본체로 둘까요? (캘린더·디지털인쇄 낱장과 동일 슬롯)

### CONFIRM-SL-6 [🔴] C12 파일명약어 — 견적 DB 귀속?
- **상황:** 파일명약어 = 생산 메타. t_* 컬럼 부재.
- **가설:** 견적 밖. 전 시트 일괄(DP-2·ST-5·BK-6·PB-6·CL-6과 동일).
- **질문:** (a) 견적 밖 (b) note 보존 (c) 신규 컬럼? (전 시트 일괄)

### CONFIRM-SL-7 [🟡] 아크릴스티커 인쇄방식 — UV(실사 아님) 라우팅 확정?
- **상황:** 유광/미러아크릴스티커(005-0007/0008) 폴더=`레이저커팅` → UV(PROC_000002). 실사 시트에 묶였으나 인쇄방식은 실사 아닌 UV.
- **가설:** 폴더가 생산라우팅 권위 — 아크릴스티커=UV 라인(레이저커팅·아크릴가공). 시트 분류(실사)와 인쇄방식(UV) 불일치는 정상(소재가 아크릴이라 UV).
- **질문:** 아크릴스티커 인쇄방식을 UV(PROC_000002)로 확정할까요? (실사 시트에 있으나 UV 라우팅)

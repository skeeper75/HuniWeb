## 2. 정체 발견 (요지)

1. **F-ID-1 소재 정체는 올바르나 자재유형(mat_typ) 오적재.** 모든 소재가 라이브에서 `MAT_TYPE.08 실사소재`로 적재됨(SELECT 실측). 그러나 도메인 권위(product-bom §0·column-dictionary §5)는 **패브릭(린넨/캔버스/그래픽천)=MAT_TYPE.05 원단·레더=MAT_TYPE.06 가죽**으로 분류. 정체상 소재명 자체는 정확(인화지/매트지/PET/린넨/레더/타이벡 등 L1 일치)하나 **유형 코드가 단일 .08로 평면화** → MIS-LOADED(자재 마스터 적재 결함). → correction C4.

2. **F-ID-2 카테고리 정체 오연결.** 28상품 전부 `CAT_000298 실사`(upr_cat_cd=NULL·cat_lvl=3인데 부모 없음 = 고아) 단일 노드에 연결. 그러나 **정상 상품명별 카테고리 노드(CAT_000067~099)가 전부 실재**(포스터=CAT_000004 하위·사인=CAT_000005 하위, SELECT 실측). 디지털인쇄 #1 C-09/C-14와 동형 — load_categories가 시트 `구분`(=실사)을 잉여 그룹으로 만들어 묶음. → correction C1.

3. **F-ID-3 부속(addon/set) 정체 대량 누락.** 액자·족자·행잉·배너·현수막의 부속(우드행거·우드봉·천정형고리·배너거치대·끈·각목·큐방)이 L1에 명시(추가 C21)되나, 라이브 addon=0·set=0(일반현수막만 round-9 CPQ로 적재). "단품+부속" 정체가 라이브에 미반영. → correction C7.

4. **F-ID-4 아크릴스티커 인쇄방식 = UV(실사 아님).** 005-0007/0008 폴더=레이저커팅 → UV 라인(PROC_000002). 실사 시트에 묶였으나 라우팅은 실사 아님. **인쇄방식 컬럼이 라이브에 없어 영향 제한적**(공정에 인쇄방식 행 부재) → 정체 인지 사항(🔴Q-SL-A). 단 카테고리는 시트커팅/스티커(CAT_000095/096)로 정상 노드 실재.

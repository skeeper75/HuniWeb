## §3 멱등 정정 SQL · DRY-RUN 결과

오염이 0건이므로 **삭제/UPDATE SQL은 발행하지 않는다**(비파괴·교정 불요). 대신 ① BLOCKER 반증을 **재현 가능한 검증 SELECT**로 고정하고, ② 만일 향후 진짜 오염이 유입될 경우를 위한 **멱등 논리삭제 템플릿**(현재 매칭 0행)을 첨부한다.

### §3-1 정상 옵션 보존 확인 (재현 SELECT)
```sql
-- 134/135 의 prd-키 옵션 = 5행, 전건 엑셀 L1 정상. (read-only)
SELECT g.prd_cd, p.prd_nm, g.opt_grp_nm, o.opt_cd, o.opt_nm,
       i.ref_dim_cd, i.ref_key1
FROM t_prd_product_option_groups g
JOIN t_prd_products p ON p.prd_cd = g.prd_cd
LEFT JOIN t_prd_product_options o
       ON o.prd_cd = g.prd_cd AND o.opt_grp_cd = g.opt_grp_cd
LEFT JOIN t_prd_product_option_items i
       ON i.prd_cd = o.prd_cd AND i.opt_cd = o.opt_cd
WHERE g.prd_cd IN ('PRD_000134','PRD_000135')
ORDER BY g.prd_cd, g.disp_seq, o.disp_seq;
-- 실측 결과(2026-06-15):
-- PRD_000134 | 가공 | OPV_000031 | 오버로크+봉미싱(4cm) | OPT_REF_DIM.04 | PROC_000080
-- PRD_000134 | 추가 | OPV_000032 | 출력만             | (null)
-- PRD_000135 | 가공 | OPV_000033 | 사각족자           | OPT_REF_DIM.04 | PROC_000082
-- PRD_000135 | 가공 | OPV_000034 | 원형족자           | OPT_REF_DIM.04 | PROC_000082
-- PRD_000135 | 추가 | OPV_000035 | 추가없음           | (null)
-- → 5행. 오염 0.
```

### §3-2 차원 오염 부존재 확인 (재현 SELECT)

```sql
SELECT prd_cd, opt_cd, ref_dim_cd, ref_key1

-- 실측 결과: 0행. (차원 오염 없음)
```

### §3-3 아티팩트 증명 (prd-키 vs 그룹-키)
```sql
-- 게이트 §P6 오염 목록이 "공유 opt_grp_cd 집계"임을 증명.
SELECT 'PRD-keyed 134/135'  AS scope, count(*) AS cnt
  FROM t_prd_product_options WHERE prd_cd IN ('PRD_000134','PRD_000135')
UNION ALL
SELECT 'GROUP-keyed (그들의 4그룹·전 상품)', count(*)
  FROM t_prd_product_options
 WHERE opt_grp_cd IN ('OPT_000013','OPT_000014','OPT_000015','OPT_000016');
-- 실측 결과: PRD-keyed = 5 / GROUP-keyed = 46.
-- 차이 41행 = 형제 상품(PRD_000018/055/066/069)의 정상 옵션 = 게이트가 본 "오염".
```

### §3-4 멱등 논리삭제 템플릿 (현재 매칭 0행 — 발행 보류)
> 향후 진짜 오염(엑셀 L1에 없는 옵션)이 134/135에 prd-키로 유입될 경우에만 사용. 현재 WHERE 절 매칭 = 0행이므로 DRY-RUN 영향 0. 실행 금지(인간 승인).
```sql
-- DRY-RUN 전용 (ROLLBACK). hard-delete 금지 → del_yn 논리삭제.
BEGIN;
UPDATE t_prd_product_options o
   SET del_yn = 'Y', del_dt = now(), upd_dt = now()
 WHERE o.prd_cd IN ('PRD_000134','PRD_000135')
   AND o.del_yn = 'N'
   -- 엑셀 L1 족자 명시 옵션집합 밖이면 오염 (현재 모두 집합 안 → 0행)
   AND o.opt_nm NOT IN ('오버로크+봉미싱(4cm)','출력만','사각족자','원형족자','추가없음')
   AND NOT EXISTS (  -- 멱등: 이미 논리삭제면 제외
     SELECT 1 FROM t_prd_product_options x
      WHERE x.prd_cd=o.prd_cd AND x.opt_cd=o.opt_cd AND x.del_yn='Y');
-- 예상 영향 행수: 0 (오염 부존재)
ROLLBACK;  -- DRY-RUN: 절대 COMMIT 금지
```
**DRY-RUN 결과: 영향 행수 0 (UPDATE 조건에 걸리는 134/135 prd-키 오염 옵션이 존재하지 않음). 멱등성 만족(NOT EXISTS).**

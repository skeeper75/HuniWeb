## 결함 D-01 [돈크리티컬·견적불가] B01 투명3T 83셀 미적재 (하삼각 누락)

- **유형:** 미적재 (이 빠진 셀)
- **블록/차원:** B01 면적격자 / `COMP_ACRYL_CLEAR3T` mat_cd=MAT_000043, use_dims=[mat_cd,siz_width,siz_height,min_qty]
- **패턴 [HARD]:** 권위 196셀(14×14 꽉 참, 가로↔세로 대칭) 중 라이브 113셀만. **미적재 83셀 = 전부 w>h 하삼각** — 라이브는 상삼각(w≤h)만 적재, 하삼각은 대칭쌍 (h,w)가 라이브에 존재함을 83/83 확인. 즉 적재 시 (가로≤세로) 셀만 넣고 대칭 미전개.
- **정답값(예):** 60x20=3400, 70x40=5100, 100x30=5300, 200x180=30700 (전 83셀 = coverage 분석 출력).
- **라이브값:** 해당 (w,h) 단가행 부재 (NULL).
- **돈영향: 견적불가/최소가 과소청구.** 엔진이 (가로>세로) 입력(예 가로100·세로30)을 받으면 단가행 미스→off-grid ceiling 또는 0/최소가. 대칭쌍을 엔진이 자동 교환(w,h swap)하지 않으면 손님이 가로>세로로 입력한 절반의 사이즈가 견적 0 또는 최소가. ★엔진이 (w,h) 정렬해서 조회하면 우회 가능하나 데이터 무결성은 결함(권위는 196 전셀 명시).
- **재현 SQL:**
  ```sql
  SELECT siz_width, siz_height, unit_price FROM t_prc_component_prices
   WHERE comp_cd='COMP_ACRYL_CLEAR3T' AND mat_cd='MAT_000043'
     AND siz_width='100' AND siz_height='30';  -- 0 rows (권위=5300)
  ```
- **라우팅:** dbm-price-import-prep(격자 재펼침) → dbm-load-execution(83행 INSERT, 대칭 전개). 또는 엔진이 min(w,h)/max(w,h) 정규화 조회로 환원하면 코드트랙(C). **데이터 트랙 권장**(권위 196셀 명시).

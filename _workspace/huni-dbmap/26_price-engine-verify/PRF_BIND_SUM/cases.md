## 2. 워크스루 (대표 2케이스)

### G1 — 중철책자·100권 (사슬 완결·엔진 경로 가능)
- 멤버십: 상품=중철책자 → COMP_BIND_JUNGCHEOL (상품별 1고정). 공식 배선됨 ✅.
- 수량구간: 100권 → min_qty 100 (≤100 최대) → unit_price 700.
- 단가형: 700 × 100권 = **70,000원**. ROUND_HALF_UP(정수) = 70,000.
- 시계열: apply_ymd 단일 → 최신 선택 자명.
- 동시매칭: comp별 min_qty 비중복 → 1행만 매칭 ✅.

### G3 — 무선책자·500권 (비경계 수량)
- 멤버십: 무선책자 → COMP_BIND_MUSEON. **공식 미배선** → 엔진 경로 계산불가(공식 우회 직접 조회).
- 수량구간: 500권 → min_qty 100 (≤500 최대·다음 1000 미달) → unit_price 500.
- 단가형: 500 × 500권 = **250,000원**.
- 비경계 입증: 500은 구간 경계(100/1000) 아님 → 100구간 단가 적용(주문수량 이하 최대 min_qty 규칙 작동).

## 3. 최소수량 미달 케이스 (sanity)
- BIND 단가행 최소 min_qty=1 → 1권부터 계산 가능. "최소 미달(계산불가)"은 본 클래스 미발생(최소구간이 1권).

## 4. 사슬 단절 표기 (정직성)
- G2~G8(무선/PUR/트윈링) = **공식에 comp 미배선** → 가격엔진(evaluate_price) 경로로는 계산불가. 값은 comp 단가행 직접 조회(공식 우회)로 골든 검증만 수행.
- 중철(G1)만 공식→comp 완결 경로. 이 단절은 임의 0원 대체 금지·"BROKEN(unwired)"로 정직 표기.

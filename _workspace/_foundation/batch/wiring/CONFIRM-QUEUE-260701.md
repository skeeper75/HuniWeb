# 사장님 확인 필요 항목 (배선 작업에서 나온 질문들) — 2026-07-01

배선 연결 작업(가격공식에 가격구성요소 잇기)을 하다가 **사람이 정해줘야 풀리는 질문**들이 나왔습니다.
추측으로 값을 넣으면 가격이 틀릴 수 있어(돈 문제) 멈추고 여쭙니다. 아래만 답주시면 막힌 게 풀립니다.

## A. 지금 당장 답이 필요한 것 (가격에 직접 영향)

> ※ 정정(2026-07-01): 화이트인쇄명함은 **코팅을 안 씁니다**(상품마스터·레드프린팅 확인). "클리어"는 코팅이 아니라
> **클리어 별색인쇄**입니다. 그래서 아래 질문도 "코팅"이 아니라 "클리어 별색" 기준으로 바뀝니다.

### 1. 화이트인쇄명함 — 클리어(별색) 기본값
- 화이트인쇄명함은 **클리어 별색인쇄 있음 / 없음** 을 고릅니다(코팅 아님).
- 손님이 클리어를 안 고르면 **클리어 없음(더 싼 쪽)** 으로 계산하면 될까요?
- ☐ 클리어 없음이 기본 ☐ 클리어 있음이 기본 ☐ 손님이 반드시 골라야 함

### 1-b. 화이트인쇄명함 — 자재(용지) 확인
- 가격표의 묶음가는 **큐리어스스킨(MAT_000137)** 기준인데, 현재 040에 연결된 자재는 다른 코드(MAT_000362~365)입니다.
- 같은 용지인데 코드만 다른 건지, 다른 용지인지 확인 필요합니다.
- ☐ 같은 용지(코드만 다름) ☐ 다른 용지 ☐ 확인 필요

### 2. 화이트인쇄명함 — 수량
- 지금 가격표에 **100장 가격만** 들어가 있습니다. 100장은 정확히 계산됩니다.
- **다른 수량(200·500·1000장)** 가격표가 있으면 주세요(있으면 채워서 전 수량 계산 가능).
- ☐ 가격표 있음(첨부) ☐ 100장만 판매 ☐ 확인 필요

## B. 막혀서 보류 중인 상품 (호스트·구성이 불명확)

### 3. 접지(접는) 카드 — 3단·6크리즈
- "접지비"라는 단가는 들어와 있는데, **이미 다른 곳에서 접지비가 한 번 더 매겨지고** 있어 그대로 이으면 **이중으로 돈이 붙습니다**.
- 질문: 접지카드는 **접는 방식(2단/3단/6크리즈)이 "다른 상품"인가요, 한 상품 안의 "선택옵션"인가요?** 그리고 **6크리즈**는 어느 상품에서 파나요?

### 4. 캘린더 — 벽걸이/탁상 제본
- 캘린더 5종(벽걸이·탁상 등)에 **가격공식 자체가 없어** 견적이 안 나옵니다.
- 또 제본 방식 코드가 상품과 안 맞습니다(상품엔 수축/타공/트윈링만 연결).
- 질문: **캘린더 가격을 어떻게 매기나요?**(제본비 + 본문 장수 가격?) 참고할 옛 가격표/예시 주시면 만듭니다.

### 5. 단가만 있고 "어느 상품인지" 모르는 것 4개
아래는 가격표에 단가는 있는데 **어느 상품에서 파는지** 안 잡힙니다. 상품명만 알려주세요.
- **포토카드 "대량"** — 세트 판매와 별도인가요? 같이 못 고르게 막나요?
- **완칼 + 타공 1구 / 2구** — 어느 상품에서 타공 1개·2개를 고르나요?
- **미러(거울) 아크릴 3T** — 어느 아크릴 상품인가요?

### 7. 폼보드/포맥스 — 가격 권위 충돌 + A1
- 가격표는 **소재(화이트보드/블랙보드·3mm/5mm) × 사이즈(A3/A2/A1)** 구조가 맞습니다(확인 완료).
- 막는 것 2가지:
  - **화이트보드 A3 가격**: 가격표(260527)=**7,000원** vs 상품마스터(260610)=**6,000원**(라이브도 6,000). 어느 게 맞나요? ☐ 7,000 ☐ 6,000
  - **A1 사이즈**: 가격표엔 A1(폼보드 화이트 20,000·블랙 24,000·포맥스 3mm 23,000·5mm 30,000)이 있는데 라이브엔 없습니다. A1도 판매하나요? ☐ A1 판매(가격표대로) ☐ A1 안 함

### 6. 박명함 단가 유형 표기
- 박명함 단가 행의 "유형" 표기가 살짝 안 맞습니다(합가형 vs 고정금액). **지금 파는 수량에선 가격이 정확**하므로 급하지 않습니다. 나중에 정리.

## 이미 끝난 것 (참고)
- **박명함**: 양면·홀로그램이 단면·일반박 가격으로 **싸게 잘못 나오던 것 → 정상가로 교정**(승인 후 적용).
- **화이트인쇄명함**: 견적 0원의 진짜 원인을 찾았습니다 — 단/양면 선택수단이 없어 인쇄비가 0이 되고 용지값(6,213원)만 나오고 있었습니다. 정답가(14,500~19,000원)는 **묶음가표에 이미 있는데 공식에 안 연결**돼 있습니다. 위 1·1-b·2 확인 후 연결하면 정상화됩니다(코팅이 아니라 클리어 별색 기준).
- **폼보드(검정)/포맥스(5mm)**: 가격은 준비됐고, **상품에 사이즈만 등록하면** 견적 나옵니다.
- **엽서북 30p**: 30p가 20p 가격으로 나오던 것 → 교정 중(보정 작업 진행).

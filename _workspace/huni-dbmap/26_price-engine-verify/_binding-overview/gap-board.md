# 제본 전수 갭 보드 (gap-board)

> 작성 2026-06-15 · [[binding-overview]] 동반. 갭 분류 + 라우팅.
> 권위 = 가격표/상품마스터 엑셀 + 라이브 실측. DB 미적재.

---

## 0. 갭 유형 정의

| 유형 | 의미 |
|---|---|
| G-WIRE | comp·단가행 적재됐으나 공식에 미배선 |
| G-BIND | comp 적재됐으나 어느 상품 공식에도 미바인딩 |
| G-FORMULA | 상품에 가격공식 자체 없음(❌미구성) |
| G-COMP | 가격표엔 제본 가격 있으나 라이브 comp 자체 없음 |
| G-PRICE | 가격표(권위) 자체 부재 |
| G-VERIFY | comp 적재됐으나 가격표 값 1:1 대조 미수행 |

## 2. 갭 집계

| 갭 유형 | 종수 | 비고 |
|---|--:|---|
| ✅ 완결 | 1 | 중철(책자·comp배선완료) |
| 🟡 G-WIRE만(값OK) | 3 | 무선·PUR·트윈링(책자4) — 값 32/32 검증됨 |
| 🟡 G-WIRE+G-BIND+G-VERIFY | 7 | 하드커버3+캘린더4 — 단가행 있으나 배선·바인딩·대조 0 |
| ❌ G-FORMULA(미구성) | 다수 | 하드커버책자/포토북/캘린더/다이어리/노트류 13상품 |
| ❌ G-PRICE(권위부재) | 2+ | 레이플랫·링바인더류 |

## 3. 가장 큰 갭 (우선순위)

1. **G-FORMULA 13상품(최대)** — 제본 쓰는 상품 18 중 13이 가격공식 없음. 손님이 하드커버책자·포토북·캘린더·다이어리·노트 가격을 못 받음. = 매출 영향 직접.
2. **G-WIRE+G-BIND 8종(하드커버3+캘린더4+싸바리)** — 단가행은 적재됐는데 공식·바인딩이 없어 "데이터는 있으나 안 쓰임". 배선+바인딩만 하면 활성화 가능(comp 신설 불요).
3. **책자4 G-WIRE(정립중)** — 가장 진척됨([[PRF_BIND_SUM]]).

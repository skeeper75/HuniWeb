## 3. 결정 후 자동 처리 경로 (결정만 주면 하네스가 실행)

- **#1 좌표 siz 채택 시**: 262 좌표 siz_cd 등록 요청서 생성 → 등록 후 placeholder 치환 → 재조립(917행 적재가능). **면적함수 채택 시**: round-2 면적매트릭스형 공식 연계 설계.
- **#2~5**: 결정값으로 siz_cd 치환/신규 요청서 → 재조립.
- **#6~9**: 결정 기준으로 correction 라운드(자재 variant / output_file_typ 정규화 / 인쇄면 / 상품 처분) → 적재본.
- 각 처리 후 dbm-validator R1~R6 + 라이브 롤백 DRY-RUN 재게이트.

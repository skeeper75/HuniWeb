## 4. 발견한 생성자 오류 (전부 미세·결론 무영향)

1. **⑥ "정상 페어 271건 무손상" → 실제 279건** (MINOR·과소 표기). 명세서 §0이 정상 페어를 main_cat_yn='Y'만 세어 271로 적었으나, 부카테고리 정상 페어(NORMAL_PAIR main='N') 8건이 추가로 존재 → 정상 페어 합계는 **279**. 삭제 대상(고아 113)과 무관하므로 결론 불변, 다만 무손상 대상 수치는 279로 정정 권고.

2. **④ §1-1 표 내부 일관성** (MINOR). 표 헤더는 "del 제외"인데 .07을 33·합계를 340으로 적음(둘 다 del=Y MAT_000339 각목 1건 포함 수치). del 제외 정확값은 **.07=32·합계 339**. 정정 대상 자재행은 동일하므로 결론 불변.

3. **④ §1-1 표 합계 "340" 라벨** (MINOR). 04 재실측의 340(del 포함)을 del 제외 표에 그대로 옮긴 것. 일관 표기 시 339(del 제외)/340(del 포함) 구분 권고.

> 위 3건은 전부 표기/집계 라벨 문제이며, **교정 대상 행 집합·BLOCKED 판정·GO 결론에는 영향 없음**. 과대/과소 주장(실제로 고치지 않을 행을 고친다고 한)이 아니라, 오히려 ④는 "정정 0건"으로 보수적이다.

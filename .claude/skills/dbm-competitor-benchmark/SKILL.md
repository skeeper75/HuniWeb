# dbm-competitor-benchmark — 경쟁사 가격 합리성 벤치마크 (round-18 확장)

우리 가격계산공식 산출값을 같은 옵션으로 경쟁사와 대조해, 우리 계산이 **합리적 범위인지**를 검증한다. 경쟁사와 가격을 맞추는 게 아니라, 우리 계산/매핑/적재에 구조 결함이 있는지를 외부 기준으로 잡아내는 것이 목적.

## 0. 게이트 원칙 (사용자 directive · HARD)

**"같은 옵션을 했는데 터무니없이 가격차이가 나면 문제가 있는 것이다."**
- 경쟁사 가격 = **합리성 오라클**(정답 아님). 우리 권위는 가격표 엑셀.
- **정상**: 차이가 가격전략(소재 등급·마진·프로모션)으로 설명 가능. 후니가 싸거나 비싸도 합리적이면 GO.
- **🔴 터무니없음**: 자릿수 차이 / 우리값 비정상(계산불가·0·음수·과대합산 round-18 D-2) / 같은 옵션인데 설명 불가 배율. → 우리측 결함 신호 → arbiter 라우팅.
- 경계는 단일 숫자 박제 금지 — 인쇄 도메인 상식(같은 사양 시장가는 통상 동일 자릿수·수배 이내). 자릿수·비정상은 무조건 🔴.

## 1. 경쟁사 가격 획득

| 경쟁사 | 출처 | 방법 |
|---|---|---|
| 레드프린팅 | 기존 캡처 우선(`_workspace/huni-widget/01_reverse`·`05_qa/captures`·`s3_raw_captures`, `result_sum.PRICE`) + 부족분 본인 계정 가격 API(`/std/prod/jobcost`, `.env.local RP_*`) | 캡처 신선(2026-06). PRICE=0(토큰만료/미설정)이면 라이브 1회 재획득 |
| 와우프레스 | 공개 WooCommerce Store API(`/wp-json/wc/store/v1/products`) | `print-quote-live-crawl` 안전모드(WebFetch·저트래픽·캐시우선·주문/폼 금지). 가격 미노출이면 구조만·대조 보류 |

레드=사용자 본인 시스템·와우프레스=공개 API → 외부 발신 승인됨(사용자 확정). 저트래픽 유지·비밀값 비노출.

## 2. 동형 옵션 매핑

우리 prd_cd + 선택(사이즈·소재·도수·수량·후가공)을 경쟁사 동일 상품군의 같은 옵션으로 1:1 대응. **같은 옵션이 비교의 전제** — 사양이 다르면 비교 무의미. 못 맞춘 축은 명기(근사 대조 = 약한 신호, 🔴 판정 금지). 매핑 근거(어느 경쟁사 상품·어떤 옵션값)를 표에 남긴다.

## 3. 대조·판정 (B-gate)

클래스별 대표 상품 × 대표 선택+수량으로:

| 상품·옵션 | 우리 재계산 | 와우프레스 | 레드프린팅 | 차이율 | 판정 |
|---|--:|--:|--:|--:|:--:|
| 예: 엽서 100x150·4도단면·100매 | 44,330 | (조회) | (조회) | ±% | 정상/주의/🔴 |

- 우리값 = `26_price-engine-verify/<class>/recompute-cases.md`(verifier 보정매칭값).
- 판정: 정상(전략차) / 주의(설명 필요한 수배차) / 🔴 터무니없음(자릿수·비정상).

## 4. 산출 (`_workspace/huni-dbmap/27_competitor-benchmark/<class>/`)

- `benchmark.md` — 3열 비교표 + 차이율 + 판정 + 동형매핑 근거 + 획득 출처(캡처/라이브).
- `absurd-gaps.md` — 🔴 목록 + 원인 가설(과대합산·판형 미해석·prc_typ 오적재·요건 미적재·옵션 미반영) + arbiter 라우팅.

## 5. 검증 게이트 (dbm-validator, B-gate)
- B1 동형 옵션 정합(사양 일치·근사는 강등) · B2 경쟁사 가격 출처 실재(캡처 파일/API 응답) · B3 차이율 산식 정확 · B4 🔴 판정 도메인 정당(자릿수·비정상만 🔴·전략차 오인 금지) · B5 우리값=verifier 산출 일치 · B6 라우팅 정직.

## 6. 비파괴·범위
- 15 가격공식 클래스 대표. DB 미적재. 경쟁사 가격 조회만(주문/결제/폼 금지). 실 교정은 arbiter→round-5/13+인간 승인.

## 0. 한 줄 결론 (적재 선결)

면적 단가(포스터사인 매트릭스 `[가로×세로]`)와 **설치/거치(아일렛·타공·각목·거치대·우드봉·봉제)는 가격·데이터 모두 분리되는 별개 축**이다. 후니 엑셀이 이미 `가공(옵션)`·`추가(옵션)`을 **면적 단가와 다른 컬럼(W/X·Y/Z)에 별도 단가**로 두었고, 경쟁사(레드프린팅) 가격 화면도 `인쇄비(면적) + 후가공 + 합계`로 **명시 분리** — 즉 **면적 단가행(component) + 설치 옵션(option_items, 자기 단가 comp)을 따로 적재**하는 것이 도메인·경쟁사 양쪽이 가리키는 정답. 면적 단가에 설치비를 녹이면 안 된다.

### 1-3. 실내 vs 실외 구분 (엑셀 명시)

- **명시 분리 있는 상품 = PET배너·메쉬배너만**: 추가 컬럼에 `실내용배너거치대 / 실외용배너거치대` 2종. 실내용=가벼운 X/Y배너 거치대, 실외용=무게추·바람 견디는 거치대(도메인 통념·경쟁사 정합). 그 외 상품엔 실내/실외 컬럼 분리 없음.
- **소재가 실외 적합성을 가른다(엑셀 소재 컬럼 묵시):** 방수포스터(PET)·접착방수(PVC)·타이벡·메쉬·현수막천 = 실외 내후. 인화지/매트지/패브릭천 = 실내. → 실내/외는 **명시 옵션이 아니라 소재 선택의 결과**(상품 정체). 별도 차원 신설 불요.

## §2. 경쟁사 리서치 — 옵션·가격·설치 갭헌팅

> WebSearch/WebFetch(2026-06-15). 답습 아닌 **갭 발견**(후니 스키마가 이미 흡수하는지 대조). Sources §5.

### 2-1. 레드프린팅(사용자 본인 설계) — 현수막 옵션 taxonomy (라이브 페이지 실측)

레드프린팅 현수막 상품(BNBNFBL) 주문화면 옵션 구조:
- **재단:** 정사이즈재단(기본) / 방풍커팅 / 모양재단
- **아일렛:** "테두리 간격(mm)" 설정 + `사각귀퉁이 / 사각테두리 / 은색아일렛` 택1
- **봉제:** 접어꿰매기 / 줄미싱 / 봉미싱 택1
- **고리:** 사방고리
- **부자재(별매):** 각목(타공 포함 선택) · 큐방(10개/1세트) · 로프(3mm/5mm·10m) · 큐브양면젤리테이프(20개/1세트)
- **가격 표기:** `인쇄비 · 후가공 · 합계 · 부가세`로 **구분 표시** + "후가공 금액이 변동될 수 있음"

> **🔴 결정적 갭헌팅 결과 — 가격 분리가 경쟁사 화면에 명시:** 레드프린팅은 **인쇄비(=면적 단가)와 후가공(=타공/봉제/아일렛)을 화면 라인으로 분리**한다. 후니 엑셀(가공 단가 X컬럼·추가 단가 Z컬럼이 면적가격 R/V와 별도)과 **동일 구조**. → 후니 스키마가 이미 표현 가능(면적 comp + 옵션 comp 분리). 신규 축 도입 불요.
> **갭 1(아일렛 세부):** 레드는 아일렛을 `테두리간격(mm) + 위치/종류 택1`로 더 세분(은색아일렛 등). 후니는 `타공(4/6/8개)`로 구수만. → 후니 `PROC_000079 타공{구수}` param이 흡수 가능(은색아일렛=자재 변형은 미세 갭, 컨펌 후보). benchmark-competitors §5 "후니 prcs_dtl_opt가 이미 흡수, 신규 불요" 정합.

### 2-2. 와우프레스 — 현수막 마감 4종 + 부자재 별매

와우프레스 현수막 마감: ① 나무+로프(봉미싱 후 막대나무+로프) ② 쇠고리/아일렛(모서리 고리·끈 연결) ③ 미싱 마감(시접 접어 박음질) ④ 열재단. **지지대·아일렛 연결 끈은 별도 구매.** → 후니 가공(봉미싱/열재단/타공) + 추가(각목/끈/큐방)와 **1:1 대응**. 후니가 더 세분(각목 가로/세로 기준)된 영역도 있음.

### 2-3. 패브릭/족자 거치 — 봉 가공 4종(국내 표준)

패브릭포스터 봉(우드봉) 거치 표준: ① 봉가공 일반(50mm 접어 봉 숨김) ② 봉가공 고리형(상단 고리 20cm 간격) ③ 작은 고리(상하 1cm) ④ 벨크로. 우드행잉=분위기용 권장. → 후니 패브릭/행잉/우드봉족자의 `오버로크 / 봉미싱(4cm) / 우드봉+면끈 포함 / 우드행거+면끈 포함`이 이 표준의 후니 구현. **갭 없음 — 후니가 동등 표현.**

### 2-4. 배너 거치대(실내/실외) — 전문 거치대몰 대조

배너스토어·printingkorea 등: X배너/Y배너 거치대, 대형거치대(실외 무게추형). 후니 `실내용배너거치대 / 실외용배너거치대` 2분류는 이 시장의 단순화 버전. → 후니는 거치대를 **추가상품(별매 tmpl_cd)** 으로 두면 충분. 거치대 세부 모델(X/Y/롤업)까지는 후니 미세분(컨펌 후보, 현재는 실내/실외 2종이면 운영 가능).

### 2-5. 경쟁사 면적 가격 산정 방식 대조

| 항목 | 후니(포스터사인) | 레드/와우프레스 | 정합 |
|------|------------------|------------------|------|
| 면적 가격 | `[가로×세로]` 매트릭스 셀(이산 격자) | ㎡ 단가 또는 사이즈 구간 매트릭스 | 동형(후니=격자 룩업, off-grid=ceiling) |

| 최소수량 | 밴드류 min_qty | 수량구간 | 동형 |

## §3. 면적매트릭스 가격 구조 + 설치/거치 가격 반영 방식

> 구조 해부는 `20_price-import/poster-sign/poster-sign-structure.md`가 26블록으로 완료(재유도 0). 본 절은 **설치/거치가 가격에 어떻게 들어가는지**(적재 결정 직결)에 집중.

### 3-1. 면적매트릭스 = 출력(색) 단가만 (silsa-price-via-poster-sign 권위)

- 포스터사인 `[가로×세로]` 매트릭스 셀 = **출력물 본체 단가**(소재+실사출력+기본재단). 차원 `use_dims=["siz_cd"]`(순포스터) 또는 `["siz_cd","min_qty"]`(밴드류). 라이브 실측 일치.
- [HARD·사용자] 실사 가격 = 이 포스터사인 매트릭스가 권위(실사 inline R/S/V 무시). off-grid = 각 축 한 단계 큰 규격 ceiling(앱 런타임·DB 미저장).
- **이 단가에는 설치/거치가 안 들어있다.** 매트릭스 셀은 "색 입힌 출력물" 가격일 뿐.

### 3-2. 설치/거치 = 별도 옵션 단가 (엑셀 W/X·Y/Z 권위)

엑셀 실사 시트 컬럼 구조가 분리를 증명:

| 컬럼 | 의미 | 가격 컬럼 | 성격 |
|------|------|-----------|------|
| 가공(W) | 타공/봉제/족자/보드마운팅 | **가격(X)** | 후니 수행 공정 = 옵션 단가 |
| 추가(Y) | 각목/끈/거치대/우드봉/액자 | **추가옵션가격(Z)** | 거치 부속 = 추가상품 단가 |
| (면적) | 포스터사인 매트릭스 | (가격표 시트) | 출력 본체 단가 |

→ **3-축 가격 = 면적단가(출력) + 가공단가(설치공정) + 추가단가(거치부속).** 엑셀이 세 컬럼으로 분리해 둔 것이 적재 청사진.

### 3-3. 라이브 가격사슬 현황 (read-only 실측)

- 28상품 전건 단일 공식 `PRF_POSTER_FIXED`에 바인딩(완비). 단 **공식이 `COMP_POSTER_ARTPRINT_PHOTO` 1개 comp만 배선** → 27상품 가격 조회불가(가격사슬 부분단절, poster-sign-structure §4.1). 이건 **면적단가 축의 결함**이지 설치 축과 무관.
- 라이브 `t_prc_price_components`에 옵션 comp(`COMP_POSTEROPT_*` 20종)도 정의돼 있으나 공식 배선·단가행이 희소. **설치/거치 옵션 가격 적재도 미완.**

## §4. 적재 함의 — 면적 단가행 + 설치 옵션 분리

### 4-1. 적재 구조 결론 (도메인+경쟁사 합치)

```
PRD(28) ──바인딩──▶ PRF_POSTER_<소재>(소재별 공식 분리)
                         │
        ┌────────────────┼──────────────────────┐
   [면적 단가 축]    [설치공정 축]          [거치부속 축]
   COMP_POSTER_<소재>  COMP_POSTEROPT_타공/봉제   추가상품(tmpl_cd)
   use_dims=siz_cd     option_items(.04 공정)   option_items(.03 자재)
   ← 포스터사인 매트릭스  ← 엑셀 가공(X)단가        ← 엑셀 추가(Z)단가
```

1. **면적 단가행(필수·1차):** 포스터사인 매트릭스를 `(siz_cd, unit_price)` long-form으로 언피벗 → `component_prices`. **🔴 선결 차단 = siz_cd 채번**(687 면적조합 중 667=97% siz 미실재, poster-sign-structure §2). 그리고 **소재별 공식 분리**(`PRF_POSTER_<MATERIAL>`)로 가격사슬 단절 해소. 이게 면적 적재의 본체.
2. **설치공정 옵션(가공W):** 타공/봉제/족자/열재단 = `option_items` ref_dim `OPT_REF_DIM.04`(공정) + 자기 단가 comp(`COMP_POSTEROPT_*`, 엑셀 X컬럼 단가). **면적 단가에 합산 금지** — 경쟁사 화면(인쇄비/후가공 분리)·엑셀(별 컬럼) 양쪽 권위.
3. **거치부속 추가상품(추가Y):** 각목/끈/큐방/우드봉/우드행거/거치대 = 추가상품(`t_prd_product_addons` tmpl_cd) 또는 `option_items` ref_dim `OPT_REF_DIM.03`(자재) + 단가(엑셀 Z컬럼). "출력만 / 거치대없음 / 추가없음" = 부속 미선택 기본값(가격 0).
4. **택일 그룹(G-SL-3):** 현수막 가공(봉미싱/열재단/타공) UI 택1이나 라이브 excl_group 부재(GRP-BOOK/CAL만). → 신규 `GRP-SL-가공` 제안 또는 option_groups 단일선택. CONFIRM-SL-3 미해소.

### 4-2. 라이브 오염 경고 (round-13 패턴 — 적재 전 정정 필요)

라이브 option 실측에서 **PRD_000134(우드봉족자)·135(족자포스터)에 타 상품 옵션이 혼입**(단면/양면 도수·몽블랑/앙상블 등 종이류·미싱/오시·유포스티커). v03 마이그레이션 오염([[dbmap-correctness-audit-round13]] 패턴). → **면적/설치 적재 전, 이 28상품 옵션 레이어를 엑셀 가공/추가 명시값 기준으로 정정(오염 제거)** 해야. 안 그러면 설치 옵션이 종이옵션과 뒤섞인 채 적재됨.

## 0. 카테고리 축 정의 (무엇이 이 축에 속하는가)

`t_cat_categories` = **판매 분류 트리**(self-ref `upr_cat_cd`, `cat_lvl` 1~3). 고객 사이트 네비게이션·상품 그룹핑.
판별 단일 기준: **"이것이 고객이 메뉴에서 탐색하는 분류 노드인가?"** — Yes면 cat. 상품의 *제조 속성*(소재/공정)이면 다른 축.
(출처: `01-rules §6.1·§6.3` · `schema-design-intent-map §2 클러스터 A`)

핵심 컬럼: `cat_cd` PK(`CAT_` 순차) · `upr_cat_cd`(self-ref tree) · `cat_lvl`(1~3) · `use_yn`.
연결 테이블: `t_prd_product_categories`(prd_cd, cat_cd, **`main_cat_yn`**).

## 2. 카테고리 권위 소스 (왜 엑셀이 약한가)

- 상품마스터에 **명시적 카테고리 컬럼은 약함.** `구분`/상품군이 후보이나 판매 분류는 라이브 `t_cat_categories` 306행 + `t_prd_product_categories`(main_cat_yn)가 권위.
- **시트 `구분`(엽서/명함…) ≠ 카테고리** [HARD 함정]: `구분`은 시트 편의 그룹(행 그룹핑 키)이지 판매 분류가 아니다(round-11 C1 함정).
  - (출처: `01-rules §6.2·§6.1` · `02 §2 ⑥` 트리)
- ∴ 카테고리 축은 **권위 엑셀이 침묵에 가까운 축** → 정답 기준선 = 라이브 트리 구조 규칙(§1) + 고아 진단(§3). 데이터 정답은 라이브가 권위.

## 4. 삼중 바인딩 (카테고리)

> 카테고리는 **커머스 축**(생산/가격 무관). `schema-design-intent-map`에 카테고리 전용 삼중바인딩 행은 없으나, 클러스터 분류상:

| 측면 | 카테고리 귀속 | 출처 |
|------|---------------|------|
| **① UI 바인딩** | 고객 사이트 네비게이션·상품 catalog 그룹핑(componentType 무관·메뉴 레이어) | `01-rules §6.1·§6.3` |
| **② 생산 바인딩** | **없음** — 카테고리는 생산 입력 아님(분류 노드). 제조 속성을 카테고리에 넣지 말 것 | `01-rules §6.1` 오염금지 |
| **③ 가격 바인딩** | **없음** — 가격은 t_prc_* / 직접단가. 카테고리는 가격 차원 아님 | `schema-design-intent-map §1.1` 클러스터 A vs D |

> CIP4/JDF에 카테고리 개념 없음(생산 표준) → 커머스 축이라 표준 갭헌팅 무의미(`01-rules §6.4`).

## 5. 정규화 / FK

- **코드:** `CAT_NNNNNN` 순차. 멱등 = `(cat_nm, upr_cat_cd)` 경로 유일(같은 경로 노드 재발급 금지).
- **FK:** `upr_cat_cd` self-ref(트리·ON DELETE RESTRICT) · `t_prd_product_categories.cat_cd`(+main_cat_yn).
- **적재 선후:** base_codes → cat_categories(self-ref 부모 lvl1 → 자식 lvl2/3) → products → product_categories. **고아 방지 위해 트리 완비 후 상품 연결.**
- (출처: `01-rules §6.5·§6.6`)

## 6. 진단가에게 넘기는 핵심

1. **카테고리축 = 판매 분류 트리만.** 시트 `구분`·제조속성(소재/공정)을 카테고리로 보지 말 것(C1 함정).
2. **고아 = 부카테고리 중복**(트리 길잃음). 정답 = **페어 삭제**(재연결/신설 아님 — 정상 잎노드 전부 실재). 라이브 COMMIT 완료(DELETE 111·UPDATE 12).
3. **main_cat_yn = 주 1개 + 보조 다중.** 상품은 leaf cat에 main='Y' 1개.
4. **권위 엑셀 침묵 축** — 데이터 정답은 라이브 트리 권위. 명함/상품권/단품형 정상노드 미확인분은 BLOCKED(C-CAT-1).

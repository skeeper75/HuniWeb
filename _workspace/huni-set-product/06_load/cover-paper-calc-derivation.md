## 2. "계산식"이 가리키는 공통 용지비 산식 (상품마스터 권위)

### 2.1 계산공식집 seq9 — 용지비 공통 산식 (calc-formula-draft-l1.csv verbatim)

```

```

- ★이것이 후니 전 상품군의 **용지비 공통 산식**. "크기별 기준단가"의 원천 시트가 바로 **출력소재 시트**(= COMP_PAPER 산지). 즉 row98 "계산식"은 "I열에 정액을 못 박으니 이 공통 용지비 산식으로 계산하라"는 포인터.
- 표지 전용지에 적용 시:
  - **크기별 기준단가** = 아트지150g 국4절(316x467) 절가 = **46.65** (F98 구매정보 "아트150" → 출력소재 row9)
  - **출력매수** = 표지 펼침(390x268 A5/532x355 A4)을 출력판형(국4절/3절)에 판걸이한 판수 × 부수 (앱 임포지션 런타임 계산)
  - **+5장(손지율)** = 출력매수에 손지 5장 가산(앱 계산 단계)

### 2.2 하드커버무선 6비목 블록(seq63~70)에 "용지비" 세부 산식이 없는 이유

- 하드커버무선 블록은 용지비를 판매가 식(seq64)에만 두고 **세부 산식을 재서술하지 않음** = **seq9 공통 용지비 산식을 그대로 상속**한다는 의미(전 상품군 공통이라 블록마다 재기재 안 함). 표지·내지 용지 둘 다 이 공통 산식으로 종이별 국4절 절가×출력매수.

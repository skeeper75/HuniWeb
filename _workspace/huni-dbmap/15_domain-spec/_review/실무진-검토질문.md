# 후니프린팅 상품 데이터 정리 — 실무진 검토 요청 (round-11)

> **작성** 2026-06-10. **대상 시트:** 디지털인쇄 · 스티커 · 책자 · 포토북 · 캘린더(+디자인캘린더).
> **목적:** 위 5개 상품군의 엑셀 데이터를 시스템 데이터로 정리하면서 **사람이 결정해야 할 항목**을 모았습니다. 데이터 분석으로 "이렇게 하면 될 것 같다"는 **권장안**까지 냈으니, 실무진께서 **맞다/다르다/보류**만 표시해 주시면 됩니다.

---

## 이 문서 읽는 법

- 각 질문은 **무엇을 / 왜 / 어떻게(선택지)** 순서입니다. 어려운 시스템 용어 대신 인쇄 작업 용어로 적었습니다.
- **「권장」** = 분석 결과 가장 맞아 보이는 안. 동의하시면 체크만, 다르면 알려주세요.
- **Part 1**(우선 결정)은 한 번 답하면 여러 상품군이 한꺼번에 정리되는 항목입니다. **여기부터 봐주세요.**
- **Part 2**는 상품군별 개별 항목, **Part 3**은 실무진 답변이 아니라 저희가 시스템에서 직접 확인할 항목입니다.

답변 예시: `▶ 답변: ☑ 권장안 동의` 또는 `▶ 답변: ☐ 다른 안 → (내용 적어주세요)`

# 스티커 — 컬럼 도메인 사전 (round-11 확대 #1)

> **작성** 2026-06-10 · round-11(컬럼 도메인 의미 + 상품 BOM + 적재명세). 설명 한국어, 식별자/컬럼/코드/SQL 영어.
>
> **목적:** 상품마스터 `스티커` 시트의 **각 컬럼이 무슨 인쇄 도메인 의미를 인코딩하는가**를 애매모호 0까지 확정하고, 각 컬럼을 라이브 t_* 적재 축에 귀속시킨다. 디지털인쇄 파일럿 패턴 적용, 시트 고유 축(커팅·조각수·인쇄방식 분기)을 집중 규명.
>
> **권위 순서(HARD):** ① 후니 PDF(공정관리·주문프로세스) ② 라이브 DB/`07_domain` KB(entity-semantic-model L3·process-recipe-tree L2·db-domain-structure-live·benchmark-competitors §2/3) ③ 국내외 인쇄 표준(보조). 추정 0 — 미지는 가설+출처+컨펌(`domain-research-notes.md`). **표준 충돌 시 후니 권위.**
>
> **확정도:** ✅ PDF/DB 권위 · 🟡 PDF부분+표준 · 🔴 미수록(가설·컨펌 필요).
> **시트 규모:** **32 composite 컬럼(C1~C32, C33~37 빈칸), 16상품, 1090행**(타투스티커가 924행으로 대부분 — 사이즈×자재 캐스케이드).

## 0. 시트 구조 — 다단 헤더(R1 구분 그룹 + R2 세부)

상품 1행(C1~C4) 아래 **사이즈가 세로로 캐스케이드**(C5~C가 사이즈별 여러 행). 디지털인쇄와 동일 골격이나 **그룹 구성이 다르다**:

- **디지털인쇄에 있고 스티커에 없는 것:** 접지·모서리·오시·미싱·박/형압·건수(옵션)·추가상품/추가가격(전부 빈값).
- **스티커 고유 그룹:** `커팅(옵션)` C24(독립 그룹) · `조각수(옵션)` C25(신규 축).
- **그룹(R1) 9개:** 사이즈(필수)·파일사양·주문방법(필수)·종이(필수)·인쇄(옵션)·별색인쇄(옵션)·코팅(옵션)·**커팅(옵션)**·**조각수(옵션)**·제작수량(필수)·후가공(옵션)·추가상품(옵션).

> **시트 1차 발견:** 스티커는 **단일 인쇄방식이 아니다.** C13 폴더가 디지털인쇄/실사출력/화이트인쇄/합판인쇄/전사인쇄 **5종으로 분기** = 상품별 인쇄방식(process-recipe-tree §1). 디지털인쇄 시트(거의 디지털 단일)와 결정적으로 다른 점.

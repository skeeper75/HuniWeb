### 3.2 산정 공식 (계산공식집초안 r4~r12 — 권위)
```
판매가 = 인쇄비 + 코팅비 + 용지비 + 후가공비 + 추가상품
(1) 출력매수 = 주문수량 / 판걸이수            [판걸이수 시트, 크기별]
(2) 인쇄비(4도) = [수량행단가] × 출력매수      [디지털인쇄비, 국4절/3절]
(3) 별색인쇄비 = [인쇄비단가] + [출력매수행]×[칼라열]   [디지털인쇄·별색·3절별색 없음]
(4) 코팅비 = [수량행단가]×[코팅타입열] × 출력매수  [코팅]
(5) 용지비 = [크기별 기준단가] × ([출력매수]+5장 손지율)  [출력소재]
(6) 후가공비 = [제작수량행]                    [인쇄후가공]
(7) 후가공박(대형) = [면적별동판비] + [군별/칼라별 합가]  [후가공_박(대형)]
(8) 추가상품 = 목록별 가격적용
```

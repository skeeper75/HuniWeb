## 3. 오염 방지 — 무엇을 하지 않았나 (round-13 교훈 반영)

| 오염 패턴 | 본 정정이 회피한 방식 |
|-----------|----------------------|
| 색/형상/사이즈를 자재로 오적재 (MAT_TYPE.08~10 오염) | 면지 색 3종=variant 옵션으로 분리 컨펌(자재 과분할 금지·Q-ST-H). 레더는 이미 가죽(.06) 정합 — 추가 오적재 0 |
| 별색을 도수로 오적재 | 문구=별색 SKU 미운영 — 해당 없음 |
| 신규 카테고리 코드 남발 | 정상 노드 119~123 **재사용**(search-before-mint), 신규 mint 0. 노드순≠prd_cd순 의미 매칭으로 오연결 방지 |
| 신규 공정 코드 남발 | 미싱제본은 제본 family 재사용 후보 0 입증 후에만 mint 제안(ddl-proposer·컨펌). 후가공 미싱(030)으로 억지 재사용 안 함(의미 충돌) |
| 고아행·중복행 hard-delete | 고아 300=논리강등(main_cat_yn='N'), .07 중복=논리삭제(del_yn='Y'+del_dt) — DELETE 안 함 |
| PK 컬럼 in-place UPDATE(충돌 위험) | usage_cd(PK 일부)는 새 행 INSERT + 구 행 논리삭제로 처리 — UPDATE로 PK 변경 안 함 |
| 적재처 없는 값 억지 삽입 | 미싱제본·PVC·합지보드는 적재처/마스터 부재로 **주석 보류**(컨펌) — NULL 강제 안 함 |
| stale 정정 강행 | 레더(ST-04) 라이브 재측정 결과 이미 .06 정합 → "고칠 것 없음"으로 격하(round-13 .08 주장은 BATCH-4 선적용 전 stale) |

## 축1 — 단가행 값 정합 (권위 verbatim 대조)

### D-C1-02 [HIGH·돈크리티컬] COMP_PRINT_DIGITAL_S1 = 흑백/칼라단면 오적재 (전 디지털 색상엽서 저청구)
**입증(53 수량구간 전수 일치):**
- 라이브 `COMP_PRINT_DIGITAL_S1` 국4절(SIZ_000499=316x467) **단면(POPT_000001)** = `3000,2000,1600,1400,1200,…,40`(53티어) = 권위 **흑백(1도) 단면(B열)** 전 구간 verbatim 일치.
- 라이브 동 comp **양면(POPT_000002)** = `4000,2800,2500,2000,1700,…,145` = 권위 **칼라(CMYK) 단면(D열)** 전 구간 verbatim 일치.
- 권위 **칼라(CMYK) 양면(E열)** = `6000,4600,4300,…` → 라이브 S1에 없음. S2(국4절 단면 POPT_000001)=`4000,2900,2500…`=권위 흑백양면(C열), S2 양면(POPT_000002)=`6000…`=권위 칼라양면(E열)인데 **S2 미배선**.

**왜 결함인가(권위·옵션 교차):**
- 상품마스터 디지털 전 상품의 `인쇄(옵션)` distinct = `{단면, 양면}` 뿐. `흑백/칼라` 선택축 부재. `t_prt_print_options`에도 흑백/칼라 코드 없음(POPT=단면/양면). 즉 이들은 **컬러(CMYK) 출력 상품**이며 색상 선택지가 없다.
- 프리미엄엽서(PRD_000016) 옵션레이어 `인쇄` 그룹 = 단면(ref_key1=1→POPT_000001)/양면(2→POPT_000002)뿐.
- 따라서 고객이 **단면** 선택 → 엔진 `_row_matches(print_opt_cd=POPT_000001)` → S1 흑백단면(3000) 매칭. **정답=칼라단면(4000). 매당 1,000원 저청구.**
- 고객이 **양면** 선택 → S1 칼라단면(4000) 매칭. **정답=칼라양면(6000). 매당 2,000원 저청구.** + 진짜 양면단가(S2, 6000)는 배선 부재로 영구 도달 불가.

**규모:** PRF_DGP_A 10상품 전부 색상엽서·쿠폰/상품권(저청구). 3절(SIZ_000077) 단면=`3500,2500,2000…`=권위 3절 흑백단면(B) 동일 패턴(동형 오적재).
**진원 추정:** 가격표 평면화 로더가 흑백열→단면, 칼라단면열→양면, 칼라양면열→S2(미배선)로 축을 어긋나게 적재. 의미축 정정 = print_opt_cd(흑백/칼라) ↔ comp S1/S2(단면/양면) 재정렬 필요. **교정 인간 승인(dbmap 위임).**

### D-C1-03 [MED] 별색 comp 분산·고아 (CLEAR/PINK/GOLD/SILVER S1·S2, WHITE_S2 배선 0)
- `COMP_PRINT_SPOT_WHITE_S1`(530행)이 실제로 5색 전부 보유: proc_cd PROC_000008(화이트)/9(클리어)/10(핑크)/11(금색)/12(은색) × 53티어 × 단면/양면 = 530. 값 = 권위 별색 5색(전부 동일가) 단면 3000/2500/2300…, 양면 6000/5000/4600… verbatim 일치. **WHITE_S1 데이터·매칭 정합.**
- 그러나 별도 comp `COMP_PRINT_SPOT_CLEAR_S1/PINK_S1/GOLD_S1/SILVER_S1`(각 53행)·`*_S2`(53)·`WHITE_S2`(53)는 **어느 공식에도 배선 0** = 고아(EXTRA). 데이터 잉여(돈영향 없음·매칭 안 됨)지만 정리 대상.
- ★별색 comp는 단면(POPT_000001)=3000, 양면(POPT_000002)=6000 으로 단면/양면 축이 **정상**(D-C1-02의 디지털 comp와 달리 흑백/칼라 혼입 없음) — 디지털 comp만 오적재임을 역으로 입증.

### D-C1-04 [HIGH·돈크리티컬] COMP_NAMECARD_STD_S1/S2 자재 커버리지 결손 (5택1 중 2만 적재 → 3택 견적불가)

## 축3 — 바인딩 정합 + 미바인딩 root cause

- C1 공식 8종 전부 use_yn=Y·바인딩 존재. **미바인딩 디지털 상품 = 0**(바인딩 24건이 디지털 전 상품 커버).
- 단, 바인딩 ≠ 견적가능: C1 24상품 중 **11상품(투명/화이트인쇄/핑크별색/금은별색엽서·모양엽서·미니접지카드·인쇄배경지2·인쇄헤더택·봉투제작·썬캡)은 옵션그룹 0** → 가격공식은 붙었으나 옵션선택 경로 부재(CPQ 미구성). **A2(가격테이블) 범위 밖 — CPQ 트랙(§7/§21) 라우팅** 참고로 기록.
- 투명엽서(PRD_000019) 출력용지규격 315x467=SIZ_000522 — COMP_PAPER·DIGITAL_S1에 SIZ_000522 행 0(국4절 SIZ_000499만 적재). 투명엽서는 옵션 0이라 현재 견적불가이나, CPQ 구성 시 plt 커버리지 결손 발현 예정(LOW·관찰).

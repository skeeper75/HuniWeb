- **권위 가격공식 = 계산공식집 시트에 7셋트 전부(088 제외 6셋트는 명시·088은 "보류중") 존재** — "진성 부재" 단정은 **오류**였다.

## 1. 7셋트 권위 가격공식 (계산공식집 시트 추출 · 출처 = calc-formula-draft-l1.csv 행번호)

### 1.1 원자합산형: 중철책자/무선책자/PUR책자/하드커버무선 (L47~L54)

> 적용 셋트: **PRD_000072 하드커버책자 · PRD_000077 레더 하드커버책자**(제본방식=하드커버무선)
> (068~071 중철/무선/PUR=단일 완제품·세트 아님이나 공식 동일군)

```
판매가 = 내지인쇄비 + 표지인쇄비 + 표지코팅비 + 제본비 + 용지비 + 후가공비          (L48)
  (1) 내지인쇄비 = [총내지매수행][도수열] × 총내지매수                              (L49)
  (2) 총내지매수 = 부수 × (페이지수 / 판걸이수)                                    (L50)  ← 앱 이중수량 곱
  (3) 표지인쇄비 = 출력매수 × 수량행단가                                          (L51)
  (4) 표지코팅비 = 출력매수 × 수량행단가                                          (L52)
  (5) 제본비   = [수량행][제본종류열]   (참조: 제본 시트)                           (L53)
  (6) 후가공박(대형) = [면적별동판비] + [면적별 A~E군][군별/칼라별 수량행 합가]      (L54)
```
- **구성 항목**: 내지인쇄비·표지인쇄비·표지코팅비·제본비·용지비·후가공비 (6비목 Σ).
- `*(가격포함)` **없음** — 각 비목이 별도 산식.

### 1.2 원자합산형: 트윈링책자 (L55~L61)

> 적용 셋트: 단일 완제품 **PRD_000071 트윈링책자**(세트 아님). 세트형 트윈링 = **PRD_000082 하드커버 링책자**가 §1.3과 §1.1 사이(하드커버트윈링).

```
판매가 = 내지인쇄비 + 표지인쇄비 + 표지코팅비 + 제본비 + 용지비 + 후가공비          (L56)
  (1) 내지인쇄비 = [총내지매수행][도수열] × 총내지매수                              (L57)
  (2) 총내지매수 = 부수 × (페이지수 / 판걸이수)                                    (L58)
  (3) 표지인쇄비 = 출력매수 × 수량행단가 × 2     ← ★표지 ×2 (앞/뒤 양면 표지)        (L59)
  (4) 표지코팅비 = 출력매수 × 수량행단가 × 2     ← ★코팅 ×2                          (L60)
  (5) 제본비   = [수량행][제본종류열]                                              (L61)
```
- **트윈링 특징**: 표지인쇄·표지코팅이 **×2**(앞/뒤 표지 2장). 사용자 제공 권위 확정과 일치.

### 1.3 원자합산형: 하드커버링책자 (L62~L70)

> 적용 셋트: **PRD_000082 하드커버 링책자**(제본방식=하드커버트윈링) · **PRD_000088 레더 링바인더**(제본방식 "보류중"=BLOCKED)

```
판매가 = 내지인쇄비 + 표지인쇄비 + 표지코팅비 + 면지인쇄비 + 면지코팅비 + 용지비 + 제본비   (L63)
  (1) 내지인쇄비 = [총내지매수행][도수열] × 총내지매수                              (L64)
  (2) 총내지매수 = 부수 × (페이지수 / 판걸이수)                                    (L65)
  (3) 표지인쇄비 = 제작수량 × 인쇄비수량행단가 × 2                                 (L66)
  (4) 표지코팅비 = 제작수량 × 코팅비수량행단가 × 2                                 (L67)
  (5) 면지인쇄비 = 제작수량 × 인쇄비수량행단가 × 2     ← ★면지 인쇄 추가·×2          (L68)
  (6) 면지코팅비 = 제작수량 × 코팅비수량행단가 × 2     ← ★면지 코팅 추가·×2          (L69)
  (7) 제본비   = [수량행][제본종류열]                                              (L70)
```
- **하드커버링 특징**: §1.2 대비 **면지인쇄비·면지코팅비 2비목 추가**(8비목 Σ)·표지/면지 전부 **×2**. 사용자 제공 권위 확정과 일치.

### 1.4 고정가형: 엽서북 / 떡메모지 (L71~L72) — `*(가격포함)`

> 적용 셋트: **PRD_000094 엽서북 · PRD_000097 떡메모지**. 참조 시트: `엽서북/떡메`.

```
판매가 = [수량행][옵션열]                                                         (L72)
```
- **고정가형**: 별도 비목 산식 없이 **수량행 × 옵션열 단가표가 곧 판매가**(용지·인쇄·제본 전부 단가표에 내장).
- 시트명이 `엽서북(가격포함)`·`포토북(가격포함)`의 **`(가격포함)`** 마커 = "별도 가격공식 없이 가격 내장"의 권위 표기. → 이것이 `*(가격포함)` 정체.
- **엽서북**(094): 라이브 `PRF_PCB_FIXED`로 이미 표현(완제품 통합단가 comp·use_dims=[siz_cd,min_qty]·페이지 20p/30p×인쇄면 S1/S2). 권위 [수량행][옵션열] = 라이브 PCB comp 단가체계와 정합.
- **떡메모지**(097): 라이브 **미바인딩**. 권위 고정가형 [수량행][옵션열] 존재 → 적재 대상.

### 1.5 포토북 (가격포함) — base24 통합형 (photobook-l1 시트 · 계산공식집 명시행 부재)

> 적용 셋트: **PRD_000100 포토북 [디자인명]**. 권위 시트 = **`포토북(가격포함)`**(photobook-l1).

- **계산공식집 시트에 "포토북" 명시 공식행은 없음**(L73~ 캘린더로 넘어감) — 그러나 시트명 `포토북(가격포함)` 자체가 **`*(가격포함)` 고정가 계열**임을 명시.
- 권위 데이터 구조(photobook-l1·row15 비고): **"상품단가 + 페이지당단가 적용"** + `*책등에따라다름`(책등=페이지 파생).

- 이전 CONFIRM-3(포토북 권위행 미특정) **해소**: 권위 시트 = photobook-l1(`포토북(가격포함)`)·공식 = base24+per2p 통합형(`*(가격포함)` 계열).

### 3.1 2층 집계

- **적재 대상(권위 공식 존재 + 라이브 미적재 = 전사 대상) = 5건**: 072·077·082·097·100.
  - 그중 §18 PRF 설계 완비 = 4건(072·077·082·100). 097은 권위 존재·§18 전용 PRF 미설계(엽서북 동형 신설 1건 필요).

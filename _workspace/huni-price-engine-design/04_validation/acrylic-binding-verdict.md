### E6-2. ★163 미니파츠 골든 날조 + 견적불가 — **FAIL [NO-GO]**
설계가 163 골든을 **"50x50→3840"**(§7-2·§0-1)으로 제시하나, **163의 등록사이즈는 50x50이 아님**:
```

FROM t_prd_product_sizes ps JOIN t_siz_sizes s ON s.siz_cd=ps.siz_cd WHERE ps.prd_cd='PRD_000163';
-- SIZ_000365  120x50  120.00  50.00   ← 유일한 등록사이즈

```
163은 nonspec='N'(자유치수 불가)이라 **120x50만 견적 가능**. 그런데 본체자재=MAT_042(1.5T) 격자(B02)는 **최대 100x100**까지만 존재:
```

-- 100.00  100.00

```
엔진 추적: width 120 > 격자 최대임계 100 → `match_component`이 ERR_ABOVE_MAX 반환 → 구성요소 합산 제외 → strict=계산불가, lenient=0원. 즉 **163은 PRF_CLR_ACRYL 바인딩해도 자기 등록사이즈로 견적 불가**.

→ 설계의 "163 GO·골든 50x50=3840"은 **존재하지 않는 사이즈로 만든 순환 골든**(dodge-hunt 적발). 실제로는 (a) MAT_042 격자가 120x50을 안 담거나 (b) 163 자재축이 틀렸거나 (c) 등록사이즈가 틀림 — 셋 중 하나의 적재 결함. **163 = NO-GO, §26 무결성/dbmap으로 폐루프**.

## ★Q2 — addon 저청구 리스크 판정 [돈크리티컬 핵심]

### 권위 정답가 = "본체 면적 + 부착비"인가 "본체 면적뿐"인가?

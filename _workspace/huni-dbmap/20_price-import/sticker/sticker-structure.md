## 0. 한 줄 요약

스티커 시트 = **7개 논리 블록**을 수직 스택. B01은 **3차원 매트릭스**(수량×사이즈6×소재그룹3), B02~B04는 2차원(수량×사이즈), B05~B06은 세트/번들 단가. **복합셀 다발**(판수+치수, 소재 묶음, 코팅, 구성요소 명시)이 분해 대상.

## 1. 논리 블록 7종 (수직 스택)

| 블록 | 셀 범위 | 제목 | 유형 | 차원 |
|------|---------|------|------|------|
| **B01** | A1:S44 | 반칼 자유형/규격 스티커 (국4절) | **3D 매트릭스** | 수량(행) × 사이즈 6 × 소재그룹 3 |
| **B02** | A45:F52 | 낱장(완칼) 자유형 스티커 | 2D 매트릭스 | 수량 × 사이즈 5(A4/B4/A3/B3/A2) |
| **B03** | A56:F63 | 낱장(완칼) 자유형 투명스티커 | 2D 매트릭스 | 수량 × 사이즈 5 |
| **B04** | A67:B74 | 대형(완칼) 자유형 스티커 | 1D | 수량 × 400x600 단일 |
| **B05** | A78:B82 | 타투 스티커 (3장단위) | **세트/번들 단가** | 기본가 2000 + 3장당 4000 |
| **B06** | A86:B89 | 스티커팩 (54장 1세트) | **세트 고정가** | 1~1000 모두 4000 |

## 2. B01 3차원 매트릭스 상세 (핵심 복합 구조)

```
        │ A5(4판)      │ A4(2판)      │ A3(1판)      │ 90*190(6판)  │ ... (사이즈 6)
        │ 124x186mm    │              │              │              │
수량 │ 유포 무광/유광 투명 │ 유포 무광/유광 투명 │ ... (소재그룹 3 반복)
─────┼──────────────┼──────────────┼ ...
 1   │ 6000 7000 7000 │ 5000 6000 6000 │ ...
 2   │ 6000 7000 7000 │ ...
 3   │ 5900 6900 6900 │ ...
```

- **행2(병합 헤더) = 사이즈 6**: `A5(4판)/124x186mm` · `A4(2판)` · `A3(1판)` · `90*190(6판)` · `100*148(8판)` · `90*110(12판)`. 각 사이즈는 3컬럼 차지(B2:D2 등 병합).
- **행3(하위 헤더) = 소재그룹 3**: 각 사이즈 아래 `유포/비코팅/미색` · `무광코팅/유광코팅` · `투명/홀로그램` 반복.
- **A열 = 수량(min_qty 축)**: 1,2,3…(상향 구간).
- **T3 = 구성요소 명시**: `"종이+인쇄+커팅"` → 이 단가가 담는 가격 구성요소(=공식 구성).
- **셀 = (수량, 사이즈, 소재그룹) 조합 단가**.

## 3. 복합셀 식별 (분해 대상)

| 복합셀 | 원본 | 섞인 요소 | 분해 방향 |

| 소재그룹2 | `"무광코팅/유광코팅"` | 무광코팅 + 유광코팅 (2종 묶음) | 🔴 코팅 = mat_cd vs proc_cd (BATCH-3 컨펌) |
| 소재그룹3 | `"투명/홀로그램"` | 투명 + 홀로그램 (2 점착지) | 점착지 소재(mat_cd) |
| 구성요소 | `T3 "종이+인쇄+커팅"` | 종이 + 인쇄 + 커팅 (3 구성요소) | 공식 구성요소(formula_components) |

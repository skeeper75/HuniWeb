# golden-cases-foil.md — 박류(foil) 가격 골든 케이스 (검증가 재현 대상)

> hpe-engine-designer 산출 · 2026-06-30 · 검증가(hpe-validator·codex)가 설계 공식으로 재계산해 대조할 골든값.
> 권위[HARD]: 가격표 박(대형/소형) 시트 verbatim · 날조 0 · 모든 값에 출처 셀 명기.
> 박 모델: 박_총액 = 동판비(1회 고정) + 박가공비(면적→등급 × 수량구간·작업 1건 고정금액). ★[REV2] 박_총액은 **본체 공식 formula_components 직접 합산**(addon 아님·명함박/접지카드 라이브 작동 패턴·design-decisions-foil-rev2.md).
> ★ **골든값 불변(validator E6 PASS·8/8 권위 verbatim 일치)** — REV2 보정은 바인딩 메커니즘(addon→본체 합산)·근거 정정만이며, 골든 산술값은 권위 셀 합산으로 변동 없음.

## 0. 계산 절차 (각 케이스 공통)

```
1. 박색상 → {일반박 | 특수박} 그룹 판정 (시트별 매핑 §2-3 engine-design)
2. 시트 판정 (상품별 고정: 명함=소형·접지카드/책자=대형 · §6 CONFIRM)
3. 동판비 lookup:
     대형 → B01[가로][세로] 면적매트릭스 (1회·×qty 0)
     소형 → 5,000 고정
4. 박가공비 lookup:
     등급 = 면적→등급격자[가로][세로] → A~E  (앱 계산·off-grid=각 축 ceiling)
     단가 = 수량별가격표[등급][수량구간 하한]  (작업 1건 고정금액·×qty 0)
5. 박_총액 = 동판비 + 박가공비
```

★ 동판비·박가공비 둘 다 비-수량비례 → **이중 ×qty 폭발 0**(돈크리티컬 가드·★[REV2] 본체 공식 합산 경로에서만 성립·addon 경로는 ×qty 폭발이라 부결). 박_총액은 주문수량과 무관하게 "그 수량구간의 작업 1건 금액". 박 미선택 시 proc_cd no_match→0(pricing.py:576).

## 케이스 G-F4 — 소형·일반박·E등급 (명함박 collapse 비교·1,000원 갭 G6)

|---|---|

| 박영역 | 가로40mm × 세로80mm (명함 풀사이즈급) |
| 수량 | 1,000 |

**계산:**
- 동판비: 소형 고정 = **5,000** (small-l1.csv B01 B3·verbatim)
- 등급: 소형 면적격자 40×80 = **E** (engine-design §3-2·small-l1.csv B02 row12 F열)

★ **G6 갭 검증[HARD]**: 라이브 명함박 COMP_NAMECARD_FOIL_S1_STD 1000매=**63,000**. 권위 박가공비(E등급 1000)=**64,000** + 동판비 5,000 = 69,000. 명함박이 63,000인 것은 (가) 박가공비를 64,000이 아닌 다른 값으로 적재 또는 (나) 동판비 미합산. **답습 금지 — 7상품 설계는 권위 verbatim(64,000+5,000=69,000)으로**. 명함박 결함은 검증/C트랙 별도(Q-FOIL-G6).

## 검증 체크리스트 (validator·codex 재현 시)

| 검증축 | 케이스 | 기대 |
|---|---|---|
| 동판비 ×qty 0 | 전 케이스 | 수량 무관 동판비 고정(G-F1·G-F6 같은 90×90→18,000) |
| 박가공비 작업1건 고정·×qty 0 | 전 케이스 | 단가표 lookup값 = 그 수량 전체 총액(개당가 아님) |
| 일반/특수 차등 | G-F1 vs G-F2 | 박가공비 120,000 vs 150,000(동판비 동일) |
| 일반/특수 시트차이 | G-F8 | 먹유광=소형 일반/대형 특수(그룹 매핑 정합) |
| 면적→등급 환산 | G-F6 off-grid | 75×85→ceiling 90×90→C등급 |
| 동판비 면적매트릭스 | G-F3(11,000)·G-F7(64,000) | 대형 가로×세로 직접단가 |

| G6 명함박 갭 | G-F4 | 권위 64,000+5,000=69,000 (명함박 63,000≠ — 답습 금지) |

### 16.2 의사코드 본체

```
function calculateQuote(input):
  // 1. 상품 분류
  category = productCode.slice(0, 3)    // "001", "002", ...
  priceModel = getPriceModel(productCode)
              // PriceTable3D | SizeMatrix2D | FixedUnit | TieredDiscount

  // 2. 기본 단가
  switch (priceModel.type):
    case 'PriceTable3D':
      // 디지털인쇄·코팅·명함 등
      row = lookupQuantityRow(priceModel.table, input.quantity, policy='nextStep')
      col = lookupInkSide(priceModel.table, input.inkType, input.side)
      unitPrice = row[col]
      basePrice = unitPrice * input.quantity
    case 'SizeMatrix2D':
      // 포스터·아크릴
      (w, h) = normalizeSize(input.size)
      basePrice = priceModel.lookup(w, h) // 매트릭스 셀
    case 'FixedUnit':
      // 타투스티커 등
      basePrice = priceModel.unitPrice * ceil(input.quantity / priceModel.step)
    case 'TieredDiscount':
      // 굿즈/파우치/말랑
      basePrice = priceModel.unitPrice * input.quantity

  // 3. 후가공 합산
  finishPrice = 0
  for f in input.finish:
    rule = getFinishRule(f, input.quantity, input.side)
    finishPrice += rule.compute()  // 합가 또는 단가별

  // 4. 옵션 (아크릴 등)
  optionPrice = sum(getOptionPrice(o) for o in input.options)

  // 5. 제본
  bindingPrice = input.binding
    ? lookupBindingPrice(input.binding, input.quantity)
    : 0

  // 6. 수량 할인 (해당 상품군만)
  quantityDiscount = 0
  if priceModel.hasTierDiscount:
    rate = tierRate(input.quantity, priceModel.tierTable)
    quantityDiscount = -(basePrice + optionPrice) * rate

  // 7. 부분 합계
  subtotal = basePrice + finishPrice + optionPrice + bindingPrice + quantityDiscount

  // 8. 배송비
  deliveryFee = computeDelivery(
    subtotal, input.delivery, input.productCode
  )
  // 무료배송 기준 충족 시 0
  // 배너(005-0001~005-0010)는 무료배송 제외
  // 제주: +5000, 도서산간: +3000~10000
  // 혼합 주문: 최고값 1건 적용

  // 9. VAT 분리 (표시용)
  vatAmount = round(subtotal * 10 / 110)

  // 10. 최종
  total = subtotal + deliveryFee

  return { basePrice, finishPrice, optionPrice, bindingPrice,
           quantityDiscount, subtotal, vatAmount, deliveryFee, total, breakdown }
```

### 16.3 보간 정책 (수량 구간)

```
function lookupQuantityRow(table, qty, policy):
  rows = table.rows.filter(r => r.qtyThreshold >= qty)
  if rows is empty:
    return table.lastRow  // sentinel 1000000 행
  switch policy:
    case 'nextStep':  return rows[0]  // 권장(D-PM-10): 다음 큰 구간 단가
    case 'sameStep':  return reverse(table.rows).filter(r => r.qtyThreshold <= qty)[0]
    case 'linear':    interpolate(prev, next, qty)
```

### 16.4 사이즈 매트릭스 보간 (포스터)

```
function lookupSizeMatrix2D(matrix, w, h):
  // 1. 정확히 일치하는 셀이 있으면 그대로 반환
  if matrix[w][h]: return matrix[w][h]
  // 2. 둘 다 매트릭스 범위 내일 때: bilinear
  (w1, w2) = bracketCols(matrix, w)
  (h1, h2) = bracketRows(matrix, h)
  return bilinearInterp(matrix, w1, w2, h1, h2, w, h)
  // 3. 범위 초과 시 면적 비례 외삽 (단가/m² × w × h / 10⁶)
```

# OPTION vs TEMPLATE 판별 가이드 (CPQ 상품군 확장용)

> 작성 2026-06-09 · 실사(silsa)·캘린더(calendar) 시트 실데이터로 도출 · round-6 widening 참조 권위.
> 권위 입력: `06_extract/silsa-l1.csv`·`calendar-l1.csv`(엑셀 명시값) · 라이브 `t_prd_templates`(봉투 SKU 9행) · `cpq-design.md`(시스템 경계).
> 설명=한국어, 식별자/컬럼=English.

## 판별 기준 (한 문장)
**"주문 시 이것이 *별도로 생산·포장·배송되는 독립 상품*인가?"**
- 예 → **TEMPLATE** (별도 주문라인 ADDON, 자기 prd_cd·size·qty / `t_prd_templates`+`t_prd_template_selections`)
- 아니오, 이 상품을 *만드는 방식*이거나 *이 상품에 붙는 것* → **OPTION** (자재·공정·사이즈로 환원, 같은 주문라인 / `option_groups`→`options`→`option_items`)

**엑셀 신호:** 컬럼명 `추가상품(옵션)` = TEMPLATE(값이 다른 상품). `사이즈/소재/인쇄/가공/코팅/별색/포장/장수` = OPTION(이 상품의 속성).

## 실사(silsa) 분류 — 29상품
| 엑셀 컬럼 | 분류 | 근거 |
|---|---|---|
| 사이즈(필수) 33종(+사용자입력) | OPTION (size 택1) | 현수막 크기 |
| 소재(필수) 19종 | OPTION (material) | 현수막 소재 |
| 화이트별색(옵션)·코팅(옵션) | OPTION (인쇄/후가공) | 인쇄·코팅 방식 |
| 가공(옵션) 20종(봉미싱·타공·열재단·양면테입…) | OPTION 그룹 "가공" (자재+공정 BUNDLE) | 현수막 가공 |
| 추가(옵션) 13종 | **혼합** | 끈·큐방·각목+끈=OPTION(부착 자재) / 배너거치대·우드행거·거치대=TEMPLATE 후보(별도 상품) |

> 같은 "추가" 컬럼이 끈=option(붙는 자재) vs 배너거치대=template(별도 SKU)로 갈림 — 별도 생산·배송 여부가 판별자.

## option/template 사용 경우 (사례 모음)
**OPTION:**
1. 택1 사양 — 코팅(없음/무광/유광)·캘린더 인쇄(양면/단면)
2. 자재+공정 BUNDLE — silsa 봉미싱(실+봉제)·캘린더 고리형트윈링제본(링+제본)
3. 수량/페이지 축 — 캘린더 장수
4. 조건부 → 제약 — 캘린더 링칼라(고리형트윈링제본 선택시만) = OPTION + CONSTRAINT(JSONLogic)
5. 붙는 액세서리 자재 — silsa 끈·큐방·각목

**TEMPLATE:**
1. 고정 구성 별도상품 — 캘린더봉투 240x230mm 10장(size·qty 확정 SKU)
2. 본체연동 동적 별도상품 — 캘린더봉투 ★사이즈선택:220x145(본체 size 연동, 설계 GAP-B)
3. 독립 액세서리 상품 — 실사 배너거치대·우드행거(별도 생산·배송)

**경계 신호:** "★사이즈선택"=본체연동 template / "포함"=대개 option(한 몸) / "추가"=대개 template(별도) / 최종 판별=별도 생산·배송 독립 상품 여부.

## widening 적용 메모
- 캘린더 적재 시: 캘린더가공=OPTION 그룹(BUNDLE)·링칼라=조건부 제약·**캘린더봉투=TEMPLATE**(base 봉투 prd_cd + size selection). 봉투 base 상품(OPP/카드/트레싱지)은 라이브 PRD_000001/002/281/282/283 실재.
- 실사 추가의 거치대/행거류는 상품군별로 option/template 재판별 필요(일반현수막=끈/큐방/각목 option만, 행잉포스터/족자=우드봉/행거 — template 후보).

### 3-4. 3분류 집계

| 분류 | 행 수 | 비고 |
|------|:--:|------|

## 4. 적재·재배선·논리삭제 순서 (FK 위상 · [HARD])

```
[STAGE 1 — 기초데이터 축 신규 적재] (base_codes 의존·선존재)
  ② siz 신규 행 append        ── 형상/용량/사이즈/방향/규격 (경로 Y: 04_사이즈정보 말미)
  ⑤(bundle) product_bundle_qtys ── 구수 1~4구·N개1팩·100개
  ③(print_side) product_print_options ── 단/양면
       │ (가격사슬 siz_cd CASCADE 대비 — 신규 siz만 안전, 기존 삭제 금지)
       ▼
[STAGE 2 — CPQ 옵션 레이어 적재] (round-6 · webadmin 경로 부재 → dbm-cpq-option-mapping)
  option_groups(택1·본체색)
    → options(색 선택지)
      → option_items (색=자유옵션 ref_dim 없음 / 또는 siz·bundle 참조 시 OPT_REF_DIM.01/.05)
       │ fn_chk_opt_item_ref: siz 참조면 STAGE1 siz 선존재 필수
       ▼
[STAGE 3 — product_materials 재배선] (84상품)
  색/형상/구수/용량/인쇄면 link 제거
    → product_sizes(형상/용량/사이즈) · product_bundle_qtys(구수) · product_print_options(인쇄면) 연결
  소재/부속 link 유지 (USAGE.07 본체·부속)
       │ (재배선 완료 = 본체 자재 연결 무손실 확인 후에만 다음)
       ▼
[STAGE 4 — 오염 자재행 논리삭제] (마지막 · hard-delete 금지)
  색/형상/구수/용량/인쇄면 자재행 use_yn='N'
    (경로 Y: 05_자재정보 use_yn 컬럼 'N'·행 보존·surrogate 보존)
  AX-1 잉크색 7행 = 컨펌까지 동결(use_yn 미변경)
```

> **위반 시 결과:** STAGE 4를 STAGE 1~3 앞에 두면 → 144 USAGE.07 link 끊김 → 본체 자재 소실 + CPQ/siz 미적재로 옵션 부재. **반드시 적재→재배선→삭제 순.** 가격사슬은 cp 0참조라 안전하나, 재배선 미선행 시 본체 BOM 손실.

## 3. 제안하지 않은 것 (정직 표기 — dodge 금지)

- **goods-pouch 형상 siz_cd(K-3 14건):** round-3 goods-pouch 시트는 **구체 치수 행을 산출하지 않았다**(load-spec §1: WxH 보유 8상품 기적재 완료·비치수 47상품 BLOCKED-D-1, 마스터 모델링 미정). HANDOFF K-3는 "siz_cd 신설 14건"을 말하나, 그 14건의 **구체 치수가 어느 산출물에도 없다**. 추정으로 14행을 발명할 수 없으므로 **코드선적재 미제안 → GAP으로 분리**(`blocked-and-gaps.md §4`). sticker 066(치수 실재)과 달리 goods-pouch는 치수 자체가 미확정.
- **sticker 058~062 형상 enum siz_cd:** 축 귀속 미확정(size 보조 vs plate 메타 vs 신규테이블, D-2 컨펌 선행) → 미제안.
- **058~062 / 066 반칼↔스완 코드 정정:** 후니 내부 결정(D-4) → 미제안.

# 가격정보 분리 목록 (round-2 이연) — 추출 보존, 정합검증 제외

[핵심 순서 원칙] 엔티티 2축 = 상품정보 / 가격정보. 본 토대는 **상품정보를 먼저 정립**한다.
단가·연당가·가격 컬럼은 **무손실로 추출·보존**하되 "가격정보" 라벨로 분리해 **round-2(가격엔진 t_prc_*)로 이연**한다.
→ 본 목록의 컬럼들은 product-info-foundation의 정합검증(9속성) 대상에서 **제외**된다(버리지 않음, 분리 보관).

추출 위치: 각 `<slug>-l1.csv`에 컬럼 그대로 존재(삭제 안 함). meta CSV `axis=price` 라벨(판걸이수·IMPORT).

## 수식 파생 가격 (보존)

- silsa S열 = `=SUM(R...)*1.1` VAT 포함가 파생 **58건** — `is_formula=true`+원본 수식 + data_only 값 둘 다 보존(meta). round-2가 파생관계 활용.

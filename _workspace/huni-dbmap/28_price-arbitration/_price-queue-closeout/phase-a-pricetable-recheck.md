## 1. MATGROUP — 명함 033 STD 소재군 동일가 검증 → **GO**

### 1-1. 가격표 실측 증거 (시트 `명함포토카드` B01 스탠다드명함)

```
A1='스탠다드명함'
B2='단면'                                         D2='양면'
A3='소재 / 제작수량'
  B3='백모조220 / 아트250 / 스노우250'   C3='아트300 / 스노우300'
  D3='백모조220 / 아트250 / 스노우250'   E3='아트300 / 스노우300'
A4=100   B4=3500   C4=3800   D4=4500   E4=4800
```

- **헤더가 소재를 "/" 로 묶어 한 가격열에 명시** = 가격표 저자가 의도적으로 같은 가격의 소재를 그룹화.
  - **3,500군(단면 B4)** = 백모조220 · 아트250 · 스노우250 (3소재 동일가, 헤더 B3 권위).
  - **3,800군(단면 C4)** = 아트300 · 스노우300 (2소재 동일가, 헤더 C3 권위).
  - 양면도 동형(D4=4,500군 / E4=4,800군).
- 큐 §2-c·§4-3-4의 소재군 묶음 주장(3,500군={074·081·091}·3,800군={082·092})이 **가격표 헤더 문자열로 정확히 입증**. 묶음 깨짐 없음 — 5소재가 단 2개 가격(3,500/3,800)으로 정확히 분기.

### 1-2. 라이브 교차 (오라클·권위 아님)

- 라이브 `COMP_NAMECARD_STD_S1` 단가행 mat_cd = **074·082 2개만**(대표가만 적재). 081/091/092 키 부재 → 033 5소재 중 3소재 룩업 단절(큐 실측 재확인). 단가값(3,500/3,800)은 가격표와 일치 = **값 정확·키 부분 누락**.

### 1-3. 판정 = **GO** (§2-c ⓐ 정당)

- 가격표가 "동일가 묶음"을 헤더로 명시하므로, **STD 단가행에 081/091/092 mat_cd 행을 대표가(3,500/3,800)로 복제**(§2-c ⓐ)는 **권위 정합**. 추측 적재 아님 — 가격표 헤더가 "이 소재들은 같은 가격"이라고 직접 규정.

### 1-4. 후속 영향
- **확정**: §2-c 권고 ⓐ를 가격표 권위로 **즉시 배선 가능 항목으로 승격**(큐 §5-1 MATGROUP 단가행 확장의 선행 컨펌 해소). round-16 `dbm-price-import-builder`에서 STD_S1/S2 단가행에 081/091/092 행 3개씩(단면·양면) 추가.

## 2. CUT_PERF_1H6 — 타공비 "합가" 라벨 확인 → **조정 (D-1b 그룹①에서 제외)**

### 2-1. 가격표 실측 증거 (시트 `커팅타공` R42:R54)

가격표는 타공을 **두 블록으로 명시 분리**:

```
A42='타공 (단가)'   D42='헤더택 / 벽걸이캘린더'   F42='타공 (합가)'
A43='옵션/제작수량'  B43='1구(6mm)' C43='2구(6mm)'  F43='옵션/제작수량' G43='1구(6mm)' H43='2구(6mm)'

...      B(1구)=전구간 0.0                         ...      G/H=실가격(2000~51000)
```

### 2-2. 라이브 교차 (오라클)

- 라이브 `COMP_CUT_PERF_1H6`(use_dims=`["min_qty"]`·prc_typ_cd=`PRICE_TYPE.01`) = 23행 **전부 0.00** → 가격표 **A42 "타공(단가)" 블록(전부 0)**에 정확히 대응. **"합가" 블록 아님**.

### 2-3. 판정 = **조정** — D-1b 그룹①에서 **CUT_PERF_1H6 제외** (정정 대상 = 13 comp 유지)

> **[검증 정정 2026-06-15]** 큐 §1-2 각주의 "COMP_PP_* 12 + CUT_PERF_1H6 1 = 13"은 CORNER_ROUND를 12에 빠뜨린 라벨이다. 실제 정정 대상 = CREASE 3 + PERF 3 + VARTEXT 3 + VARIMG 3 + **CORNER_ROUND 1 = 13 comp**(전부 진성 누진 총액형). **CUT_PERF_1H6은 큐 §1-4 IN절에 애초 부재**(0원 placeholder)이므로 "빼는" 게 아니라 처음부터 대상 아님 → **정정 대상은 13 그대로**. dbm-validator V1 라이브 결판(Phase B §3-1 일치).

- 큐 §1-2 각주49의 "CUT_PERF_1H6 가격표 합가 라벨 직접 확인 권장"에 대한 답: **가격표 라벨은 "단가"이고 값이 전구간 0원**. 따라서:
  - CUT_PERF_1H6은 **D-1b(단가형 오적재) 대상이 아니다** — 가격표가 실제로 "단가" 블록으로 분류했고 값이 0이라 prc_typ가 .01이든 .03이든 **가격 기여 0(무영향)**. prc_typ 정정 불요.
  - 큐 §1-2 각주49가 이미 예견한 "단조성상 정정 영향 미미"가 가격표로 확정 — **0원 placeholder라 정정해도 무의미**.
- **진짜 타공 합가**는 `COMP_CUT_FULL_PERF_1H6/_2H6`이며, 이는 **그룹②(완칼·합가, §1-2 그룹②의 COMP_CUT_FULL_PERF_1H6/2H6)에 이미 포함**됨. 즉 큐가 그룹①(CUT_PERF_1H6)과 그룹②(CUT_FULL_PERF_1H6/2H6)에 타공 관련 comp를 양쪽에 둔 것은 **가격표 두 블록(단가 0원 / 합가 실가)을 정확히 반영**한 것 — 단 **그룹①의 CUT_PERF_1H6은 0원이라 정정 대상에서 빠져야** 한다.

### 2-4. 후속 영향
- **확정**: D-1b 그룹① 후가공 .04 정정 대상 = **13 comp**(CREASE 3·PERF 3·VARTEXT 3·VARIMG 3·CORNER_ROUND 1 — 전부 진성 누진 총액형). 큐 §1-4 멱등 UPDATE의 WHERE IN 목록(13개)이 곧 정정 대상이며, `COMP_CUT_PERF_1H6`은 애초 거기 없음(0원 placeholder·정정 무영향). **CORNER_ROUND는 0원이 아님**(1매2000→300매4000→1000매51000 누진)이라 13에 포함 필수.

## 3. WIRE-3 — 박 가격구조: 명함 종이포함 가정 + 042 공용 박가공비 시트 동일성 → **GO**

### 3-1. (a) 명함 박 단가가 종이포함 완제품가인지 → **확정: 종이포함 (가격표 명시)**

가격표 실측 (시트 `명함포토카드` B09 오리지널박명함 R57:R69):

```
A57='오리지널박명함'   F57='종이+동판+박가공비'         ← ★완제품가 명시
B58='단면'             D58='양면'
A59='소재/제작수량'  B59='금유광, 은유광, 먹유광, 청박, 적박, 동박'  C59='홀로그램 / 트윙클' ...
A60='기본가(아연판)'  B60=5000   D60=5000               ← 동판셋업 5,000
A61=200   B61=19200   C61=24800   D61=19200   E61=24800
A62=300   B62=24800   C62=33200 ...
A69=1000  B69=63000   C69=92000 ...
```

- **F57 = `종이+동판+박가공비`** — 가격표 저자가 명함 박 단가표가 **종이를 포함한 완제품 합가**임을 헤더에 직접 명시. 큐 §3-1·§3-2의 "명함 FOIL=종이포함 완제품가(합가)" 가정이 **가격표 권위로 입증**(추측 아님).
- **라이브 교차**: `COMP_NAMECARD_FOIL_S1_STD` 단가행 = 200매 19,200 / 300매 24,800 / … / 1000매 63,000 → 가격표 B61~B69(일반박 단면)와 **byte 일치**. comp_typ=`.06`(완제품가)·SETUP=5,000(`.05`). → 명함 박 단가=종이+동판+박가공 합가 확정.

### 3-2. (b) 042용 순수 박가공비가 명함과 같은 시트인지 → **별 시트 존재 (면적등급×수량+동판비)**

가격표 실측 (시트 `후가공_박(소형)` — 순수 박가공비 권위):

```
A1='동판비'        A2='수량' B2='80x40mm'    A3=1  B3=5000     ← 동판비 5,000 (수량무관 1회)  A4='*아연판'
A8='소형박 가공비 일반박 면적당가격 정의 (금유광, 은유광, 먹유광, 청박, 적박, 동박, 펄박)'
   A9='가로/세로' B9..F9=10/20/40/60/80mm   ← 면적등급 매트릭스(가로×세로 → A~E)
   A10='10mm' C10='A' D10='A' E10='B' F10='C'   A11='20mm' B11='A'…F11='D'   A12='40mm' …
H8='소형박 가공비 일반박 수량별 가격 (…)'
   H9='분류/수량' I9='A' J9='B'                  ← 등급(A/B…)×수량 → 박 가공비(순수)
   H10=200  I10=12200  J10=15000
   H11=300  I11=14300  J11=18500   …   H27=10000 I27=218000 J27=358000
A31/H31='소형박 가공비 특수박 …(백박, 홀로그램박, 트윙클박)'   ← 특수박군 별도 매트릭스+수량표
```

(`후가공_박(대형)` 시트도 동형: A1 동판비 매트릭스(11,000~64,000) + 면적등급 정의 + 일반박/특수박 등급×수량 가격.)

- 이 시트는 **순수 박 가공비**(종이 미포함)다. 헤더 명시 "박 가공비". 구조 = **동판비 + 면적등급(가로×세로→A~E) + 등급×수량 가격**.
- **명함 박(19,200)과 다른 값**: 소형박 순수 가공비 200매 A등급 = **12,200**(I10) ≠ 명함 박 200매 = **19,200**(종이+동판+박가공 완제품가·F57). 차이 7,000 = **종이분 중심**(동판 5,000은 명함 완제품가 F57에 이미 포함됐는지 단가표 구조에 의존 → 동판은 양쪽 공통 가산항으로 보고 차액 본질은 종이). → **명함 FOIL은 042에 그대로 못 씀(종이 이중계상 확정)**. ※ 정확한 종이분 금액·명함 박 단가표의 동판 포함 여부는 042 박 단가행 적재 + 명함 박 단가표 구조 확정 후 검증(WIRE-3b 동반).
- **라이브 교차**: 소형박 순수 가공비(12,200/14,300/15,000/18,500…)는 라이브 `t_prc_component_prices` 어디에도 **부재**(SELECT unit_price IN (12200,15000,14300,18500) → 명함/제본/포스터 등 무관 행만). 즉 **박 가공비 시트(소형/대형)는 라이브 미적재** — 라이브 박 comp는 명함전용 6개(COMP_NAMECARD_FOIL_*)만.
- **042 라이브 교차**: PRD_000042(프리미엄 쿠폰/상품권)의 PRF_DGP_A 배선 comp = 인쇄·용지(COMP_PAPER)·코팅·오시·미싱·가변뿐, **박 comp 0개**(string_agg 실측). → 042 박 add-on은 가격 comp 부재(신설 필요) 확정.

### 3-3. 판정 = **GO** (§3 권고 ⓐ 권위 확보)

- 큐 §3 권고 ⓐ(042=종이별도 공용 박가공비 comp 신설 + 명함=기존 FOIL 배선)가 **가격표로 뒷받침됨**:
  - 명함 박(B09)=종이포함 완제품가 → **042 단순 재사용 불가**(종이 이중계상). ⓑ 기각 정당.
  - 042용 순수 박가공비 권위 = `후가공_박(소형/대형)` 시트(면적등급×수량 + 동판비 5,000). 명함 박과 **권위 시트는 다르나**(명함=명함포토카드 B09 완제품가 / 042=후가공_박 시트 순수가공비), **둘 다 동일 박 도메인의 가격표 안에 존재** → 공용 박가공비 comp(`COMP_FOIL_AREA_<등급>` 또는 색군별 + `COMP_FOIL_SETUP`)를 후가공_박 시트 권위로 신설하면 042 add-on 의미 정합.
- **단 정밀화(큐 §3 ⓐ보다 한 단계 구체화)**: 042 박 시트가 **소형인지 대형인지**는 042 상품 박 사이즈(쿠폰/상품권 박 면적)에 따라 결정 — 후가공_박(소형)은 면적등급 가로 10~80mm·세로 10~40mm(작은 박), 대형은 30~170mm. 상품권 박이 작은 로고/문양이면 소형, 큰 면적이면 대형. **이 선택은 042 박 디자인 면적에 달림 → `[CONFIRM:WIRE-3b]`(042 박이 소형/대형 어느 시트 권위인지) 추가**. 가격표는 두 시트 모두 면적등급×수량 동형이라 **구조는 공유·단가표만 시트 선택**.

### 3-4. 후속 영향
- **확정**: §3 권고 ⓐ를 가격표 권위로 GO. 명함 031 박 = **기존 FOIL(종이포함 .06) 배선만**(comp 신설 불요). 042 박 = **후가공_박 시트 권위로 순수 박가공비 comp 신설 + 동판셋업 5,000 + 배선**.
- **확정(돈 오류 방지)**: ⓑ(명함 FOIL 042 재사용)는 **종이 이중계상 = 명백한 과대청구**로 가격표 입증 → 기각 확정.
- **변경(큐 §3 정밀화)**: 042 박은 면적등급(A~E) 차원이 필수(순수 박가공비가 등급×수량 2D) → `GAP-PARAM`(§4 박 면적등급 보존처)·`C36-FOILSIZE`(박 크기 차원)와 **동반 결정**. 명함 박은 완제품가라 면적등급이 단가표에 이미 흡수돼(명함 박은 수량 1D) 등급 차원 불요 — **042만 등급 차원 신설**.
- **신규 컨펌**: `[CONFIRM:WIRE-3b]` — 042(쿠폰/상품권) 박이 후가공_박(소형) vs (대형) 어느 시트 권위인지(박 디자인 면적). 실무진/디자인 확인.

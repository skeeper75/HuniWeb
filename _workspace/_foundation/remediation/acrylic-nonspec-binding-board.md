# 자유치수(nonspec='Y') 아크릴 11종 바인딩 점검 보드

작성 2026-06-27 (main 라이브 실측·읽기전용). 목적 = 코드버그(siz_cd×면적) 무관한 자유치수 아크릴을 안전 바인딩 가능한지 점검.

## 결론(요약)
- ★**마스터 §2-B "투명3T 바인딩 한 줄로 견적 가능"은 부정확.** 공유 단가표(COMP_ACRYL_CLEAR3T·MAT_043)가 **불규칙하게 듬성**(113행, 가로×세로 매트릭스가 듬성). 자유치수라 사용자가 표에 없는 조합(예 70×30) 입력 시 **조용히 0원**(no_tier_row). → 바인딩 전 **격자 보강(densify)** 선행 필요. 빠른 성과 아님.
- 부속자재(핀·자석·집게·헤어끈·고리)는 가격행 0 = 가격 무관(BOM 구성품). 몸체 MAT_043(투명3mm)이 유일 가격 결정.
- 자재 선택 옵션 없음(147~156·164) → 몸체 고정. 146 작동 선례도 동일 모델.

## 단가표 격자 실측 (COMP_ACRYL_CLEAR3T · MAT_043 · 113행)
- 가로 티어: 20,30,40,50,60,70,80,90,100,120,140,160,180,200 (14종)
- 직사각(가로≠세로) 99행 + 정사각 14행. **풀매트릭스(14×14=196) 아님 = 듬성**.
- 예: 가로80 행의 세로 = {30,50,80,90,120,180,200} (60·70·100·140·160 없음). 70×30 = 행 없음.

## 다음 선택지
1. **격자 보강 트랙**: 권위(아크릴 시트 면적가)에서 빠진 (가로×세로) 조합 단가행을 채워 표를 촘촘히 → 그 후 9종 BIND. 권위가 듬성한지(=원래 이산 사이즈만 판매) vs 적재누락인지 먼저 확정 필요.
2. **153 골드실버 MINT**: 권위 골드/실버 단가 전개(독립·코드 무관).
3. **보류**: 격자 미해소 시 9종 바인딩하면 직사각·off-격자에서 조용한 0원 = 돈결함. 바인딩 금지.

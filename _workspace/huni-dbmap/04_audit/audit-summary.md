## §2 판정 그룹

### GO 3 — 사이즈 · 인쇄옵션 · 페이지룰
- 공통상품 거의 전건 정합, 결손 0~3건 소규모. **신뢰 기반 양호**. 사이즈는 A1 3건 적재만 잔여(조건부 GO), 인쇄옵션은 별색 분기로 거짓 MISSING 회피 후 91/91 정합, 페이지룰은 11/11 완전 정합.

### MAJOR 4 — 자재 · 공정 · 묶음수 · 추가상품
- **마스터/코드/FK는 건전하나 상품별 연결 적재가 대량 결손**. 거짓 정합 위험은 없음(결손이 명확히 드러남). 적재 결손 성격:
  - 자재: 별도설정 상품의 IMPORT 실자재 미적재(15).
  - 공정: 별색공정 9 + 형압공정 3 미적재(12).
  - 묶음수: 엑셀 22상품 묶음값 대비 DB 1상품(4행)뿐.
  - 추가상품: 실addon 7 미적재(+ 가격전용 4 오분류).

### CONDITIONAL 1 — 판형사이즈
- BLOCK 아님. **속성 정의 불일치**(엑셀=전지 vs DB=작업사이즈)로 행대조 무의미. DB plate는 작업사이즈축으로 내부 정합. **의미축 확정 선행** 후 진행.

### CONFIRM 1 — 공정택일그룹
- **엑셀 권위 원천 부재**(택일그룹 직접 컬럼 없음, 도출 추론값). 추론 대조 금지 → DB 13행 내부정합(sel_typ_cd 유효·일관)만 기술. 정합 입증 불가, 후니 확인 필요.

## §3 교차 이슈 (속성 경계 결손 — 단일 속성 시야로 안 보이는 것)

### ① 별색 연계 결손 (공정 ↔ 인쇄옵션)
- 공정 MISSING 9(별색공정) + 인쇄옵션 별색전용 2(화이트인쇄명함/엽서) = **동일 근원**. 별색이 인쇄옵션·공정 **양쪽에서 미적재**. → **PROC_000008(화이트)/PROC_000009(클리어) 적재 1건으로 양 속성 동시 해소**. 별색=공정 분기(round-2)가 옳음을 양 속성 교차로 재확인.

### ② 자재 별도설정 ↔ 출력소재(IMPORT)
- `*별도설정`은 플레이스홀더 아님 — IMPORT 시트가 실자재(종이 7~47종/상품) 정의. DB가 IMPORT 종이를 미적재(자재 MAJOR의 본질). EXTRA 60은 IMPORT 종이가 아닌 굿즈 사이즈변형 → IMPORT 해소가 EXTRA를 줄이는 게 아니라 MISSING을 드러냄(이전 단정 supersede).

### ③ 판형 의미축 (전지 vs 작업사이즈)
- `t_prd_product_plate_sizes`가 전지(엑셀)와 다른 작업사이즈축을 담음. 전지 마스터는 t_siz에 부재. 판형 단독이 아닌 **사이즈 속성·판걸이수 브리지와 연동**해야 의미 확정.

### ④ 공정택일그룹 엑셀 원천 부재
- 엑셀에 택일그룹 직접 컬럼이 없어 9속성 중 유일하게 권위 대조 불가. 공정(4번)의 옵션-컬럼 구조에서 도출한 추론이므로 **공정 속성 검증과 묶어** 후니 의도 확인 필요.

## §4 후니 확인항목 통합 우선순위

### (A) 적재 결손 — 적재로 해소 (최우선)
1. **별색공정 적재**(교차 ① — 1건으로 공정 9 + 인쇄옵션 2 동시): 투명/별색 상품 PROC_000008/009.
2. **자재 IMPORT 실자재 적재**: 별도설정 8상품(엽서/명함/접지카드/중철·무선 내지표지) IMPORT 종이 + 트윈링/캘린더 7 부분보강.
3. **묶음수 적재**: 정정 18행(봉투/케이스/매트 9상품) + 엑셀 22(명함Y/조각/페이지 의미확정 후).
4. **형압공정 적재**: PUR책자·무선책자·형압명함 양각/음각.
5. **추가상품 실addon 7 적재**: 배너거치대·만년스탬프잉크·현수막끈·족자고리·아크릴스탠드.
6. **사이즈 A1 적재**: 프레임리스우드액자·포맥스보드·폼보드 594x841.
7. 인쇄옵션 접착투명포스터(단면), 공정 스프링노트 트윈링제본(MISMATCH 확인).

### (B) 의미 정의 확정 (적재 전 선행)
1. **판형 의미축**(교차 ③): `t_prd_product_plate_sizes`=작업사이즈 vs 전지 확정, 전지 마스터 적재 방식.
2. **공정택일그룹 도출근거**(교차 ④): 공정 옵션-컬럼=택일그룹 추론이 후니 의도와 일치하는지, 다중선택(SEL_TYPE.02) 필요 상품.
3. **묶음수 도메인**: 명함 `Y`(플래그 vs 수량), 조각→QTY_UNIT 매핑, 페이지 P vs page_rule 분기.
4. **추가상품 가격전용 4 재분류**: 방수/접착방수포스터·폼보드·지비츠★ → 옵션가격(round-2)으로 이동.

### (C) 모델링 세분화
1. **자재 두께/평량/투명도 소실**: DB `투명아크릴`(두께 미분리)·`PET`(평량 미분리) — 가격구분 필요 시 세분화. 백모조↔백색모조 명칭통일.
2. **output_paper_typ_cd 계열**: 전지 국전(.01)/46(.02) 구분 적재(현재 .03 기타만).
3. 복합자재 17건(`아트250+무광코팅`) 분해.

### (D) 모집단 (사전 정의)
1. **엑셀 보류/신규 7★**(버즈/에어팟케이스★·투명포스터★·링바인더 보류중 등): 등록 대상 확정 — 현재 DB 미등록은 정합 위반 아님.
2. **DB 반제품 28**(PRD_TYPE.02 내지/표지/면지): 엑셀 통합상품↔DB 분해 매핑 규칙.
3. 엑셀 잡음 1행(노랑배경 주석) 분모 제외.
4. EXTRA(DB 잉여) 전 속성 삭제금지, 출처 병기(사이즈 superset 3·자재 60·판형 150·묶음 1).

## §5 round-3 종합 판정

**마스터는 건전, 상품별 연결에 적재 결손 다수 (MAJOR 4).**

- **마스터·코드·FK 계층은 정합**(Phase 3 CONDITIONAL GO 확인 + Phase 4 재확인): 코드 11그룹 FK 전건 해소, addon FK PASS, 사이즈 표기차 브리지 100%, 자재 13확인필요 전부 표기차로 해소, 공정 마스터 깨끗한 계층.
- **9속성 연결**: GO 3(사이즈·인쇄옵션·페이지룰) / MAJOR 4(자재·공정·묶음수·추가상품 — 적재 결손) / CONDITIONAL 1(판형 의미축) / CONFIRM 1(공정택일그룹 원천부재).
- **거짓 정합 위험 없음**: 별색 공정축 분기·자재 IMPORT 해소·판형 의미축 분리로 결손이 명확히 드러남(자기 오탐 2건 정직 철회: 먼슬리 page_rule, 자재 EXTRA60=별도설정 가설).
- **결론**: DB↔엑셀 매핑은 **구조적으로 건전**하나 **상품별 옵션 적재가 미완**(MAJOR 4의 적재 결손 + 의미 정의 2건). 적재 결손은 대부분 적재 1단계로 해소 가능(특히 별색 1건으로 2속성 동시). **정정·적재는 별도 단계** — 본 하네스는 검증까지, DB 미적재 유지.

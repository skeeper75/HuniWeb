# 출력용지규격(전지/판형) 권위 전수 감사 — 디지털인쇄 (260701)

권위=인쇄상품가격표 **판걸이수 시트** + **출력소재(IMPORT) 시트**. 라이브 읽기전용 실측. 실 교정 인간 승인.
사용자 3과제: ① 전수 판형 등록 ② 판형 사이즈 재검토 ③ 썬캡 진단·교정.

## ① 전수 판형 등록 — 결과: 국4절 22상품 정상, 결함은 3절 클러스터
- **국4절 디지털인쇄 22상품**(016~047·284·285): 전부 `SIZ_000499` 바인딩 = 권위 일치 ✓. **추가 등록 불필요**. (※판걸이수 값 오차는 별건 C트랙.)
- **3절 4상품**:
  - 와이드벽걸이캘린더(112)·와이드접지리플렛(049)·지그재그엽서(030): 판형 `SIZ_000475`(330×660) 정상 바인딩이나 **가격공식 미바인딩(frm 없음)** → 미출시·견적불가(판형 아닌 공식 결손).
  - **썬캡(051): 판형이 국4절(SIZ_000499)로 오적재** — 권위는 3절 330×540. (③ 참조)

## ② 판형 사이즈 재검토 — 3절 정합 붕괴 3건
1. **SIZ_000077(300×625) = phantom 판형**: `COMP_PRINT_DIGITAL_S1`에 212행 적재됐으나 **어떤 상품도 바인딩 안 함**. 권위 3절=330×660(SIZ_000475·4×6전지 3절)인데 인쇄 3절 단가행은 300×625(국전 3절 추정)에 키잉 → **인쇄 3절 단가가 실사용 판형(SIZ_000475)에 매칭 안 됨**.
2. **COMP_PAPER 3절 용지비 전무**: 81행 전부 국4절(SIZ_000499). 출력소재 가격(3절) 컬럼은 존재하나 **라이브 미적재** → 전 3절 상품 용지비 0.
3. **3절 330×540(썬캡) 미등록** + 330×660(SIZ_000475)와 별도 치수. 썬캡만 다른 3절 사용(또는 540은 660 오기 가능성·권위 확인 필요).

→ 결론: 3절 출력 4상품은 **인쇄 3절 단가(판형 오키잉)·용지 3절 단가(미적재)** 양쪽 모두 결손이라 현재 데이터로 가격 불가. 국4절 라인과 달리 3절은 적재가 미완성.

## ③ 썬캡 진단·교정 — 데이터 단독 교정 불가(권위 미완성)
근본: 출력용지규격/판걸이수 매칭 미반영(엔진은 기하 fn_calc_pansu만). 썬캡 결손 4중:
1. 판형 국4절 오적재(작업 320×404 > 국4절 재단 306×457 → fn_calc_pansu=0 → 견적불가).
2. 권위 3절 330×540 **미등록**(SIZ_000475 330×660과도 불일치).
3. **아이보리 3절 용지단가 출력소재에 없음** + COMP_PAPER 3절 전무 → 용지비 적재 불가(날조 없이).
4. 인쇄 3절 단가는 SIZ_000077(300×625) phantom에만 → 썬캡 판형에 매칭 안 됨. + 가격공식 공란(미출시).
→ 썬캡은 **미출시 신규상품으로 권위(인쇄상품가격표) 자체가 3절 단가·판수 미완성**. 데이터 교정 불가, **권위 보강 선행**.

## 권장 조치 (라우팅)
- **C트랙(개발팀·핵심)**: 엔진이 판걸이수 시트 권위를 쓰도록 `fn_calc_pansu` → `t_siz_pansu` lookup 우선([[pansu-authority-fn-calc-pansu-260628]]·`CTRACK-fn-calc-pansu-authority-pansu.md`). 3절 상품은 기하계산이 0/오차라 이게 선결.
- **dbmap 적재(데이터·인간 승인)**: ① COMP_PAPER 3절 용지비 적재(출력소재 가격(3절) 5종 → SIZ_000475 등) ② 인쇄 3절 단가 phantom SIZ_000077 → 권위 판형(SIZ_000475)으로 재키잉 또는 정합 ③ 3절 330×540 등록 여부 권위 확인.
- **권위 확인(실무진)**: 썬캡 3절 330×540 vs 330×660·아이보리 3절 단가·썬캡 가격공식·3절 4상품 공식 바인딩(미출시 완성).
- **데이터 안전 교정 가능분 없음**(이번엔 COMMIT 없음) — 3절 단가가 권위에 불완전/미적재라 날조 없이 못 채움.

## 핵심 교훈
국4절(주력 22상품)은 판형 정상·인쇄/용지 단가 완비. **3절 라인 전체가 미완성**(인쇄 phantom 키잉·용지 미적재·일부 공식 결손)이고, 엔진이 출력용지규격/판걸이수 권위를 안 쓰는 C트랙이 그 위에 겹침. 썬캡은 그 미완성의 극단(견적 0).

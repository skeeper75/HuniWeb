## 1. 발견한 충돌 — 아크릴 핵심(round-3 G-AC 재확인 + round-11 신규)

엑셀 실측(L1 260610) × 07_domain × round-3 라이브 적재 3자 대조.

### 1-1. 인쇄사양=UV변형 ≠ print_side (G-AC-5 핵심 재확인)
- L1 `인쇄사양` = 배면양면(11)·풀빼다(7)·투명테두리★(6)·단면인쇄(1). **db-structure §159: UV(PROC_000002) `변형` enum과 정확 일치**(일반/배면양면/풀빼다/투명테두리/단면).
- **round-3 라이브: 아크릴키링·마그넷이 `t_prd_product_print_options.print_side`에 `배면양면`·`풀빼다`·`투명테두리`로 적재**(front/back 도수쌍 부여) = **의미축 오매칭**.
- **올바른 모델:** UV 변형 → PROC_000002 process + `변형` param. print_side는 실제 단/양면으로 정정. → **CONFIRM-AC-4.**

### 1-2. 두께 소실 (G-AC-3)
- L1 소재 = 투명아크릴 3mm(16)·8mm·1.5mm·골드/실버 3mm·접합·라미(10T). **두께가 다르면 다른 자재**(마스터 MAT_000042 1.5mm/043 3mm/044 8mm 존재).
- **round-3 라이브: 22상품 전부 MAT_000192(두께 없음) 일괄.** 코롯토(8mm)·미니파츠(1.5mm)도 192 = 두께 MISMATCH.
- **올바른 모델:** 코롯토/포카코롯토→044(8mm)·미니파츠→042(1.5mm)·나머지 3mm→043 두께 교정(또는 192+두께 속성축). → **CONFIRM-AC-2.**

### 1-3. 가공=부속자재+부착공정 2축 (G-AC-3/4)
- L1 가공(C20) 37종 = 고리/자석/핀/집게/바디/끈/와이어링/볼펜색/파츠. db-structure: MAT_TYPE.07 부속(MAT_000045~057) + PROC_000081 부착{대상}. **2축 정합**.
- **round-3 라이브: 맥세이프(부착)만 1행 — 나머지 부속 전무.**
- **Q5 운영원칙(실무진 확정):** 컬럼 필수/옵션 먼저 확인 → 옵션 용도 파악 후 귀속. 볼펜색/바디/파츠=본체색/디자인 variant일 수 있어 부속과 분기. → **CONFIRM-AC-3.**

### 1-4. 완칼 묵시 필수 vs over-reach (G-AC-1 + round-3 보정)
- 아크릴 굿즈는 도메인상 die-cut(완칼 PROC_000053)이 필수(모양대로 절단). 엑셀 명시 컬럼 없음(묵시).
- **round-3 라이브: 23상품 완칼 0건(G-AC-1 MISSING)** + **그러나 161/168/169(판아크릴·입체코롯토·입체블럭) 완칼 over-reach 보정 이력** — 단순 판/입체 블록은 모양 절단 불요.
- **올바른 모델:** 형상 굿즈(키링/뱃지/스탠드/코스터/카라비너)에만 완칼. 판/입체류 제외. → **CONFIRM-AC-1.**

### 1-5. 조각수 미적재 (G-AC-2)
- L1 조각수(C19) = 자유형스탠드 2~6조각·미니파츠 10조각(조합형만). benchmark §4: 후니 결정=조각수는 묶음수+공정param 둘 다.
- **round-3 라이브: 자유형스탠드/미니파츠 process 0행.**
- **Q8 확정:** 묶음수(권/세트 기준)≠조각수(판당개수+제한) 둘 다. → **CONFIRM-AC-5.**

**검증된 도메인 정합(07_domain 예측 ↔ 엑셀 실측 일치):**
- 인쇄사양 4값 = UV 변형 enum 정확 일치 ✅
- 두께 1.5/3/8mm = 마스터 MAT_000042/043/044 존재 ✅
- 가공 부속명 = 마스터 MAT_000045~057 존재 ✅
- 조각수 2~10 = PROC_000053/055 조각수 param ✅
- 폴더 UV인쇄/루아샵 = UV 라인(process-recipe §1) ✅

> **스티커 G-SK-2(형상 size 흡수)와 부분 유사하나 아크릴은 다름:** 아크릴 형상부기(코스터 원형/사각·카라비너 4형상)는 **완칼 모양 param**으로 귀속(스티커는 합판도무송 37형상이 size로 흡수돼 escalate). 아크릴 형상은 소수(코스터 2·카라비너 4)라 over-model 위험 낮음. 아크릴 고유 미확정=UV변형 적재위치(AC-4)·두께(AC-2)·가공 부속(AC-3)·완칼(AC-1).

## 2. 🔴/🟡 컨펌 질문 (인간 결정 대기)

### CONFIRM-AC-1 [🟡·핵심] 완칼(die-cut) 모델링 — 공정 vs 내포? (G-AC-1)
- **상황:** 아크릴 굿즈는 모양대로 절단(die-cut)이 도메인 필수이나 엑셀 명시 컬럼 없음. round-3 라이브 완칼 0건. 단 판아크릴/입체류는 over-reach 보정.
- **가설:** 형상 굿즈(키링/뱃지/스탠드/코스터/카라비너)에 PROC_000053 완칼 적재(+사용자입력/형상=`모양` param). 판아크릴(161)·입체코롯토(168)·입체블럭(169)은 단순 판/입체라 완칼 제외.
- **질문:** 아크릴 절단을 완칼 공정(PROC_000053)으로 모델링할까요(형상 굿즈만), 아니면 사이즈/가격에 내포해 공정행 없이 갈까요? 판/입체류 제외가 맞나요?

### CONFIRM-AC-2 [🟡·핵심] 두께 자재 — 별 mat_cd vs 192 단일+속성축? (G-AC-3)
- **상황:** 두께(1.5/3/8/10mm)가 다르면 다른 자재(마스터 042/043/044 존재). round-3 라이브 22상품 192 일괄=두께 소실. 코롯토(8mm)·미니파츠(1.5mm) 오매핑.
- **가설:** 코롯토/포카코롯토→044(8mm)·미니파츠→042(1.5mm)·나머지 3mm→043으로 두께 교정.
- **질문:** 두께를 별 mat_cd(042/043/044)로 구분 적재할까요, 192 단일 + 두께 속성축으로 둘까요?

### CONFIRM-AC-3 [🟡·핵심] 가공 부속 귀속 — 자재 vs 공정 vs 둘 다 vs variant? (G-AC-3/4·Q5)
- **상황:** 가공(C20) 37종 = 고리/자석/핀/집게/바디/끈/와이어링/볼펜색/파츠. db-structure는 부속자재(MAT_TYPE.07)+부착공정(PROC_000081) 2축 정합. round-3 라이브 맥세이프(부착)만 1행. 단 볼펜색/바디/파츠는 본체색/디자인 variant일 수 있음.
- **가설:** 부착 부속(고리/자석/핀/집게/끈/와이어링/맥세이프)=자재(.07)+부착공정 2축. 볼펜색/바디/파츠=본체색/디자인 variant(부속 아님).
- **질문:** Q5 운영원칙(컬럼옵션 확인 후 귀속)대로, 부착 부속은 자재+공정 2축으로, 볼펜색/바디/파츠는 variant로 분기할까요?

### CONFIRM-AC-4 [🟡] 인쇄사양 UV변형 적재 위치 — process vs print_side? (G-AC-5)
- **상황:** 인쇄사양(배면양면/풀빼다/투명테두리/단면)=UV `변형`이나 round-3 라이브 print_side에 오적재. 인쇄방식(PROC_000002~6) 272 전상품 미연결 글로벌 갭.
- **가설:** UV 변형을 PROC_000002 process + `변형` param으로, print_side는 실제 단/양면 정정.
- **질문:** UV 변형을 UV 공정(PROC_000002 `변형`)으로 옮길까요, print_side를 변형 슬롯으로 공식 인정할까요? (인쇄방식 미연결 글로벌 갭과 연동)

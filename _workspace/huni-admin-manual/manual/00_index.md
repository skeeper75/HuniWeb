# 후니 상품·가격 DB 관리자 운영자 매뉴얼

> **대상 독자:** 후니 상품·가격 데이터를 직접 등록·수정하는 운영자(비개발자)
> **시스템:** 후니 상품·가격 DB 관리자 (Django admin · Unfold 테마 · Railway PostgreSQL)
> **이 매뉴얼은** 화면을 보며 따라 하는 **업무 가이드**입니다. "무슨 일을 어떻게 끝내는가"에 답합니다.

---

## 이 매뉴얼을 읽는 법

- 처음이라면 **[00 시작하기](01_getting-started.md)** 부터 읽으세요. 로그인·화면 구조·모든 화면에 공통인 동작을 한 번에 익힙니다.
- 특정 작업을 바로 하려면 아래 **업무별 절차** 에서 해당 챕터로 가세요.
- 화면에 무슨 항목이 있는지 빠르게 확인하려면 **[09 화면 레퍼런스](09_screen-reference.md)** 를 보세요.
- 용어가 낯설면 **[10 부록 - 용어집](10_appendix.md)** 을, 저장이 안 되면 **[10 부록 - 문제해결](10_appendix.md)** 을 보세요.
- "이걸 하려면 어디로 가야 하지?" 싶을 땐 **[10 부록 - 작업→화면 역인덱스](10_appendix.md)** 를 보세요.

> 💡 화면 안의 라벨·버튼은 실제 화면 글자 그대로 따옴표("저장", "추가")로 표기합니다.
> 괄호 안의 영문(`prd_nm` 등)은 **시스템 항목 이름** 입니다. 개발자에게 문의할 때 그대로 전하면 됩니다.

## 전체 화면 한눈에 보기

이 시스템에는 두 종류의 화면이 있습니다.

1. **표준 관리 화면** (좌측 메뉴에서 바로 열림) — 목록을 보고, "추가" 버튼으로 한 건씩 등록·수정합니다. 상품·자재·사이즈·가격공식 등 **기준이 되는 데이터** 가 여기 있습니다.
2. **상품 뷰어** (특수 화면) — 한 상품을 펼쳐 그 상품의 사이즈·자재·공정·옵션·제약·SKU를 **한 화면에서** 관리합니다. 상품의 세부 구성은 거의 모두 여기서 다룹니다.

> ℹ️ **중요:** 일부 데이터(가격사슬 일부·등급별 할인율 등 8종)는 **이 관리자 화면에 나타나지 않습니다.** 데이터는 데이터베이스에만 있고 화면 편집이 불가능합니다. 자세한 목록은 [09 화면 레퍼런스](09_screen-reference.md) 의 "화면이 없는 데이터" 표를 보세요. 이 데이터를 고쳐야 하면 개발 담당자에게 요청하세요.

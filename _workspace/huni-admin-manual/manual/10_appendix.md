## 용어집

운영자가 화면에서 겪는 현상으로 풀어 쓴 용어입니다. 괄호 안은 시스템에서 쓰는 영문 이름(개발자 문의 시 사용).

| 용어 | 뜻 |
|------|-----|
| **필수 항목** (NOT NULL) | 라벨에 빨간 별표(`*`)가 붙은 칸. 비우면 저장되지 않습니다. |
| **선택 항목** | 별표가 없는 칸. 비워도 저장됩니다. |
| **자동채번** (자동 생성 코드) | 코드 칸을 비우면 저장할 때 `PRD_000123` 식으로 시스템이 코드를 자동으로 만듭니다("비우면 저장 시 자동 채번" 안내). |
| **마스터** | 모든 상품이 공유하는 기준 데이터(카테고리·자재·사이즈·도수·공정·기초코드). |
| **상품 뷰어** | 상품 하나를 펼쳐 사이즈·자재·옵션·제약·SKU를 한 화면에서 다루는 특수 화면. |
| **섹션** | 상품 뷰어에서 상품 상세를 이루는 9개 구성 카드(사이즈·자재·공정 등). |
| **드릴다운** | 옵션그룹 → 옵션 → 구성요소처럼 단계를 파고드는 편집(계층 편집). |
| **옵션그룹 / 옵션 / 구성요소** | 옵션의 3계층. 그룹=선택 묶음, 옵션=고객이 고르는 항목, 구성요소=그 옵션을 이루는 자재·공정. |
| **SKU (구성 템플릿)** | 옵션 조합을 미리 짜둔 판매 단위 상품. |
| **선택값** | SKU가 어떤 차원을 어떤 값으로 고정했는지를 나타내는 행. |
| **제약 규칙** | 옵션 선택을 제한하는 규칙(금지·필수동반). |
| **폼빌더** | 제약 규칙을 만드는 화면 도구. 드롭다운으로 짜면 내부 논리가 자동 생성됨. |
| **차원** (`ref_dim_cd`) | 사이즈·판형·자재·공정·묶음수·도수·셋트 같은 선택 축. |
| **사용여부** (`use_yn`) | Y(사용)/N(중지). N으로 바꾸면 다른 화면 선택지에서 사라집니다(데이터는 남음). |
| **논리삭제** (`del_yn`) | 실제로 지우지 않고 "삭제됨" 표시만 하여 목록에서 숨기는 삭제. 기록은 남습니다. |
| **표시순서** (`disp_seq`) | 목록·드롭다운에 보이는 순서. 비우면 맨 뒤로 자동 배치. |
| **트리 드롭다운** | 상위/하위가 들여쓰기로 구분되어 보이는 부모 선택 드롭다운(카테고리·자재·공정·기초코드). |
| **사용처 배지** | 그 행이 옵션·SKU 등에서 쓰이고 있다는 표시. 클릭하면 어디서 쓰이는지 보입니다. |
| **검증 미리보기** | 제약 폼빌더 우측에서 샘플 조합을 검사하는 테스트 도구(저장 안 함). |
| **선택 미리보기** | 옵션 편집 화면에서 고객 화면 모습을 미리 보는 참고 패널(저장 안 함). |
| **기초코드** | 상품유형·자재유형 등 드롭다운 선택지의 원천 코드. 그룹(`MAT_TYPE`)과 하위값(`MAT_TYPE.01`)으로 구성. |

## 자주 겪는 문제 (저장이 안 될 때)

저장 버튼을 눌렀는데 안 되거나 빨간 오류가 뜨면, 아래를 차례로 확인하세요.

| 증상 | 원인 | 해결 |
|------|------|------|
| 빨간 오류 + "이 필드는 필수입니다" | **필수 항목**(별표 `*`)을 비웠음 | 별표 항목을 모두 채우세요. 상품은 상품명·상품유형·비규격여부·파일업로드여부·에디터여부·사용여부가 필수. |
| `*_yn` 칸에서 저장 거부 | Y/N 외의 값을 넣었음 | **Y 또는 N만** 입력. |
| 할인 입력 시 저장 거부 | 할인율과 할인액을 **둘 다** 입력했음 | 둘 중 **하나만** 입력하세요. |
| 드롭다운에 원하는 코드/마스터가 없음 | 그 마스터가 아직 등록 안 됨, 또는 사용여부=N | 먼저 해당 [마스터](07_masters.md)에 등록하고 사용여부를 Y로 두세요. |
| 옵션 구성요소·SKU 선택값 저장 거부 | 참조하는 자재/공정이 그 상품에 등록 안 됨 | [상품 하위정보](03_product-sections.md)의 자재·공정 섹션에 먼저 추가하세요(무결성 검사). |
| 다차원 단가 저장 거부 | 같은 차원 조합(8키)이 이미 있음 | 기존 행을 찾아 수정하세요(중복 등록 불가). |
| 코드/마스터 삭제·중지 거부 | 다른 데이터가 그 코드를 쓰고 있음(사용 중) | 먼저 그 코드를 쓰는 곳을 정리한 뒤 삭제하세요. **[사용처](06_constraints.md)** 로 어디서 쓰는지 확인. |
| 상품 편집 화면에 사이즈·자재 칸이 없음 | 표준 편집 화면은 기본 정보만 다룸 | 상단 **"상품 뷰어" 링크** 로 이동해 세부 구성을 편집하세요. |
| 구성 템플릿(SKU) 행 클릭 시 다른 화면으로 넘어감 | SKU는 표준 폼이 아니라 SKU 선택값 화면으로 연결됨(정상) | [04 구성 템플릿(SKU)](05_sku-templates.md) 절차를 따르세요. |
| 직접 입력한 주소가 "Not Found"(404) | 그 데이터는 관리자 화면이 없음(DB 전용) | [09 화면 레퍼런스 - 화면이 없는 데이터](09_screen-reference.md) 참조. 개발 담당자에게 문의. |
| SKU 선택값 저장 후 경고 배너 | 그 조합이 제약 규칙을 위반함 | 저장은 됐지만 조합이 규칙에 맞는지 [제약 규칙](06_constraints.md)에서 확인하세요. |
| 로그인 직후 잠깐 "Not Found" | 알려진 표시 문제(세션은 정상) | 관리자 주소(`/admin/`)로 다시 들어가면 정상. |

> ⚠️ **이름·코드 표기 통일:** "인쇄면"(`print_side`)과 "출력파일유형"(`output_file_typ`)은 드롭다운이 아니라 **직접 입력** 칸입니다. 같은 뜻을 다르게 적으면("단면" vs "1면") 데이터가 갈립니다. 기존 표기를 따라 통일해 입력하세요.

## 작업 → 화면 역인덱스

"이걸 하려면 어디로 가야 하지?" 싶을 때 보세요.

| 하려는 일 | 가는 곳 | 챕터 |
|-----------|---------|------|
| 새 상품 등록 | 메뉴 "상품정보" → ⊕추가 | [01](02_product-register.md) |
| 상품 이름·유형 수정 | 메뉴 "상품정보" → 행 클릭 | [01](02_product-register.md) |
| 상품의 사이즈·자재·공정 등 세부 구성 편집 | "상품 뷰어" → 상품 → 섹션 "편집" | [02](03_product-sections.md) |
| 상품 인쇄면·도수 설정 | 상품 뷰어 → "도수/인쇄옵션" 편집 | [02 ③](03_product-sections.md) |
| 책자 페이지 수 규칙 | 상품 뷰어 → "페이지룰" 편집 | [02 ⑨](03_product-sections.md) |
| 상품 옵션(가공·코팅 등) 만들기 | 상품 뷰어 → "옵션그룹" 편집 | [03](04_options.md) |
| 옵션에 자재·공정 묶기 | 옵션 편집 화면 → 옵션 카드 "+ 구성요소 추가" | [03-2](04_options.md) |
| 미리 짜둔 조합 상품(SKU) 만들기 | 상품 뷰어 → "구성템플릿(SKU)" 편집 | [04](05_sku-templates.md) |
| 옵션 선택 제한(금지·필수동반) | 상품 뷰어 → "제약규칙" 편집 | [05-1](06_constraints.md) |
| 규칙 조합 테스트 | 제약 화면 → "검증 미리보기" | [05-1](06_constraints.md) |
| 어떤 값이 어디 쓰이는지 확인 | 세부 구성 행 → "사용처" 배지 | [05-2](06_constraints.md) |
| 전체 SKU 둘러보기 | 주소 `/admin/sku-catalog/` | [05-3](06_constraints.md) |
| 새 자재(종이·필름 등) 등록 | 메뉴 "자재정보" → ⊕추가 | [06-1](07_masters.md) |
| 새 사이즈(치수) 등록 | 메뉴 "사이즈정보" → ⊕추가 | [06-2](07_masters.md) |
| 새 공정 등록 | 메뉴 "공정정보" → ⊕추가 | [06-4](07_masters.md) |
| 상품 분류(카테고리) 정리 | 메뉴 "카테고리" → ⊕추가 | [06-5](07_masters.md) |
| 드롭다운 선택지 자체 추가 | 메뉴 "기초코드정보" → ⊕추가 | [06-6](07_masters.md) |
| 가격공식·구성요소·단가 관리 | 메뉴 "가격공식"/"가격구성요소"/"구성요소 다차원 단가" | [07](08_pricing.md) |
| 수량구간 할인표 제목 관리 | 메뉴 "수량구간할인" → ⊕추가 | [07-4](08_pricing.md) |
| 고객 관리 | 메뉴 "고객" → ⊕추가 | [07-5](08_pricing.md) |
| 가격이 안 맞아 연결을 고쳐야 함 | 화면 없음 — 개발 담당자 문의 | [09 C](09_screen-reference.md) |
| 할인 구간·할인율 상세 조정 | 화면 없음 — 개발 담당자 문의 | [09 C](09_screen-reference.md) |

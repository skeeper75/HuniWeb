## 0. 결론 (한 줄)

아크릴 = **면적매트릭스형 단일 동형 클래스**(본체 매트릭스 3종 = CLEAR3T·CLEAR1.5T·MIRROR3T로 자재분기, 가격공식·옵션구성 동형). **대표 = 아크릴키링(PRD_000146)** — 시트 실측 superset(사이즈 8·자재 3[투명3T 본체 + 은색/금색 고리 부속]·UV공정·CLEAR3T 매트릭스 참조·후가공 옵션·볼체인 addon). 미출시(use_yn=N 6 + L1 미등록 쉐이커/지비츠)는 §3에서 제외.

## 1. 면적매트릭스형 = 무엇이 동형인가 (계산공식집초안 권위)

토대 §5 면적매트릭스형 정의: `[가로][세로]` 면적 룩업 + ceiling. 아크릴 본체 가격은 **자재(두께·종류)별 면적매트릭스에서 (가로,세로) 좌표로 단가를 룩업**하고, 주문수량은 단가형 곱셈(`단가 × 수량`)으로만 작동(수량할인은 별도 t_dsc). 이 구조가 23 아크릴 상품에 동형으로 적용된다 — 차이는 어느 매트릭스(CLEAR3T/15T/MIRROR3T)를 참조하느냐 + 후가공 옵션 종류뿐.

**동형 핵심(엽서·명함과 다른 점):**
- **엽서(합산형)**: `Σ 구성요소`(인쇄비+코팅비+용지비+후가공비) — 수량은 출력매수 변환(`ceil(수량/판걸이수)`).
- **명함(고정가형)**: `[수량행][소재×면열]` 룩업 — 용지포함·×수량 안 함·단일 100매 구간.
- **아크릴(면적매트릭스형)**: `[가로][세로]` 룩업 — **수량축 자체가 없음**(min_qty=NULL)·단가형 ×수량·**off-grid=한 단계 큰 크기 가격(ceiling·런타임)**.

`siz_cd`가 가로×세로 면적조합 1개를 가리킨다(예 `20x20`→SIZ_000336·`30x40`→SIZ_000331). 라이브 실측 `use_dims=["siz_cd"]`·나머지 9차원 NULL(와일드카드). 도수("양면9도/단면7도 통용 단가")=clr 무관(NULL)·자재=구성요소명에 박힘(mat_cd 차원 미사용·디지털 별색 SPOT_* 패턴).

## 4. 대표 선정 — 시트 실측 superset (토대 §6-1·일반화 금지)

대표를 추상 원리로 박지 않는다. 아크릴 시트 + 라이브 실측으로 옵션·구성요소가 가장 완전한 주문가능 상품을 고른다.

**대표 = PRD_000146 아크릴키링.** 근거(라이브 실측):
- sizes 8(SIZ_000329 20x30 ~ SIZ_000011 50x50) = 본체 면적매트릭스 좌표 최다 노출 → 면적룩업 + off-grid 검증에 최적.
- materials 3(MAT_000043 아크릴 투명 3mm[USAGE.07] + MAT_000051 은색고리 + MAT_000052 금색고리) → 본체 자재 + 부속(고리) BUNDLE 구조 superset([[dbmap-option-material-process-bundle]]).
- proc PROC_000002 UV(평판인쇄) → 아크릴 단일 생산방식.
- CLEAR3T = 라이브 적재 최다(47행) 대표 매트릭스 → 가격 골든 대조 데이터 풍부.

> superset 이유(토대 §6-2): 키링을 부분집합 상품(마그넷=본체만·뱃지=사이즈 3)으로 안전 전파 가능. 최소 상품(포카키링 sizes 1)을 대표로 뽑으면 부속·사이즈 누락이 전파됨.

## ① 무손실 추출 방법론 (블록탐지 = 핵심 난도)

### 1-1. 블록 모델
- **블록** = 제목셀(블록타이틀)로 anchored 된 직사각 영역. `block_id`·`block_title`·`row_range`·`col_range` 부여.
- 한 시트에 여러 블록이 **세로 stacking**(코팅 국4절→3절) + **가로 동거**(인쇄후가공 가변텍스트|가변이미지) 둘 다.
- 블록 제목은 A열(또는 가로분할 컬럼)에 반복 등장(제본 "제본비"→"하드커버 제본비"→"캘린더 제본비").

### 1-2. 블록탐지 규칙 (extract_price_sheets.py `detect_blocks` 2패스)
1. **제목후보** = 비수치 문자열 AND `_heads_a_region`(같은 컬럼 아래 4행 내 band_key/수치/mm 등장).
2. **축값/밴드라벨 배제**: mm치수(`20mm`)·분류문자(`A~E`)·구간문자(`1~49`)·규격사이즈(`A3/A2/A1`)·
   band_key 라벨(`수량`/`제작수량`/`소재 / 제작수량`/`가로 / 세로`/`단면`/`양면`/`옵션`…)은 제목 아님.
3. **밴드헤더행 leaf 제거**: 제목후보 좌측에 band_key 라벨이 있으면 그 행은 헤더행 → 비좌측 문자열은
   밴드 leaf(옵션명/소재명)이지 제목 아님.
4. **헤더존 leaf 제거(2패스)**: 컬럼A 제목의 헤더존(제목행+1 ~ 첫 수치행-1) 안 비좌측 제목후보는
   다단헤더 leaf(흑백/칼라, 화이트(단면)+클리어)이므로 제거.
5. **경계**: 우경계 = 같은 제목행대(±1행) 우측 다음 제목 컬럼 −1(가로동거 분리). 행범위 = 컬럼 겹치는
   다음 세로 제목행 −1.

### 1-3. 밴드헤더 합성 (band_header_path)
- 머지셀을 풀어 각 leaf 컬럼의 밴드경로를 위→아래 헤더행 순서로 `>` join.
- 검증 예: 디지털인쇄비 D3 = `칼라(CMYK) > 단면`, 코팅 C5 = `무광코팅 > 양면`, 명함 C4 = `단면 > 아트300 / 스노우300`.
- **한계(문서화)**: 면적정의 2D class 매트릭스(foil 면적정의)는 행키/컬럼 모두 mm축이고 body가 class문자라
  band_path가 의미 없는 누적(`20mm > A > B`)을 낼 수 있음. 해당 블록은 "2D class 매트릭스"로 별도 해석(아래 ⑩⑬).

### 1-4. 무손실 보증 = 셀단위 long 산출
- `price-<slug>-l1.csv` = **전 nonempty 셀 1행1레코드**(sheet·block_id·block_title·row_seq·col·cell_ref·
  row_key·band_header_path·value·cell_meta_json). 블록탐지가 틀려도 **셀은 전부 살아있음**(안전망).
- `price-<slug>-l1-meta.csv` = 블록 인벤토리 + 셀메타(머지·숨김·수식·코멘트·★·배경/글자색).
- 9게이트(cell_count·nonempty_preserve·roundtrip·col/row_coverage·merge·formula·comment·hidden) 전 시트 PASS.

### 1. 디지털인쇄비 (`digital-print-price`) — 수량×도수×단/양면 밴드매트릭스

- **데이터영역**: r1–116 × cA–O. nonempty 1092셀. 머지 11. **2블록**.
- **블록 인벤토리**:
  | block | title | 행범위 | 컬럼 |
  |-------|-------|-------|------|
  | B01 | 디지털인쇄 출력비(국4절) | r1–60 | A–O |
  | B02 | 디지털인쇄 출력비(3절) | r61–116 | A–O |
- **밴드헤더 깊이 = 2단**: row2 = 외밴드 7종(`흑백(1도)`/`칼라(CMYK)`/`별색(화이트)`/`별색(클리어)`/`별색(핑크)`/`별색(금색)`/`별색(은색)`, 각 2컬럼 머지) × row3 = 내밴드(`단면`/`양면`). leaf 14컬럼(B~O).
- **수량축 정체**: A열 = `수량(국4절)`/`수량(3절)` = **출력매수**(print-sheet count, NOT 주문수량). 1~1400 비선형 구간.
- **특이점**:
  - **국4절/3절 2블록**은 같은 옵션축, 출력규격(판형)만 상이 → 규격 차원으로 분리.
  - 별색 5종(화이트/클리어/핑크/금색/은색)은 **별색=공정**(확정규칙①, clr_cd 아님). 단가는 별색인쇄비 comp.
  - 단/양면은 시트 단가 그대로(양면≠단면×2, 확정규칙②).
- **차원 귀속 후보**: 도수외밴드(흑백/칼라)→`clr_cd`(흑백=1도/칼라=CMYK), 별색→**공정**(clr 아님), 단/양면→상품옵션 또는 단면/양면 comp 분리, 출력매수→`min_qty`, 국4절/3절→`siz_cd`(출력규격).

### 2. 코팅 (`coating`) — 수량×코팅종×단/양면

- **데이터영역**: r1–56 × cA–E. nonempty 246. 머지 6. **2블록**(국4절 B01 r1–30, 3절 B02 r31–56).
- **밴드헤더 2단**: row2 = `무광코팅`/`유광코팅`(각 2컬럼 머지) × row3 = `단면`/`양면`. leaf 4컬럼(B~E).
- **수량축**: A열 `수량(국4절)`/`수량(3절)` = 출력매수. 1~5000 + **1000000 sentinel**(상한개방 마커, 확인 필요: 실상한 아닌 듯).
- **차원 후보**: 코팅종(무광/유광)→`mat_cd` 또는 공정, 단/양면→`coat_side_cnt`(1/2, 확정규칙②), 출력매수→`min_qty`, 국4절/3절→`siz_cd`.

### 3. 접지옵션 (`folding`) — 수량×접지단수

- **데이터영역**: r1–105 × cA–E. nonempty 446. 머지 3. **2블록**(카드접지 B01 r1–55, 리플렛접지 B02 r56–105).
- **밴드헤더**: row2 = `2단`/`3단`/`6단`(카드) — row3 = 접지옵션명(`2단가로접지 / 2단세로접지` 등). 제목 = `...(오시+접지)` → 합가성.
- **수량축**: A열 `접지옵션명/ 제작수량` 헤더 후 1~100000(sentinel). = 제작수량(주문수량 추정, 확인 필요).
- **특이점**: 제목에 `(오시+접지)` = 오시+접지 **합가**(확정규칙④ — 분해 금지). B01(카드: 2/3/6단) vs B02(리플렛: 반접지/3단…) 옵션집합 상이.
- **차원 후보**: 접지단수/옵션명→공정 또는 `mat_cd`, 제작수량→`min_qty`. 합가→상품단가 종속 가능(규칙④).

### 4. 인쇄후가공 (`post-process`) — 다블록 가로+세로, 전부 합가

- **데이터영역**: r1–57 × cA–I. nonempty 320. 머지 9. **5블록**(가로동거 포함).
- **블록 인벤토리**:
  | block | title | 행범위 | 컬럼 | 비고 |
  |-------|-------|-------|------|------|
  | B01 | 모서리 (귀돌이, 합가) | r1–11 | A–I | 직각/둥근모서리 |
  | B02 | 100장당 1000원씩 올립니다. | r12–14 | A–I | **증분 규칙 note**(데이터 아님) |
  | B03 | 오시 (합가) | r15–31 | A–I | A–D=오시, **F–I=미싱 (합가)** 가로동거 |
  | B04 | 가변(텍스트) | r32–57 | A–E | 1줄/2줄/3줄 |
  | B05 | 가변(이미지) | r32–57 | F–I | B04와 가로동거 |
- **밴드헤더 1단**: row2/row16/row33 = 옵션라벨(`직각모서리`/`1줄`·`2줄`·`3줄`/`1개`·`2개`·`3개`).
- **수량축**: A열 `제작수량` 1~5000. = 주문수량(제작수량) 추정.
- **특이점**: 전 블록 제목 **"(합가)"** = 후가공 합산가(확정규칙④, 분해 금지). 오시|미싱, 가변텍스트|가변이미지가 **컬럼 가로동거**(F열부터 별도 블록). 증분 규칙 note(B02 "100장당 1000원씩")는 cell로 보존(데이터 아닌 규칙).
- **차원 후보**: 후가공종(모서리/오시/미싱/가변)→공정(후가공비 comp), 옵션(직각/둥근, 1줄/2줄)→`mat_cd` 또는 옵션, 제작수량→`min_qty`. 합가→상품단가/후가공비 comp.

### 5. 커팅타공 (`cutting`) — 완칼 + 타공(단가/합가) 가로동거

- **데이터영역**: r1–66 × cA–H. nonempty 162. 머지 6. **3블록**.
  | block | title | 행범위 | 컬럼 | 비고 |
  |-------|-------|-------|------|------|
  | B01 | 커팅(완칼) 가공비 (국4절) | r1–38 | A–H | B열 단면. **C1=인쇄배경지/C2=인쇄비+소재+커팅**(합가 note) |
  | B02 | 1장당 2000원씩 올립니다. | r39–41 | A–H | 증분 규칙 note |
  | B03 | 타공 (단가) | r42–66 | A–H | A–C=타공단가, **F–H=타공(합가)** 가로동거, D42=헤더택/벽걸이캘린더 note |
- **수량축**: A열 `수량(국4절)`/`옵션/ 제작수량` = 출력매수/제작수량. 1~1000000.

- **블록 인벤토리(상위)**: 아트프린트포스터(인화지) r1–17 / 아트페이퍼포스터 r18–34 / 방수포스터(PET) / 접착방수(PVC) / 접착투명(투명PVC) / 아트패브릭(그래픽천) / 린넨패브릭 / 캔버스패브릭 / 레더아트프린트 / 타이벡(하드/소프트) / 메쉬프린트 … 이후 r189+ **다른 레이아웃**(프레임리스우드액자·레더아트액자·족자·캔버스행잉·린넨우드봉·PET배너·메쉬배너·일반현수막·메쉬현수막·미니스탠딩보드·미니배너 — 각 상품 + `사이즈 / 수량` 좌측축 sub-header).
- **밴드헤더 1단(상단)**: row2 = 가로축(`600mm`~`3000mm`) × A열 = 세로축(`600mm`~). **면적매트릭스**. **F열 = `코팅포함가`**(코팅 합가, 확정규칙④).
- **수량축**: 상단 블록 = 면적(가로×세로) 축, 수량 없음. 하단(r189+) 상품 = `사이즈 / 수량`(A4/A3/A2 규격 × 수량) — 규격×수량 축으로 전환.
- **특이점**:
  - **상품 31블록**(인화지/매트지/PET/PVC/패브릭/현수막/배너…) — 자재(소재)가 상품 분기.
  - **상·하 2레이아웃 혼재**: 상단 = 면적매트릭스(가로×세로mm), 하단(r189~) = 규격(A4~A1)×수량 + 좌측 `사이즈 / 수량` 축. 하단의 `사이즈 / 수량` sub-header가 블록으로 다수 잡힘(정상 — 각 상품 영역 헤더).
  - F열 `코팅포함가` + `1000x1000 : 20000` note(특정 사이즈 고정가).
- **차원 후보**: 가로×세로/규격(A4~A1)→`siz_cd`, 소재(인화지/매트지/PET/패브릭…)→`mat_cd`, 수량(하단)→`min_qty`. 코팅포함가→상품단가(합가).

## ③ 시트 간 공통/상이 구조 종합

### 공통 구조 패턴
- **수량축 = A열, 출력매수 또는 제작수량**: 디지털/코팅/스티커/커팅 = **출력매수(국4절/3절)**, 접지/후가공/제본/봉투/합판/명함/엽서북 = **제작수량**. (출력매수≠주문수량 — 판걸이수로 변환, 확정규칙·DDL §4.)
- **상한개방 sentinel**: 코팅/접지/스티커/커팅의 마지막 수량행 `1000000`/`100000` = 상한개방 마커(round-1 max_qty=NULL 패턴, DDL C-3와 동형). **실상한 아님**(확인 필요 표기).
- **밴드헤더 다단**: 대부분 2단(외밴드 머지 × 내밴드). 최심 = 엽서북 **3단**(사이즈×인쇄×페이지). 면적시트(아크릴/포스터/박대형)는 1단(가로축) + A열 세로축 = 2D.
- **합가 만연**: 인쇄후가공(전 블록 "합가")·커팅(인쇄비+소재+커팅)·포스터(코팅포함가)·제본(삼각대포함)·스티커(종이+인쇄+커팅)·접지(오시+접지)·박명함(종이+동판+박). → 확정규칙④(분해 금지, 상품단가 반영) 적용 대상.

### 상이 (동일기준 적용 불가 핵심)
| 축 | 수량×옵션 밴드 | 면적매트릭스(2D) | 2D class 2단룩업 |
|----|--------------|----------------|----------------|
| 시트 | 디지털·코팅·접지·후가공·커팅·스티커·합판·봉투·명함·엽서북·제본 | 아크릴·포스터·박대형(동판비) | 박소형·박대형(가공비) |
| 행축 | 수량(출력매수/제작수량) | 세로(mm) | 가로(면적정의)/수량(가격) |
| 열축 | 옵션밴드(도수/소재/단양면) | 가로(mm) | 분류문자(A~E) |
| 차원 후보 | min_qty + clr/mat/coat | siz_cd(면적) | **면적→분류→가격 = 단순차원 표현 어려움(fit-gap)** |

- **가로 동거 블록**(같은 행대 좌우 2블록): 인쇄후가공(오시|미싱·가변텍스트|가변이미지), 커팅(타공단가|타공합가), 박소형/대형(면적정의|수량별가격), 명함(포토카드|투명포토카드), 아크릴(카라비너|구간할인). → block 좌우경계 분리 필수(detect_blocks 우경계 규칙).
- **다블록 stacking**: 코팅/접지/디지털(2), 합판(3), 제본(3), 스티커(7), 명함(12), 포스터(31). 제목 반복 등장 → 블록경계 놓치면 밴드헤더 섞여 무손실 깨짐(방어: 셀단위 long 산출).
- **수식 시트 = 아크릴만**(81셀, 미러=투명×2). 나머지 0. → 수식 보존(is_formula+formula) 필수, 차원 확정 시 파생관계 인지.
- **2단 룩업(박)**: 면적→분류문자(A~E)→가격. component_prices 6차원으로 직접 표현 난해 → **다음 단계 fit-gap 우선검토 대상**.
- **round-1 영역 혼입**: 아크릴 B04/B07(구간할인)·포스터 일부는 가격 아닌 할인율 → round-2 재매핑 금지(확정규칙⑨).

### t_prc_component_prices 6차원 귀속 후보 종합 (단정 아님)
| 차원 | 강한 후보 시트/축 | 비고 |
|------|------------------|------|
| `siz_cd` | 국4절/3절(디지털·코팅·스티커·커팅), 사이즈판수(스티커), 모양·mm(합판), 면적 가로×세로(아크릴·포스터·박), 규격 A4~A1(포스터하단) | 면적은 work/cut 치수 |

## ④ 다음 단계로의 인계 (이번 단계 비대상 — 명시)

본 단계 산출은 **무손실 추출 + 구조 카탈로그**까지. 아래는 **다음 단계**(평면화·매핑):
1. **평면화**: 셀단위 long → component_prices long-format(comp_cd·6차원·unit_price)으로 변환.
2. **차원 확정**: 위 ⑦ 후보를 fit-gap으로 확정(특히 박 2단룩업·엽서북 페이지·면적형).
3. **공식 배선**: 계산공식집초안 ↔ formula_components ↔ 각 단가시트 참조 연결.
4. **적재 CSV**: t_prc_* 테이블별 load CSV(DB 미적재).

**미해결/확인 필요(다음 단계 입력)**:
- 수량 sentinel `1000000`/`100000` 실상한 여부(상한개방 마커로 추정하나 미확증).
- 박 2단룩업(면적→분류 A~E→가격)의 component_prices 차원 표현(직접 차원 부재 가능 → fit-gap).
- 엽서북 페이지(20P/30P)·아크릴 도수(양면9도/단면7도)의 차원/옵션 귀속.
- 면적정의 2D class 매트릭스 블록의 band_path 무의미(구조상 정상, 평면화 시 별도 모델).

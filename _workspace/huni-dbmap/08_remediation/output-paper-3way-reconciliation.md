# 출력용지규격 3자 관계 분석 — 상품마스터·출력소재(IMPORT)·판걸이수

> 작성 2026-06-07. 사용자 지시("판걸이수·출력소재(IMPORT)·상품마스터 각 시트 출력용지규격의 관계를 다시 연결해서 확인 + 전체 셀 재확인해 놓친 것 규명")에 따른 종합 재규명.
> 권위: 엑셀 L1 추출본(`06_extract/`) 전체 셀 + 라이브 railway DB read-only(2026-06-07). 식별자/SQL 영어·해석 한국어.

## 0. 핵심 결론 (놓친 것)

이전 통일 설계(`plate-price-unify-design.md`, 국4절+3절 2종)는 **3가지를 놓쳤다**:
1. **출력용지규격 권위 = 상품마스터 `파일사양_출력용지규격` 컬럼** — 각 시트가 상품×사이즈옵션별로 직접 명시. 라이브 plate(작업사이즈)만 보고 이 컬럼을 미대조.
2. **투명 출력용지규격 315×467** — 투명엽서·투명포토카드·투명명함은 316×467(국4절)이 아니라 **315×467(투명 PET 전지)**. 이전 설계가 국4절로 통일하려던 것은 오류.
3. **종이별 용지비 미적재** — IMPORT의 종이별 `연당가·가격(국4절)·가격(3절)`이 라이브 component_prices에 없음(디지털인쇄 가격은 종이 무관 인쇄 공정비만).

## 1. 3자 시트의 출력용지규격 표현 (전체 셀 확인)

| 시트 | 컬럼 | 의미 | 값(디지털인쇄) |
|------|------|------|----------------|
| **상품마스터** 각 시트 | `파일사양_출력용지규격` | 상품×사이즈옵션별 출력용지규격 **명시 권위** | 316x467(56)·330x660(5)·315x467(3) |
| **출력소재(IMPORT)** | `종이사이즈` + `가격(국4절)`/`가격(3절)`/`연당가` | 종이별 출력용지규격 + **용지비** | 종이사이즈 9종·종이 120종 |
| **판걸이수** | `디지털인쇄(국4절)`=316x467 / `디지털인쇄(3절)`=330x660 / `col_I`(반칼전지) | 작업사이즈별 출력용지규격 + **판걸이수**(임포지션 수) | 국4절/3절/반칼전지 |

세 시트 모두 출력용지규격을 표현하며 **316×467(국4절)·330×660(3절)·315×467(투명)이 일관**.

## 2. 출력용지규격 종류 (전수)

### 2.1 상품마스터 명시 (디지털인쇄 계열)
- **316x467** (국4절) 56행 · **330x660** (3절) 5행(지그재그엽서) · **315x467** (투명) 3행(투명엽서·투명포토카드·투명명함)

### 2.2 출력소재(IMPORT) 종이사이즈 전체 (9종 — 종이 종속)
| 종이사이즈 | 종이수 | 비고 |
|-----------|-------:|------|
| 316x467 | 49 | 국4절(일반지) |
| 330x470 | 6 | 스티커 전지 |
| 330x660 | 5 | 3절(아트지150g·스노우지250g·몽블랑130/190/240g) |
| 315x467 | 4 | 투명 PET |
| 330x480 | 3 | 투명/홀로그램/크라프트 스티커 |
| 464x320 | 2 | 띤또레또 200/250g |
| 330x540 | 2 | 아이보리·백색모조 |
| 315x470 | 1 | 수분리 스티커 |
| 210x297 | 1 | 타투전용지 |

### 2.3 전지(원지) ≠ 출력용지규격(종이사이즈)
- 전지 = 구매 원지 규격: 국전(939×636)·4*6(1091×788)·700×1000·330×470·464×320·210×297 등.
- 출력용지규격(종이사이즈) = 원지에서 인쇄용으로 자른 단위. 예: 국전·4*6·700×1000 원지 → 전부 **316×467(국4절)** 로 자름.
- **3절은 5개 종이 한정**(특정 제품이 아니라 특정 종이). 4*6 원지에서 330×660로.

## 3. 종이 종속 — 한 상품이 출력용지규격 다수

matrix(상품×종이 ●) + IMPORT 종이사이즈 조인 결과, 일부 상품은 종이 선택에 따라 출력용지규격이 2종:
- 접지카드·벽걸이캘린더·소량전단지·프리미엄명함·프리미엄엽서·탁상형/엽서캘린더 = **316x467 + 464x320**(띤또레또 종이 선택 시).
- 즉 **출력용지규격은 제품 고정이 아니라 (상품 × 종이) 조합으로 결정**. 상품마스터 `출력용지규격` 컬럼이 사이즈옵션 단위로 이를 명시.

## 5. 교정 방향 (수정 — 권위=상품마스터 출력용지규격 컬럼)

1. **plate ← 상품마스터 `파일사양_출력용지규격`**: 라이브 plate를 작업사이즈가 아니라 상품마스터 명시 출력용지규격(316x467/330x660/315x467)으로. (사이즈옵션별 다종 가능 — 종이 종속 반영.)
2. **출력용지규격 siz**: 316x467=`SIZ_000499`(재사용) / 330x660=신규 / **315x467=신규(투명)**. (이전 설계가 투명 누락 → 추가.)
3. **종이별 용지비 적재(별 트랙·round-2 가격)**: IMPORT 종이별 연당가/국4절가/3절가를 가격엔진에 — 용지비 comp 신설 또는 디지털인쇄 가격의 종이 차원(mat_cd) 도입. **규모 큼·별도 검토**.
4. **출력용지규격 9종 vs 디지털인쇄 3종**: 스티커(330×470 등)·띤또레또(464×320)·타투(210×297)는 디지털인쇄 외 계열 — 별 트랙.

## 6. 미해결 / 결정 필요
- 🔴 종이별 용지비 적재 방식(용지비 comp vs 디지털인쇄 mat 차원) — round-2 가격 재설계 영역.
- 🟡 투명 315x467 신규 siz + 3절 330x660 신규 siz 채번(인간).
- 🟡 출력용지규격이 사이즈옵션별 다종(띤또레또 464×320 등)일 때 plate 행 구성.
- 🟢 권위 확정: 상품마스터 `파일사양_출력용지규격` 컬럼이 plate 적재 권위.

# 가격사슬 완전성 — 프리미엄엽서 (PRD_000016) · round-18 파일럿

> 라이브 읽기전용 실측(2026-06-14) + 가격뷰어 3원 대조. 권위 = 라이브 `t_prc_*`/`t_dsc_*` + Phase11 명세(11-CONTEXT/prcx01). DB 미적재.

## 0. 상품 정체
- PRD_000016 프리미엄엽서. 작업사이즈 7종(SIZ_000001~007: 73x98·98x98·100x150·135x135·95x210·110x170·148x210).
- **출력판형 = SIZ_000499(316x467, OUTPUT_PAPER_TYPE.01)** — `t_prd_product_plate_sizes` 단일 매핑.

## 1. 가격 소스 (우선순위: 템플릿>직접단가>공식)
| 소스 | 라이브 | 판정 |
|---|---|---|
| 템플릿단가 | 없음 | — |
| 상품 직접단가 `t_prd_product_prices` | 0행 | 미사용 |
| 상품 공식 `t_prd_product_price_formulas` | **PRF_DGP_A** 바인딩 1건(공식 정의 `t_prc_price_formulas.use_yn=Y`) | ✅ 현재 적용 |

→ 가격뷰어 3원 일치(뷰어 "상품공식 PRF_DGP_A" 표시 = DB).

## 2. 공식 사슬 (PRF_DGP_A → 구성요소 → 단가행)
- `t_prc_formula_components`: **29개 구성요소 배선**(disp_seq 1~29), 전부 addtn_yn=Y.
- 전 구성요소 `t_prc_price_components.prc_typ_cd = PRICE_TYPE.01(단가형)`. **합가형 0**.
- 전 구성요소 `component_prices` 단가행 보유(인쇄 212·별색 53×10·코팅 92×2·용지 49·후가공 9~23). **단가행 0 구성요소 없음** → 데이터상 사슬 완결.
- 구성요소군: 디지털인쇄(단/양면 S1/S2, dims siz·clr·min_qty) · 별색 5색×단/양면(dims siz·min_qty) · 코팅 유광/무광(dims siz·coat_side_cnt·min_qty) · 용지(dims siz·mat) · 후가공 오시/미싱/가변텍스트/가변이미지/모서리(dims min_qty).

## 3. 차원 키 정합 (실측 핵심)
- **인쇄/별색/코팅/용지 단가행 siz_cd = SIZ_000499(출력판형)** — 상품 작업사이즈 SIZ_000001~007과 **불일치**. 가격은 작업사이즈가 아니라 출력판형으로 조회됨(→ 결함 D-1).
- 도수 clr_cd: 단가행은 **CLR_000002(1도흑백)·CLR_000005(CMYK 4도)** 2종만. 상품 인쇄옵션 opt_id1(단면 front=CLR_000005)·opt_id2(양면 front/back=CLR_000005). 4도 매칭 OK, 그 외 도수는 단가행 부재.
- 종이: 상품 적재 종이 다수(MAT_000074/082/092…)와 COMP_PAPER siz=499 단가행 교집합 존재(MAT_000082 아트지300g=93.30 등).

## 4. 할인 (수량구간 → 등급)
| 단계 | 라이브 | 판정 |
|---|---|---|
| 수량구간 `t_prd_product_discount_tables` | **0행 — 미연결** | 🔴 할인테이블 미연결 |
| 등급 `t_dsc_grade_discount_rates` | 전역 0행 | 스킵(후속) |

→ 엽서는 할인 미적용 상태. 단, 인쇄/코팅 단가행 자체가 수량구간(min_qty)별 차등 = 볼륨 디스카운트는 단가에 내재.

## 5. 가격 시뮬레이션(엔진)
- 라이브 가격뷰어 ⑤ "가격엔진(다음 단계) 완성 후 활성화" — **계산 미구현**. 사슬 데이터는 있으나 계산 주체 없음.

## 종합
데이터 사슬 = **완결(소스·배선·단가행 전부 존재)**. 그러나 ① 출력판형 해석 ② 옵션→구성요소 매핑이 없으면 계산 불가/오류(§defects). 할인 미연결.

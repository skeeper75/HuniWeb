### 1.4 잔여 BLOCKED — 기본가 2000 (Q-STK-1b)

## 2. Q-STK-3 — 스티커팩(065) prc_typ 오적재 교정

### 2.1 가격표 verbatim (스티커 시트 B06·행 86~89) + 라이브
```
A86='스티커팩 (54장 1세트)'   A87='옵션/제작수량'  B87='75x110'
A88=1     B88=4000
A89=1000  B89=4000
라이브: COMP_STK_PACK prc_typ_cd=PRICE_TYPE.01(단가형)  use_dims=["siz_cd","min_qty"]
        행: (SIZ_000068, min_qty=1, 4000) · (SIZ_000068, min_qty=1000, 4000)
```

### 2.2 오적재 진단
- 현 `.01 단가형` → 엔진 `4000 × 수량`. 즉 54장 1세트 주문(수량=54?) 시 4000×54=216,000 = **터무니없음**. 가격표 의미 = "54장 1세트 = 4000"(세트 총액).
- "54장 1세트" = 1세트(=54장)당 4000. 합가형(.02) `4000 ÷ 54 = 74.07/장 × 수량`. 수량 단위가 "장"이면 74.07/장, "세트"면 별도.

### 2.3 교정 정립
| 항목 | 현재 | 교정 | 근거 |
|------|------|------|------|
| prc_typ_cd | PRICE_TYPE.01 | **PRICE_TYPE.02 합가형** | "54장 1세트 4000"=세트총액 |
| min_qty | {1, 1000} 2행 | **54 단일행**(세트 단위) | A86 "54장 1세트"·엔진 base=tier_min_qty |
| unit_price | 4000 | 4000 verbatim | B88 |
| siz_cd | SIZ_000068(75x110) | 유지 | 라이브 실존 |
| 공식 | (없음) | `PRF_STK_PACK`(신규)·바인딩 PRD_000065 | 미배선 해소 |

- **★min_qty=54 필수**(엔진 .02·base≤0 ValueError·아크릴 A5 교훈 동일). 현 2행(1·1000)은 단가형 전제라 .02 전환 시 둘 다 부적합 → **min_qty=54 단일행으로 교체**(기존 2행 삭제).

### 2.4 잔여 컨펌 (Q-STK-3b)
- 환산 단위: 주문 "수량"이 **장 단위**(÷54)인지 **세트 단위**(세트당 4000·min_qty=1)인지 = 위젯 수량 입력 의미 확인. 권고 = 세트 단위라면 min_qty=1·unit=4000(.01도 가능)·장 단위라면 min_qty=54·.02. **위젯 수량 의미 컨펌 후 확정.** 소재(Q-STK-6)도 미확정(가격표 미명시)이나 use_dims에 mat_cd 없으므로 적재 무관.

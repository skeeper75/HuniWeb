## 0. 한눈에 (비전문가용 결론)

- **명함 3상품은 디지털인쇄 시트의 고정가형(PRF_NAMECARD_FIXED) 클래스** — 낱장 인쇄물. 가격은 "구성요소 합산"이 아니라 **세트당 완제품 고정단가(용지포함)를 표에서 직접 조회**(가격표 명함포토카드 시트: 행=수량·열=소재×면). 계산공식집초안 R42 "[세트당 고정단가]" 확정.
- **기초데이터(사이즈·종이·공정·도수)는 라이브에 잘 적재됨.** 031=종이 16·공정 12(모서리2+가변2+박8)·도수 2, 032=종이 2·공정 4(코팅2+모서리2)·도수 2, 033=종이 5·공정 4(모서리2+가변2)·도수 2. 사이즈는 SIZ_000008(90×50)·일부 추가 / 출력판형 SIZ_000499(316×467 국4절).
- **★ 엽서·상품권과 같은 좋은 점: CPQ 옵션 레이어가 3상품 다 실적재되고 모든 옵션이 차원행으로 100% 환원된다.** 031=5그룹/30옵션, 032=4그룹/22옵션, 033=4그룹/22옵션. 트리거 정합 위반 0 — 손님 선택이 자재/공정/도수로 환원(②생산정보 성립).
- **★★ 고정가형 가격사슬의 결정적 결함(NAMECARD-WIRE·🔴 돈-크리티컬·신규 유형)**: 명함종별 가격 comp(STD·COAT·PREMIUM·PEARL·FOIL 등 25 comp)가 **데이터(component_prices)는 다 적재됐는데**, 가격공식 PRF_NAMECARD_FIXED에 **STD(스탠다드)만 배선**되고 PREMIUM/COAT/FOIL은 미배선이다. → **031 프리미엄·032 코팅이 자기 가격을 못 찾음**:
  - **033 스탠다드** → STD 배선됨 → MAT_000074(3,500)·MAT_000082(3,800) 룩업 가능(단 5소재 중 2소재만 직접 매칭 — §3-4).
  - **031 프리미엄** → PREMIUM_S1/S2_MGA/MGB 데이터 실재(A단면 4,500·B단면 5,000 = L1 정합)인데 **공식 미배선** → 031의 16소재 중 STD 룩업 매칭 **0개** → **031 전 견적 가격 단절(완전 NO-GO)**.
  - **032 코팅** → COAT_S1/S2 데이터 실재(아트250단면 5,500·아트300단면 5,800 = L1 정합)인데 **공식 미배선** → 032가 STD로 떨어지면 아트300=3,800(코팅 5,800이어야)·아트250=룩업 없음 → **032 가격 오산(과소) 또는 단절**.
- **★ 명함 박 가격구조(042 참고 패턴)**: 명함 박은 **COMP_NAMECARD_FOIL_S1/S2_STD/HOLO(수량별 9구간 합가, 200~1000매: 단면 STD 19,200~63,000·HOLO 24,800~92,000) + COMP_NAMECARD_FOIL_SETUP_S1/S2_STD(동판셋업 5,000 수량무관)**으로 **데이터가 실재**한다(가격표 후가공_박(소형) 시트 권위 — 동판비 5,000·면적등급A~E×수량). 단 이 FOIL comp들은 **오리지널박명함(별도 상품)용**으로 적재됐고 **PRF_NAMECARD_FIXED에 미배선** → 031 박칼라 8색(PROC_000037~044)은 UI·차원환원 완비됐으나 가격 미배선. **★상품권 042(박 comp 0개·데이터 자체 부재)와 대조: 명함은 박 가격 primitive가 이미 있다 → 042 박 해결의 참고 패턴(comp 신설 불요·면적등급/수량 구조 재사용).**
- **결론(정직):** 명함 **033 스탠다드 기본옵션(인쇄+종이) 견적 = GO**(STD 배선·일부 소재). **031 프리미엄·032 코팅 = 명함종 comp 미배선으로 가격 단절/오산(NO-GO)**. **031 박 = FOIL 데이터 실재·미배선(NO-GO·배선만 필요)**. 종합 = **부분 GO(033 일부만)·핵심 결함 NAMECARD-WIRE(명함종 comp 미배선)**. 엽서·상품권보다 가격사슬 결함이 무겁다(고정가형 배선 자체가 STD 외 누락).

## 3. 컬럼사전/선례 대비 라이브 정정·명함 고유 (실측이 권위)

### 3-1. 명함 고유 — 사이즈 1규격(90×50)·모서리 옵션 보유·추가상품 없음

### 3-2. 명함 핵심 = 명함종(STD/PREMIUM/COAT/PEARL...)이 가격 1차축
- 가격표 명함포토카드 시트: **명함종별 별 블록**(스탠다드 R0·프리미엄 R7·코팅 R14·펄 R21·투명 R28·화이트인쇄 R35·모양 R42...). 각 블록 = 행(제작수량=100 단일)·열(소재×면). **명함종 = 가격 comp 식별자**(STD/PREMIUM/COAT...). 031=프리미엄·032=코팅·033=스탠다드. **명함종이 다르면 가격 comp가 다르다** — 이게 고정가형 배선의 핵심.

### 3-3. ★★ 고정가형 가격사슬 — NAMECARD-WIRE 결함(🔴 신규 유형·합산형과 다름)

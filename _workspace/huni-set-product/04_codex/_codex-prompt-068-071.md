# 독립 교차검증 의뢰 — 후니프린팅 셋트상품 책자 4종(068~071) 설계 검토

당신은 인쇄 ERP 가격엔진/셋트상품 구성의 독립 2차 검토자다. 아래 설계안을 **백지에서 독립으로** 검토하라.
다른 검토자(Claude)의 판정은 일부러 주지 않는다 — 당신 스스로 결함/적정 여부를 판단하라.
판정은 가설로 받아들여진다(라이브/권위로 확정 전 사실 아님). 추측이면 추측이라고 명시하라. 환각 금지 — 근거 없는 단정 금지.

읽기전용이다(파일 수정 금지). 한국어로 답하라.

## 0. 시스템 배경 (가격엔진 계약)
- 셋트상품 = 부품조립형: 셋트 완제품(prd_cd) ← 반제품 구성원(sub_prd_cd), 테이블 t_prd_product_sets.
- 가격 계약 `evaluate_set_price(set_prd, selections, copies)` =
  Σ(각 구성원 evaluate_price) + 셋트 부모공식 evaluate_price + 할인(셋트 1회).
- 구성원/부모공식 모두 "가격공식(PRF_*)" → formula_components → price_components(COMP_*) → component_prices(단가행) 사슬.
- 한 비목(인쇄/코팅/제본/용지)이 부모공식과 구성원 양쪽에 들어가면 **이중합산**(과대청구). 누락되면 **저청구**.
- PRICE=0 = 결함신호(공식/단가행 미배선).
- 내지 페이지 가변: 총내지매수 = copies × ⌈pages / pansu⌉ (derive_inner_sheets). pansu=판걸이수(임포지션).

## 1. 검토 대상 4종 (현재 라이브 상태)
4종 전부 라이브에 **셋트 구성원 0행(미구성)**. 각자 제본비 comp 1개만 바인딩되어 있음(종이·인쇄·표지값 전부 누락 상태).
- PRD_000068 중철책자, PRD_000069 무선책자, PRD_000070 PUR책자, PRD_000071 트윈링책자.
- 4종은 셋트 완제품(prd_typ_cd=PRD_TYPE.01)로 라이브 실재.
- 구성원이 될 반제품(표지·내지)도 라이브 0행(미존재).

## 2. 설계안 요지
각 책자 = **표지(역할 02) 1구성원 + 내지(역할 01) 1구성원 = 2구성원**(면지 없음).

부모 셋트공식(신설 예정·BLOCKED): 표지인쇄(COMP_PRINT_DIGITAL_S1) + 표지코팅(COMP_COAT_MATTE/GLOSSY) + 제본(COMP_BIND_<METHOD>) 3비목 분해 배선.
내지 구성원공식: 기존 PRF_DGP_INNER 재사용 = 내지인쇄(S1/S2) + 내지용지(COMP_PAPER), page파생 qty.
표지 구성원: 가격공식 없음(가격 contribution 0). 표지인쇄·코팅 비용은 전부 부모공식에.

비목 단일 귀속 주장:
- 내지인쇄비 → 내지 구성원만
- 표지인쇄비/표지코팅비/제본비 → 부모공식만
- 내지용지비 → 내지 구성원만
- 후가공박(069/070만) → 부모공식만

책자별 인스턴스:
| 셋트 | 제본 comp | 내지 page min/max/incr | 내지인쇄 면 | 표지×N | 후가공박 | 고유 |
|---|---|---|---|---|---|---|
| 068 중철 | COMP_BIND_JUNGCHEOL | 4/28/4 (4배수·접지) | 양면(S2) | ×1 | 없음 | — |
| 069 무선 | COMP_BIND_MUSEON | 24/300/2 | 양면(S2) | ×1 | 박/형압 | — |
| 070 PUR | COMP_BIND_PUR | 24/300/2 | A5=단면(S1)/A4=양면(S2) | ×1 | 박/형압 | — |
| 071 트윈링 | COMP_BIND_TWINRING | 8/100/2 (링한계) | A5=단면(S1)/A4=양면(S2) | ×2(앞뒤표지) | 없음 | 링컬러·제본방향·투명커버 |

셋트행 설계(t_prd_product_sets): 각 셋트 2행(표지 sub_prd_qty=1·min/max=1/1 / 내지 sub_prd_qty=1·min/max/incr=위 page범위).
표지·내지 반제품은 mint 필요(라이브 0행) → 셋트 행 실 적재는 BLOCKED(부품 mint 선결, 별도 트랙·인간 승인). 부모공식 신설(PRF_BIND_*_SET 4종)도 BLOCKED(별도 트랙). 단가/comp 신규 창조는 0(전부 기존 라이브 단가행 재사용).
071 표지×2는 부모공식에서 표지인쇄·표지코팅 ×2 배수로 반영(셋트행 표지 qty는 1 유지).

코드결함(엔진) C트랙: DBLPANSU(내지 ÷pansu 이중적용 의심·내지 ~0.4배 과소)·양면용 COMP_PRINT_DIGITAL_S2 부활(del_yn Y) 필요.

## 3. 권위(상품마스터 260610 booklet-l1) 구성요소 경계 (시트 빈칸=경계 밖)
- 068 중철: 사이즈 A5/A4·내지양면·내지페이지 4~28/+4·표지단면(A5)/양면(A4)·표지코팅 코팅없음/무광/유광·제본 중철·수량 2~1000/+1. 박/형압 없음. 면지 빈칸(없음).
- 069 무선: A5/A4·내지양면·24~300/+2·표지코팅 3종·무선제본·박(없음/있음)·형압(없음/음각/양각)·개별포장(없음/수축)·수량 2~1000/+1. 면지 없음.
- 070 PUR: A5/A4·**내지인쇄 A5=단면·A4=양면**·24~300/+2·표지코팅 3종·PUR제본·박/형압·개별포장·수량 2~1000/+1. 면지 없음.
- 071 트윈링: A5좌철/A5상철/A4/A4·**내지 A5=단면·양면**·8~100/+2·**표지인쇄 단/양면(표지×2 앞뒤)**·표지코팅 3종·트윈링제본·제본방향(좌/상철)·링컬러(화이트/블랙/메탈)·투명커버(없음/유광/무광)·개별포장·수량 **1**~1000/+1. 박 없음. 면지 없음.
- 권위 계산공식집 seq63: 판매가 = 내지인쇄비 + 표지인쇄비 + 표지코팅비 + 제본비 + 용지비 + 후가공비 (068/069/070 공유). 071은 seq72(표지×2·후가공박 제외).

## 4. 후니 이전 사이트(거울·가격 오라클 아님) 라이브 폼 실측 사실 (2026-06-29 읽기전용·verbatim)
이건 "권위 엑셀 구조의 거울"이다(가격 정답 아님). 불일치 시 권위 엑셀이 절대. 아래는 관찰 사실만 — 당신이 해석하라.
- 4종 모두 표지가 자체 인쇄(단/양면)+용지+코팅 3슬롯 보유(tmp_p21_*_a/b/c).
- 068 중철 내지인쇄(tmp_p09) = **양면만**.
- 069 무선 내지인쇄 = **양면만**. 박/형압 슬롯 있음.
- 070 PUR 내지인쇄(tmp_p09) = **양면만 노출**(단면 옵션 폼에 부재). ← 설계는 A5=단면 가정.
- 071 트윈링 내지인쇄 = **단면|양면 둘 다** 노출. 링색깔·투명커버·제본방향 슬롯 있음. 표지 슬롯(tmp_p20)=맞춤표지|표지없음 **단일 select**(앞뒤 ×2 별도 슬롯 없음).
- 4종 폼 모두 내지 페이지수 `<select>` 미노출(텍스트입력 추정·스캔범위 밖).
- 면지 슬롯(tmp_p21_*_d)은 068~071에서 비활성(하드커버/돈피 전용 잔재).

## 5. ★독립 판정 요청 항목 (각 항목 GO/조사필요/결함 + 근거)
1. **오구성**: 각 셋트가 표지+내지 2구성원(면지 없음)이 적정한가? 구성원 전부 반제품(완제품/기성/디자인 혼입 0)인가?
2. **이중합산/저청구**: 부모공식(표지인쇄+표지코팅+제본) + 내지구성원(내지인쇄+용지) 비목 분담에서 같은 비목 이중 귀속이 0인가? 누락 비목은 없나? evaluate_set_price로 PRICE≠0 가능한가(부품 mint 가정)?
3. **구성요소 경계 오염**: 각 책자에 자기 시트 경계(§3) 밖 구성요소가 끌려오지 않았나? 공유공식 PRF_DGP_INNER가 4종에 silent 적용되며 각 책자 제본 comp가 다른 책자에 새거나 누락되지 않나? (068 4배수가 타 책자 유입·071 표지×2가 068~070 유입·070/071 A5단면 경계 등)
4. **조사신호 독립 판정**: (a) 070 PUR 내지 A5=단면 설계 vs 이전사이트 양면만 노출 — 어느 쪽이 맞다고 보나? 돈영향은? (b) 071 표지×2를 부모공식에 강제하는 게 타당한가, 과대청구 위험인가? (c) 내지 페이지축이 이전사이트 폼에 없는데 설계의 페이지범위를 그대로 써도 되나?
5. **BLOCKED 타당성**: 부품 8개(표지4+내지4)·부모공식 4개·코드트랙(DBLPANSU·S2부활)을 선결 차단으로 두는 게 정당한가, 아니면 과한 차단(false-positive)인가?
6. **false-positive 가드**: 위 설계 중 "정당한 부품조립 셋트 구조"를 결함으로 오판할 만한 지점이 있나? (정상인데 결함으로 깎으면 지적)
7. **추가로 발견한 것**: 위 6항목 밖에서 당신이 독립적으로 발견한 결함·위험·돈크리티컬 이슈가 있으면 모두.

각 항목마다 verdict(GO / 조사필요 / 결함)와 한두 문장 근거를 달라. 마지막에 "가장 돈크리티컬한 위험 Top3"를 꼽아라.

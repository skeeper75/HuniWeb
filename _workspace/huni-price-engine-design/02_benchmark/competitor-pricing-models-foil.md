## 0. 후니 권위 모델 (대조 기준 — 흡수 대상 아님, 비교의 자)

출처: `_workspace/huni-dbmap/06_extract/price-foil-large-l1.csv`(48행)·`price-foil-small-l1.csv`(50행). 두 권위 엑셀 시트 = **후가공_박(대형)** / **후가공_박(소형)**.

| 구성요소 축 | 박(대형) | 박(소형) |
|---|---|---|
| **동판비(setup)** | 가로×세로 mm **면적 매트릭스**(30~170mm 격자, 11,000~64,000원) | **고정** 80x40mm = 5,000원(아연판) |
| **박가공비 — 면적분류** | 가로/세로 mm → **A~E 등급 lookup** 표 | 가로/세로 mm → A~E 등급 lookup 표 |
| **박가공비 — 수량가격** | 등급(A~E) × 수량구간(10~10,000) 매트릭스 | 등급(A~E) × 수량구간(200~10,000) 매트릭스 |
| **색상 차등** | **일반박**(금유광/금무광/은유광/은무광/동박/청박) vs **특수박**(먹유광/백박/홀로그램/트윙클/적박/녹박) = **2개 별도 블록** | 일반박(금유광/은유광/먹유광/청박/적박/동박/펄박) vs 특수박(백박/홀로그램박/트윙클박) = 2개 별도 블록 |

**후니 박 계산식(권위):**
```
박_총액 = 동판비(setup) + 박가공비
  동판비    = (대형) 면적매트릭스[가로][세로]  OR  (소형) 5,000 고정
  박가공비  = 수량가격표[등급][수량]
              등급 = 면적분류표[가로][세로]   ← A~E
              표 선택 = 일반박이냐 특수박이냐 (별도 2블록)
```
요점: **동판비 = 1회성 setup(면적 또는 고정)**, **박가공비 = 면적등급(A~E) × 수량구간 매트릭스**, **색상은 일반/특수 2단계 차등**(특수박 표가 더 비쌈). 양면/단면 무관(박가공 = 면 단위 아님 — pricing-rules.md PRC-005 관찰).

### 1.1 박 옵션 구조 (명함 캡처에서 직접 관찰)

| select | id/name | 옵션값(allOptions, verbatim) |
|---|---|---|
| **박 종류** | `spdata_00_awk115281` (sJob0), optionCount 9 | 금박유광/금박무광/은박유광/은박무광/적박/청박/녹박/먹박/홀로그램박 |
| **박 크기(앞면)** | `spdata_00_awk115282` (sJob0), optionCount 5 | 박(앞면) 선택하세요 / **15mm이하 / 30mm이하 / 70mm이하 / 90mm미만** |
| **박 크기(뒷면)** | `spdata_00_awk116822` (sJob0), optionCount 5 | 박(뒷면) 선택하세요 / 15mm이하 / 30mm이하 / 70mm이하 / 90mm미만 |

→ **박 종류(색상)** 와 **박 크기 단계** 가 독립 select. 앞면/뒷면을 각각 선택(면 단위로 선택은 하나 — 후니 "면 무관"과 미세 차이, 단 가공비 자체는 크기 기반).

라이브 보강(2026-06-30): 박 명함이 **엠보싱금박명함 / 엠보싱은박명함 / 엠보싱홀로그램박명함** 등 **상품 자체로 분리** 되어 있기도 함(`/ordr/prod/`). 즉 와우프레스는 박을 ① 일반 명함의 후가공 옵션 ② 박 전용 상품 두 경로로 운용.

### 1.2 계산방식 — 항목별 소계 합산형 API

**가격 API(POST, 읽기전용 관찰):**
- `https://wowpress.co.kr/ord/calc/jobqty0` — 수량/단가 계산
- `https://wowpress.co.kr/ord/calc/jobcost0` — 비용 집계

postData 예: `ProdNo=40073&Job=PRS&JobNo=3110&...&OrdQty=500&SizeNo=10732&WSize=50&HSize=90&PaperNo=22907&ColorNo=255` — **`Job` 단위로 항목을 쌓는 구조**(Job=PRS=인쇄). 박 선택 시 별도 Job 항목이 추가되어 합산되는 형태.

**응답(jobcost0) verbatim 구조:**
```
[인쇄비]
  합판옵셋인쇄
  기본 인쇄비    3,300 원
  소계           3,300 원
...
```
→ **`[항목명] ... 소계` 블록을 더하는 합산형**. 박을 선택하면 `[박]` 또는 `[후가공]` 블록이 추가되어 소계 합산(캡처에는 박 미선택 cost만 있어 박 라인 미수집 — **정직 표기**).

### 1.3 와우프레스 박 모델 요약

| 질문 | 와우프레스 답 |
|---|---|
| 동판비/setup 별도? | **캡처상 별도 동판비 select 미관찰**. 박 종류+크기 select만. 동판비가 가공비에 흡수됐을 가능성(라이브 박 cost 미수집 → **불명, 정직 표기**) |
| 면적 vs 수량 | **면적(크기 단계 4구간: 15/30/70/90mm)** 이 명시 축. 수량은 전체 OrdQty에 반영(박 단가가 수량 따라 변하는지는 박 cost 라인 미수집으로 불명) |
| 색상 차등 | 박 종류 9~18종 select(금/은/적/청/녹/먹/홀로그램). 종류별 가격 차등 여부 **cost 미수집으로 불명**(단 홀로그램=특수 통례상 더 비쌈) |
| off-grid | 크기를 **단계(15/30/70/90mm)로 양자화** — 후니의 면적등급(A~E)과 동형 발상 |
| 계산식 | **항목별 소계 합산형**(Job 단위 누적) |

## 4. 후니 모델 합리성 판정 (Q4 sanity check)

후니의 **"동판비(면적매트릭스) + 박가공비(면적등급 A~E × 수량구간)"** 모델은:

1. **동판비/가공비 분리** = 레드프린팅(스티커 동판금형)과 정합. 실제 박 공정의 물리(동판=1회 제작비, 가공=수량비례 압인비)를 정확히 반영. **합리적**.
2. **면적 + 수량 둘 다** = 후니가 가장 정교(와우프레스=면적 단계, 레드=면적 구간 위주). 박 압인은 면적(박 사용량)과 수량(압인 횟수) **둘 다 비용 동인** — 후니가 도메인적으로 가장 정확. **합리적, 오히려 경쟁사보다 정밀**.
3. **일반/특수 2블록** = 두 경쟁사 모두 동일 패턴(특수박 프리미엄). **합리적·표준 정합**.
4. **면적등급 A~E lookup** = 와우프레스 4단계·레드 4구간과 동형(연속면적을 등급으로 양자화). **합리적**.

**판정: 후니 박 모델은 두 경쟁사 대비 합리적이며, 면적+수량 2축을 모두 쓴다는 점에서 가장 정밀하다.** 경쟁사 흡수로 후니 모델을 **바꿀** 근거는 없음. 단 흡수 후보(아래 absorption 문서)에서 보강할 표현력 갭(UX 양자화·면적비율 추가금 등)만 점검.

## 5. 미수집·불명 항목 (정직 표기)

- 와우프레스 박 **cost 라인의 실제 금액**: 캡처가 박 미선택 cost만 보유(PRS Job only). 라이브 POST 가격조회는 박 선택 cost 라인 수집 미실시(저트래픽 원칙·박 라인 재현 추가 캡처 필요 시 별도). → 와우프레스 박 단가의 면적/수량 의존 곡선은 **미확정**.
- 레드프린팅·와우프레스 박 **종류별 가격표**: 양사 비공개(가격표 미노출). 일반/특수 차등 존재는 확인, 구체 배수는 불명.
- 레드프린팅 명함 **동판비 유무**: 명함 페이지 비명시(스티커는 명시) — 명함은 가공비 흡수 추정이나 확정 아님.

이 갭들은 **후니 권위 엑셀에 이미 실제 값이 있어**(동판비·등급별 수량표 전수) 설계에 지장 없음. 경쟁사는 구조(축·분리·차등 패턴)만 흡수.

## 1. GC-BK1 — 중철책자 제본비 (단일 prd·A형·중철 4부) [돈크리티컬·G-BK-1/W2]

| 항목 | 값 |
|------|----|
| 상품 | PRD_000068 중철책자 |
| 입력 | 제본방식=중철(PROC_000018)·부수 qty=4 |
| 매칭 | min_qty TIER: qty=4 → min_qty=4 행 선택 |
| **현 라이브(오염)** | unit=3,000(트윈링값) × 4 = **12,000** ❌ |
| **교정 후(B01 정답)** | unit=**2,000**(B01 중철 4부) × 4 = **8,000** ✅ |
| 골든(제본비만) | **8,000** (교정 후·부당 2,000원/부 × 4부) |

★ **검증 포인트**: 교정 전후 차이 = 12,000 vs 8,000(과청구 50%·돈크리티컬). 골든 = **교정 후 8,000**(W2 단가행 교정 적용). 단가 출처 = COMP_BIND_JUNGCHEOL PROC_000018 min_qty=4 → 2,000(가격표 B01 verbatim).

★ **stale 배선(W1) 영향**: PRF_BIND_SUM이 삭제 JUNGCHEOL을 가리키면(현 라이브) — del 필터 미적용 시 JUNGCHEOL=정답 2,000(우연 정합) / del 필터 적용 시 comp 0개=**0원**(no_match). **재배선(W1·활성 TWINRING) + 단가행 교정(W2) 둘 다 해야 8,000 정상 산출**.

## 2. GC-BK2 — 무선책자 제본비 (A형·무선 100부) [W1·정상 단가행]

| 항목 | 값 |
|------|----|
| 상품 | PRD_000069 무선책자 |
| 입력 | 제본방식=무선(PROC_000019)·부수 qty=100 |
| 매칭 | min_qty TIER: qty=100 → min_qty=100 행 |
| 산식 | unit=**500**(B01 무선 100부) × 100 = **50,000** |
| 골든(제본비만) | **50,000** |

★ **검증 포인트**: 무선 proc_cd는 오염 없음(정상). 단 **W1 재배선 + proc_cd 주입 필수** — 현 라이브는 PRF_BIND_SUM이 JUNGCHEOL(중철)만 참조 → 무선책자 견적 시 **중철값 misfire 또는 0원**. 재배선 후 proc_cd=PROC_000019 주입해야 무선 단가 매칭. (proc_cd 미주입 시 4 proc_cd 다중매칭 silent 합산 위험·§2.3 가드.)

## 3. GC-BK3 — PUR책자 제본비 (A형·PUR 1000부) [볼륨할인 사다리 입증]

| 항목 | 값 |
|------|----|
| 상품 | PRD_000070 PUR책자 |
| 입력 | 제본방식=PUR(PROC_000020)·부수 qty=1000 |
| 산식 | unit=**1,500**(B01 PUR 1000부) × 1000 = **1,500,000** |
| 골든(제본비만) | **1,500,000** |

★ **검증 포인트(돈크리티컬 해소 입증·§5)**: PUR 단가 사다리 = 1=5000 → 50=3000 → 1000=1500(**min_qty↑ → unit↓ = 부당 단가 볼륨디스카운트**). **묶음총액이면 1000부에서 1,500으로 떨어질 수 없음**(총액은 증가) → **unit=부당가** 확정 → `.01 × qty` 정합. 디지털 명함(묶음총액 ×qty 폭발)과 단가 의미 정반대 = 교정 불요.

## 7. GC-BK7 — 다부품 합산 완성가 (DT-BIND-SCOPE·표지+내지+제본 Σ) [W3·확신도 중]

> ★**조건부 골든**: DT-BIND-SCOPE=부품 합산 방향 확정 + 표지/내지 단가 소스(디지털인쇄 종이비/인쇄비) 가격표 재대조 후 확정(Q-BK-COVER). 본 케이스는 **합산 구조 검증용**(개별 부품 단가는 디지털 종단 골든 재사용·표지/내지 단가표 확정 전까지 구조만).

| 부품 | comp | 단가 소스 | 예시 산식 |
|------|------|----------|-----------|
| 제본비 | COMP_BIND_SSABARI/PROC_000023 | B02 하드커버무선 4부 | 20,000 × 4 = 80,000 |
| 표지 용지비 | COMP_PAPER(표지 mat_cd) | 디지털 출력소재 시트(절가) | (가격표 재대조·Q-BK-COVER) |
| 표지 인쇄비 | COMP_PRINT_DIGITAL_S1 | 디지털인쇄비 시트 | (도수·출력판형) |
| 내지 용지비 | COMP_PAPER(내지 mat_cd) | 디지털 출력소재 시트 | × 페이지수(앱 곱) |
| 내지 인쇄비 | COMP_PRINT_DIGITAL_S1 | 디지털인쇄비 시트 | × 페이지수(앱 곱) |
| **완성가** | Σ(addtn_yn=Y) | | **제본비 + 표지비 + 내지비 + Σ후가공** |

★ **검증 포인트(구조)**: ① 모든 부품 comp가 addtn_yn=Y로 Σ ② 제본비는 부수 차원만(×4)·내지비는 부수×페이지(앱 2중 곱·DB는 1면 단가) ③ COMP_PAPER 표지/내지 2회 배선 시 mat_cd 분기로 각자 1행(충돌 없음·§3.3 AD-BK3 가드) ④ 면지 색(sub_prd)은 합산 항 아님(§5 이중계상 가드). **부품 단가 확정값은 Q-BK-COVER(가격표 재대조) 후 golden 보강**(현재는 구조·합산 메커니즘 검증).

## 8. GC-BK8 — 이중수량 페이지 파생 (W6·앱계산 vs DB룩업) [구조 검증]

| 항목 | 값 |
|------|----|
| 상품 | PRD_000069 무선책자(page_rule 24/300/2) |
| 입력 | 부수 qty=10·페이지수=100p(입력 차원·page_rules 24≤100≤300·incr 2 정합) |
| 제본비 | 무선 10부 = unit 1,000 × 10 = 10,000 (부수 차원만) |
| 내지비(구조) | 1면 단가 × 100페이지 × 10부 = 앱 2중 곱(DB는 1면 단가 룩업) |
| 책등 | 100p × 내지두께 = 앱 파생(DB 미저장·표지 면적 영향) |

★ **검증 포인트(W6 돈크리티컬 가드)**: ① 페이지수=입력 차원(siz_cd SKU 베이크 금지·page_rules 제약) ② 제본비는 부수만(×10·페이지 무관) ③ 내지비만 부수×페이지(앱 곱) ④ 책등=앱 파생(DB 단가행/축 신설 0). 두 수량 축 혼동 금지(제본비에 페이지 곱하면 오모델).

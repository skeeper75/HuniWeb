## 과적재·가격형 분리 (HARD)
- 엑셀 공정컬럼은 공정명 외 **옵션값·파라미터·사이즈·안내문 과적재**(마스터 정합 §2.3: 명칭 22 + 비명칭 ~280). 공정명만 추출 후 `proc_cd→t_proc_processes` 대조.
- **DB가 과적재를 흡수하는 방식**(`ref-processes.csv` `prcs_dtl_opt` JSON 실측): 코팅 유광/무광=면 enum(단면/양면), 박=크기 mm, 오시/미싱=줄수 0~3, 형압 양각/음각=크기 mm, 반칼=조각수. 즉 엑셀 `무광코팅(단면)`·`박크기`·`1줄/2줄`·`형압(음각)`은 **공정명이 아닌 공정의 옵션값** → DB 공정 JSON 입력으로 흡수, 직접 대조 제외.
- **가격형 분리**: `*가격표참고`·`0.0`·수치(`10 / 12 / 14`) = 공정 아님, 분리.
- 추출 규칙: `무광코팅`→무광, `유광코팅`→유광, `박(있음)`→박(자식 금유광/동박… 존재 시 충족), `형압(음각)`→음각.

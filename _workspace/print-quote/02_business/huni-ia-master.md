## 추출 후 발견 사항

### A. V1/V2 분류가 xlsx에 명시되어 있는가? — **No (미입력)**

- `우선·순위`(K열), `개발·규모`(L열) 컬럼이 **헤더만 정의되고 데이터는 전부 비어 있다**. 따라서 본 xlsx는 V1/V2 우선순위를 **명시하지 않는다**.
- 범례(R119)에 규모 S/M/L/XL 기준이, R120에 담당 롤 기준이 정의되어 있으나 각 기능 행에는 미할당.
- 상태 범례(R118: O/△/X/★)도 정의만 있고 `일정 반영·상태`(H열) 미입력 → **현행 대비 반영 상태 미표기**.
- **결론:** xlsx는 "현행 기능 인벤토리(As-Is)"이며 우선순위·규모·담당·V1/V2는 **후속 채움 대상**이다. V1/V2 결정의 권위는 여전히 policy-checklist §2.1(추론 67/38)이지만, **그 추론은 본 인벤토리 112건 기준으로 재검증 필요**.

🟡 **DECISION (pq-pm 수집):** xlsx의 우선순위·규모·담당롤·V1/V2 컬럼이 공란이다. 경영진이 이 컬럼을 채워줄 것인지, 아니면 분석팀 추론(policy-checklist)을 그대로 V1/V2로 확정할 것인지 결정 필요.

## 해석 모호 항목

1. **R33~36 영역 라벨 "마이페이지"** — `메인 / 서브메인(랜딩) / LIST(검색결과) / 상품페이지(상품옵션)`가 영역=마이페이지로 입력됨. 내용상 **사이트 전역 페이지(메인·검색·상품)**이며 마이페이지 소속이 아닐 가능성이 높다. xlsx 입력 오류로 추정하나 원문 그대로 보존.
2. **R2 메타 "사용자101+관리자140=241"** vs 실제 입력 112건 — 목표치인지 다른 산정 기준인지 불명. xlsx는 미완성 스냅샷.
3. **G열 "샵바이/오로라"** — 헤더만 있고 데이터 공란. 각 기능을 샵바이/오로라 중 어디에 매핑할지 결정 의도였으나 미완.
4. **No 12 "나의 상품Q&A" vs No 22 "상품Q&A"** — 마이페이지/고객센터에 각각 상품Q&A 존재. 동일 게시판의 진입점 2개인지, 별개 기능인지 모호.
5. **R117 "합계" 행 값 공란** — 집계 미수행.

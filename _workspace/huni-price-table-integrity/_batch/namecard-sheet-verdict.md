## 요약

**명함 시트 = 배선 결함 4건(스티커/도무송과 정반대).** 코팅·프리미엄·화이트 전용 단가행은 권위대로 정확 적재됐으나 **공식(PRF) 생성 + 상품 바인딩이 누락**돼 고아 상태 → 031 견적0·032 저청구(-2000/100매)·040 견적불가. 038 형압은 component 자체 부재(전체 미설계). 정상 6상품(033/034/035/036/037/039)의 §29 UNSCORED는 평가 축미설정 가짜플래그. **교정=순수 배선(공식 신설+바인딩, 단가행 재적재 불필요)이라 저비용·고임팩트**. 실 교정은 인간 승인 후 §18 설계→§7 적재 위임(032 코팅 저청구가 매 주문 발생 중이라 최우선).

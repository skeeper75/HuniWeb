## 1. 엽서 (합산형 / 원자합산)

### 1.1 가격축 (가격을 바꾸는 차원)

| 축 | 값 도메인 | 권위 출처 | 역할 |
|----|----------|----------|------|
| **사이즈** | 73x98 / 98x98 / 100x150 / 135x135 / 95x210 / 110x170 / 148x210 mm … | 가격표 `판걸이수`:B2~B8 (열A=사이즈옵션명) · 마스터 `디지털인쇄`:E3 사이즈(필수) | **판걸이수 결정** → 출력매수 분모 |
| **인쇄 도수/면** | 흑백(1도) / 칼라(CMYK) / 별색(화이트·클리어·핑크·금색·은색) × 단면/양면 | 가격표 `디지털인쇄비`:B2~O3 (열헤더 = 도수×면) | 인쇄비 단가행 열 선택 |
| **종이(자재)** | 백모조/아트/스노우/몽블랑/랑데뷰 … 평량별 | 가격표 `출력소재(IMPORT)`:C열 종이명·I열 가격(국4절) · 마스터 `디지털인쇄`:종이(필수) | 용지비 단가 |
| **코팅(옵션)** | 무광/유광 × 단면/양면 (또는 없음) | 가격표 `코팅`:B2~E3 | 코팅비 |
| **후가공(옵션)** | 오시/미싱/귀돌이/타공/별색 등 | 가격표 `인쇄후가공`·`커팅타공`·`접지옵션` | 후가공비 |
| **제작수량** | 건수(주문수량) | 마스터 `디지털인쇄`:제작수량(필수) 건수/최소/최대/증가 | 룩업 행 + 출력매수 분자 |

### 1.2 단가형/합가형

- **합가형(원자 합산)** — 각 원자(인쇄비·코팅비·용지비·후가공비)를 별도 룩업 후 **합산**. 단일 매트릭스 단가 아님.
- 각 원자 내부: **수량구간 단가형** (수량행 룩업 단가 × 출력매수).

### 1.3 골든 산정식 (verbatim — [공식집:행4~9])

```
판매가 = 인쇄비 + 코팅비 + 용지비 + 후가공비 + 추가상품          [공식집:행4]
(1) 출력매수(=판수) = 주문수량 / 판걸이수                       [공식집:행5]  *크기별 기준(판걸이수 시트)
(2) 인쇄비(4도) = [수량행단가] × 출력매수                       [공식집:행6]  *국4절 또는 3절
(3) 별색인쇄비 = [인쇄비단가] + [출력매수 수량행]×[칼라열]        [공식집:행7]  *3절 별색 없음
(4) 코팅비 = [수량행단가] × [코팅타입열] × 출력매수             [공식집:행8]
(5) 용지비 = [크기별 기준단가] × ([출력매수] + 5장(손지율))     [공식집:행9]
(6) 후가공비 = [제작수량행]                                    [공식집:행10]
```

> **핵심 도메인 규칙(권위):** 인쇄비/코팅비의 **수량행** = 주문수량이 아니라 **출력매수(판수)** 로 룩업. 판수 = ceil(주문수량 / 판걸이수). 판걸이수는 사이즈별 고정([판걸이수] 시트). 용지비 손지율 = +5장(출력매수 기준).

### 1.4 골든 단가 출처 (셀 좌표)

- **판걸이수**(사이즈→판수 분모): `판걸이수`:F열(판걸이) — 예: 73x98mm 엽서=18 [`판걸이수`:F2], 98x98=12 [F3], 100x150=8 [F4], 135x135=6 [F5].
- **인쇄비 단가**(출력매수행 × 도수면열): `디지털인쇄비` 국4절 블록 행4~56(수량행=출력매수), 열B~O(흑백/칼라/별색 × 단/양면). 예: 칼라CMYK 단면 출력매수1행 = 4000 [`디지털인쇄비`:D4], 출력매수2 = 2800 [D5], 출력매수3 = 2500 [D6].
- **코팅비 단가**(출력매수행 × 코팅타입열): `코팅` 국4절 블록 행4~26, 열B~E(무광단/무광양/유광단/유광양). 예: 무광단면 출력매수1=2000 [`코팅`:B4].
- **용지비 단가**(종이별 국4절 1장): `출력소재(IMPORT)`:I열 `가격(국4절)`. 예: 백색모조지 220g = 70.64 [`출력소재(IMPORT)`:I6], 백색모조지 100g = 30.73 [I4].
- **별색인쇄비**: `디지털인쇄비`:F~O열(화이트/클리어/핑크/금색/은색 × 단/양). 예: 별색화이트 양면 출력매수1행 = 6000 [`디지털인쇄비`:G4].

## 2. 실사 / 현수막 (면적매트릭스 — 가로 × 세로)

### 2.1 가격축

| 축 | 값 도메인 | 권위 출처 | 역할 |
|----|----------|----------|------|
| **소재(자재)** | 아트프린트(인화지)/아트페이퍼(매트지)/방수PET/접착방수PVC/접착투명PVC/아트패브릭/린넨/캔버스/레더/타이벡/메쉬/일반·메쉬현수막 | `포스터/사인` 각 블록 제목행 (예:행1·18·35·52·69·86·103·120·137·154·171) | **매트릭스 블록 선택** |

| **세로(height)** | 600~3000 mm (블록별 행헤더 200mm 간격) | `포스터/사인` 각 블록 행헤더 열A (예 행3~15) | 매트릭스 **행** |

> **축 방향 확정(권위):** 매트릭스 코너 셀 = "가로 / 세로". **열 = 가로(width), 행 = 세로(height)**. [공식집:행101] "판매가 = [세로mm행][가로mm열]" 정합. dbmap 신안 `siz_width`(가로=열)·`siz_height`(세로=행)와 일치.

### 2.2 단가형/합가형

- **면적매트릭스 단가형** — [세로행][가로열] 한 셀 = **완성 단가(코팅포함가)**. 합산 아님(소재 블록에 "코팅포함가" 명시: 행1·35·52 등). 후가공(린넨 등)은 별도(authority-gaps 참조).
- **수량 무관** — 매트릭스에 수량축 없음(개당 단가). 주문 수량은 [공식집:행104 고정가형 폼보드 등]과 달리 실사는 면적매트릭스 단일.

### 2.3 off-grid 처리 (권위)

- 매트릭스 행/열 헤더값 = 구간 **상한**. 주문 가로/세로 ≤ 임계 중 **가장 작은 임계 구간**(다음 큰 구간 = ceiling). 예: 1000x1000 코너에 별도 명시 "1000x1000 : 20000" [`포스터/사인`:F5 등] = off-grid 보정 사례.
- 라이브 엔진(`pricing.py` TIER)이 이 ceiling을 내장(dbmap [[dbmap-area-matrix-wh-dimension]]·`poster-sign-component-redesign.md` §0). **권위 = 매트릭스 셀값 그대로**(엔진이 구간 매칭).

## 3. 아크릴 (면적매트릭스 + 두께축 + 수량할인 + 후가공)

### 3.1 가격축

| 축 | 값 도메인 | 권위 출처 | 역할 |
|----|----------|----------|------|
| **소재/두께** | 투명3T / 투명1.5T / 미러3T | `아크릴`:A1(투명3T)·A21(1.5T)·A35(미러3T) | 매트릭스 블록 선택 (두께축과 직교) |
| **가로(width)** | 20~200mm (3T) / 20~100mm (1.5T·미러) | `아크릴`:B2~O2 (3T), B22~J22 (1.5T·미러) | 매트릭스 **열** |
| **세로(height)** | 20~200mm (3T) / 20~100mm | `아크릴`:A3~A16 (3T), A23~A31 (1.5T), A37~A45 (미러) | 매트릭스 **행** |
| **수량(구간할인)** | 1~49/50~99/100~299/300~499/500~999/1000~10000 | `아크릴`:A51~A56 수량구간 / B51~B56 할인율 | 인쇄가공비에만 할인 적용 |
| **후가공(옵션)** | 키링(고리)/뱃지/마그넷/집게/명찰/스마트톡/지비츠/볼펜/머리끈/카라비너 | `아크릴`:A61~B86 (옵션명·단가) | 제작수량당 가산 |

> **축 방향(권위):** `아크릴` 코너 셀 = "가로 / 세로" → **열=가로, 행=세로**. [공식집:행108] "소재별 면적[세로mm행][가로mm열]" 정합. dbmap 권고: **siz_nm WxH 파싱(W=앞·H=뒤)**, work_width/height 금지(`acrylic-wh-isomorph-design.md` §2 — 돈크리티컬).

### 3.2 단가형/합가형

- **면적매트릭스 단가형** (인쇄가공비) — [세로행][가로열] 셀 = 개당 인쇄가공비.
- **수량할인 = 인쇄가공비에만** 적용([공식집:행109]). 후가공은 할인 미적용.
- **두께 직교:** 1.5T = 3T × 0.8 (verbatim 검증: 3T 30×30=3100, 1.5T 30×30=2480 = 3100×0.8). 미러는 별 매트릭스(직교 아님).
- **최종 = 합가형(인쇄가공비 × 할인 + 후가공)** [공식집:행107].

### 3.3 골든 산정식 (verbatim — [공식집:행107~110])

```
판매가 = 인쇄가공비 × 수량별구간할인 + 후가공                  [공식집:행107]
(1) 인쇄가공비 = 소재별 면적[세로mm행][가로mm열]              [공식집:행108]  아크릴+인쇄+레이저커팅
(2) 수량별할인 = 인쇄가공비에만 적용                          [공식집:행109]
(3) 후가공 = 제작수량당 부과                                 [공식집:행110]
```

> **해석(권위):** 수량별구간할인 = (1 − 할인율). 예 수량 100개 → 할인율 0.2 → 인쇄가공비 × (1−0.2) × ... 단, "인쇄가공비"가 개당인지 총액(×수량)인지 = **컨펌 필요**(authority-gaps Q-ACR-ENGINE). 매트릭스 셀=개당이므로 통상 `개당 × 수량 × (1−할인율) + 후가공×수량`.

### 3.4 골든 단가 출처 (셀 좌표)

- **투명3T 매트릭스:** `아크릴` 행3~16 × 열B~O. 골든: 세로30×가로30 = 3100 [`아크릴`:C4]; 세로30×가로40 = 3400 [D4]; 세로100×가로100 = 12700 [J11].

- **미러3T 매트릭스:** `아크릴` 행37~45 × 열B~J. 골든: 세로30×가로30 = 6200 [`아크릴`:C38].
- **수량할인율:** `아크릴`:A51~B56. 1~49=0.0[B51], 50~99=0.1[B52], 100~299=0.2[B53], 300~499=0.3[B54], 500~999=0.4[B55], 1000~10000=0.5[B56].

# 가격 계산공식 유형 — 후니 권위 정본 (price formula types)

> **출처(권위):** 후니프린팅(고객사) 제공 가격계산공식 — 사용자 전달 2026-06-06. **이 문서가 가격 공식유형의 최상위 권위**이며, round-2 추출이 이와 어긋난 부분은 정정 대상이다.
> **용도:** POSTER/ACRYL 가격 siz 차단(917행) 처리의 기준. round-2가 28개 상품을 일괄 `PRF_POSTER_FIXED`(면적-좌표)로 모델링했으나, 후니 권위로는 **면적매트릭스 13 + 고정가형 15**로 갈린다.

---

## 1. 공식 유형 (후니 정본)

### ① 면적매트릭스형 — 실사·현수막
```
판매가 = [세로mm 행][가로mm 열]   (2D 매트릭스 직접 조회)
```

### ② 면적매트릭스형 — 아크릴
```
판매가 = 인쇄가공비 × 수량별구간할인 + 후가공
 (1) 인쇄가공비 = 소재별 면적 [세로mm 행][가로mm 열]
 (2) 수량별할인 = 인쇄가공비에만 적용
 (3) 후가공 = 제작수량당 부과
```

### ③ 고정가형 — 수량×옵션 (폼보드·포맥스·프레임리스액자·레더아트액자·캔버스행잉포스터·린넨우드봉족자·족자포스터·PET배너·메쉬배너·시트커팅·아크릴스티커·미니스탠딩보드·미니배너)
```
판매가 = [수량 행][옵션 열]   (면적 무관)
```

### ④ 고정가형 — 아크릴코롯토·아크릴카라비너
```
판매가 = 인쇄가공비 × 수량별구간할인
```

### off-grid 규칙 (HARD, 후니 결정)
- 아크릴(면적매트릭스)에서 가로×세로가 표에 **정확히 없으면 → 한 단계 큰 크기의 가격**으로 계산(ceiling/step-up).
- "효율적이지 않을 수 있으나 고객사 후니 결정사항." → **가격계산 런타임(위젯/엔진) 로직**으로 구현. DB엔 매트릭스만 저장.

## 3. 처리 계획 (round-2 정정)

| 그룹 | 처리 | 산출 |
|------|------|------|
| 면적매트릭스 13(실사/현수막) + 아크릴 | [세로][가로] 좌표 사이즈 등록 요청서 + 매트릭스 component_prices 적재본 | 좌표 siz 등록(후니) 후 적재 |
| 고정가형 15 | 현 좌표 placeholder **폐기** → 가격표에서 **[수량][옵션] 재추출**(excel-analyst) → 고정가 공식유형으로 재매핑 | 신규 PRF_*_FIXED(수량×옵션) + product_prices/component |
| off-grid ceiling | 가격계산 런타임 규칙으로 문서화(위젯/엔진) | DB 미저장 |

> **상태**: 권위 모델 박제 완료. 분류 ①/③ 사용자 확인 후 — 면적 13+아크릴 좌표 적재본 + 고정가 15 수량×옵션 재추출 착수. (사용자 확인 대기)

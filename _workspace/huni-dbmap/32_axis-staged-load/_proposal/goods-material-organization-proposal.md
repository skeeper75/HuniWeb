# 굿즈/파우치 자재정보 정리 제안서 (round-22 · 전수조사 + RP·WP 권위 통합)

> **작성** 2026-06-16 · MoAI 통합(전수조사 `goods-pouch-full-survey.md` + 경쟁사 권위 `goods-material-competitor-research.md`).
> **사용자 질문:** "굿즈/파우치 자재정보를 어떻게 정리하면 좋을까? 굿즈는 틴거울·머그컵처럼 소재와 다른 형태(완제품)도 있다."
> **방법:** 굿즈파우치 시트 103상품 전수 조사 + 레드프린팅·와우프레스 권위 리서치 → 정립 제안.

---

## 0. 결론 한 줄

**자료 모델은 바꿀 필요가 없다.** 후니 자재 모델(`t_mat_materials` = MAT_TYPE 분류 + usage_cd + 물성 + self-ref)은 **경쟁사 두 곳(RP·WP)이 독립적으로 검증한 정답 구조**다. 완제품 굿즈(거울·머그컵)도 **"자재"로 두는 것이 맞다**(별도 품목/SKU 모델 신설 금지). 필요한 일은 **모델 신설이 아니라 "정리"** — ① 본체 소재를 자재로 채우고 ② 옵션값 오염을 걷어내고 ③ 근본적으로 상품마스터 시트에 "본체 소재" 칸을 만드는 것.

## 1. 전수 조사 결과 (103상품)

| 분류 | 상품수 | 본체소재 라이브 연결 | 본체 물성 종류 |
|------|:--:|------|------|
| **완제품/기성품류** | 32 | **0** (전 물성 자재행 라이브 부재) | 20종: 유리/거울·세라믹·트라이탄·우드·규조토·코르크·벨벳·양철·금속·PC/TPU·PBT/ABS·면니트·극세사·러버·직물·PP·고무·종이 |
| **소재가공류** | 61 | **41 연결됨**(레더·캔버스·린넨·메쉬·GPM 적용) | 7종: 가죽·캔버스·린넨·메쉬(자재행 존재) / 타이벡(존재·13상품 미연결)·광목·연질PVC(자재행 부재) |
| **모호** | 10 | 0 | 7종: PVC/직물·자석·아크릴/보드·실리콘·직물끈·아크릴+PVC |

**시트 구조의 근본 문제 4가지:**
1. **"본체 소재" 컬럼이 시트에 아예 없다** — 소재가 상품명 텍스트("레더코스터")에만 박힘 → load_master(무변환)가 자재로 넣을 입력이 없음.
2. **`상품(옵션)` 한 칸에 4+개 의미축 혼재** — 형상/사이즈/색/인쇄면/구수가 무차별로 `.09 파우치` 자재행으로 적재.
3. **`.09 파우치` 자재유형이 굿즈 옵션값 만물상**(74행 전부 옵션값·본체소재 0행).
4. **완제품·소재가공이 한 시트에 무구분 혼재**(생산형태·물성 구분 칼럼 부재).

## 2. 경쟁사 권위 (RP·WP 독립 수렴)

| 질문 | 레드프린팅 | 와우프레스 | 수렴 |
|------|-----------|-----------|------|
| 완제품 본체(거울·머그) | 자재코드 `MTRL_CD`(SXSML001 틴거울·SXMIRW06 보틀) | 재질 슬롯 `paperinfo.papergroup`("스텐"·"캔버스")·`pgram=null`로 비종이 신호 | **본체=자재** |
| 본체색 | `PCS_DTL_NME` 합성(틴거울 핑크) | `papername` 합성(아이보리/블랙) | **재질행 합성**(과분할 금지) |
| 인쇄방식 | `DOSU_COD`(SID_X 무인쇄·SID_S 단면) | 도수 슬롯 분리 | **재질과 분리** |
| 소재가공(파우치) | 원단 `MTRL_CD`+가공 PCS(파우치가공·지퍼·재단) | 캔버스 paperinfo+가공 | **원단자재+가공공정** |
| 굿즈 전용 모델? | 없음(인쇄상품과 동일 프로시저) | 없음(인쇄상품 5슬롯 재사용) | **전용 모델 신설 안 함** |

> **두 경쟁사가 독립적으로 동일 결론에 도달** = 강한 권위 신호. 후니 자재 모델(MAT_TYPE 명시 분류 + usage_cd + 물성)은 이를 **흡수·일부 능가**(WP는 본체물성을 평량 null로만 신호하지만 후니는 MAT_TYPE로 명시 분류). 후니 기존 `huni-goods-option-mapping`(파우치 MAT_000062~067 본체색 합성)이 이미 이 정답 패턴.

## 3. 정리 5원칙 (경쟁사 검증 + 후니 도메인)

1. **굿즈 본체 = 자재**(`t_mat_materials`). 완제품(거울·머그컵)도 별도 품목/SKU가 아니라 자재로. 인쇄상품과 같은 모델.
2. **본체색 = 재질행 합성**(과분할 금지). 화이트/블랙 2~3종은 자재 variant, 다수 선택색은 옵션.
3. **인쇄방식(UV/각인/전사/승화) = 자재와 분리**된 도수/공정 축.
4. **소재가공 = 원단/가죽 자재 + 가공 공정**(봉제·지퍼·재단 = 공정 축).
5. **옵션값(형상/사이즈/구수/인쇄면) = `.09`에서 분리**해 siz/bundle/option/print_side로(B-3). `.09 파우치` 만물상 해체.

## 4. 실행 로드맵 (전수 103상품)

| 단계 | 대상 | 작업 | 상태 |
|------|------|------|------|
| S1 | 소재가공 41(레더/캔버스/린넨/메쉬) | 본체 소재 자재 link | ✅ **라이브 COMMIT 완료** |
| S2 | 소재가공 잔여(타이벡13·광목1·연질PVC) | 타이벡=하드/소프트 컨펌 후 연결·광목/연질PVC=신규 자재 mint+연결 | 컨펌·mint |
| S3 | 완제품 32 (물성 20종) | 본체 물성 자재 **신규 mint**(유리/세라믹/양철/우드/규조토/코르크/벨벳…) + 연결 | mint·컨펌 |
| S4 | 모호 10 | 본체 물성 도메인 컨펌 후 분류 | 컨펌 |
| S5 | 폰케이스 5 | 라이브 미등록 → 상품 등록 선행 | 상품등록 |
| S6 | `.09` 오염 행(형상/색/구수) | 본체 link 선적재(S1~S3) 후 옵션/siz/bundle 축이동 + 오염행 use_yn='N'(B-3·GPM-4) | 본체 후 |

> **★순서 [HARD]:** 본체 소재 link 선적재(S1~S3) → 오염 link 제거(S6). 역순이면 본체 BOM 영구 공백.

## 5. ★근본 해결 (실무진/개발자 — 회귀 방지)

라이브 직접 교정(경로 X)은 개발자가 v03 재적재하면 소멸한다. **근본은 입력 양식 교정:**

1. **[최우선] 상품마스터 굿즈파우치 시트에 "본체 소재" 컬럼을 신설**한다. 그래야 load_master가 자재로 적재하고, 재적재에도 유지되며, 완제품 20종·신규 mint도 체계적으로 채워진다. **이것이 진짜 정리의 핵심** — 현재는 소재가 상품명에만 있어 구조적으로 자재가 안 들어간다.
2. `상품(옵션)` 한 칸의 혼재(형상/사이즈/색/구수)를 **의미축별 컬럼으로 분리**(또는 옵션 레이어로). `.09 파우치` 만물상 해체.
3. 완제품 본체 물성용 **자재 코드 신규 mint**(GP-Q2: 기존 `.12 굿즈` mat_typ 활용 vs 신규 mat_typ — 권고는 물성별 `.05 원단`·신규 `재질` 분류 확장).

## 6. 컨펌 큐 (도메인 — 인간/실무진 승인)

- **GP-Q1** 완제품 32 본체물성을 자재로 등록하는 정책 확정(경쟁사 권위는 YES).
- **GP-Q2** 완제품 물성 자재의 mat_typ 범위(`.12 굿즈` vs 신규 분류·우드/규조토/세라믹/양철…).
- **GP-Q3** 모호 10(자석·실리콘·아크릴+PVC) 본체 물성.
- **GP-Q4** 고아 색합성행 MAT_000061~067 처리(살려서 본체로 vs 폐기).
- **GP-Q5** 타이벡 13 = 하드(MAT_000187) vs 소프트(188).
- **GP-Q6** 머그 용량(비치수 siz)·만년스탬프 잉크색(도수 vs 별색)·말랑PVC 귀속.

## 7. 요약 (사용자 질문 답변)

- **"어떻게 정리하면 좋을까"** → 모델은 그대로(후니 자재 모델 = RP·WP 검증된 정답). 완제품(거울·머그컵)도 **자재로** 두면 된다.
- **"소재와 다른 형태(완제품)"** → 별도 품목 모델을 만들지 말고, 본체 물성(유리·세라믹·양철)을 **자재 코드로 신규 등록**(MAT_TYPE 분류) + 본체색 합성 + 인쇄방식 분리. 경쟁사도 정확히 이렇게 한다.
- **진짜 해야 할 일** → ① 라이브: 본체 소재 자재 채우기(41 완료·완제품 mint 진행) + 오염 걷어내기 ② **근본: 상품마스터 시트에 "본체 소재" 컬럼 신설**(실무진) — 이게 빠져서 지금 자재가 안 들어간 것.

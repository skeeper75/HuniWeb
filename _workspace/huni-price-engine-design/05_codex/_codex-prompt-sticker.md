# 독립 가격엔진 설계 재판정 — 후니프린팅 스티커 상품군

당신은 인쇄 가격계산엔진 설계의 **독립 2차 검증자(gpt-5.5)** 다. 아래는 후니프린팅 "스티커" 상품군의
가격공식+가격구성요소+단가행 그릇 설계와, 권위 가격표 분해본 실측, 라이브 가격엔진 코드(pricing.py)다.
다른 검증자의 판정은 주어지지 않는다 — **당신의 독립 판정만** 답하라. 추측은 추측이라 명시하라.

## 엔진 동작 사실 (pricing.py·라이브 코드 verbatim)

- 단가행 매칭: 비수량 차원 `NON_QTY_DIMS = (siz_cd, plt_siz_cd, print_opt_cd, mat_cd, proc_cd, opt_cd, coat_side_cnt, bdl_qty)`은
  **정확매칭**(행 차원이 NULL이면 와일드카드). 행 차원이 선택값과 다르면 그 행 제외.
- 수량 차원 `min_qty`는 '이상' 하한 TIER(주문수량 ≥ min_qty 중 최대 임계 행 선택). siz_width/siz_height는 '이하' 상한.
- 소계 계산 `component_subtotal(prc_typ, unit_price, tier_min_qty, qty)`:
  - **단가형 PRICE_TYPE.01** → `unit_price × qty`
  - **합가형 PRICE_TYPE.02** → `(unit_price ÷ tier_min_qty) × qty`. **tier_min_qty가 0/NULL이면 ValueError(견적 불가)**.
- 공식 = formula_components에 배선된 comp들의 합산(addtn_yn). 한 comp 안에서 동시 매칭 단가행이 2개 이상이면 ERR_AMBIGUOUS.
  서로 다른 comp는 각자 매칭되어 합산된다(차원 NULL=와일드카드면 둘 다 통과 → silent 이중합산 가능).
- 면적매트릭스(siz_width/siz_height)는 off-grid를 ceiling으로 흡수하지만, **이산 siz_cd 정확매칭은 미매칭 시 no_match**(가격 0/오류·ceiling 없음).

## 당신이 독립 판정할 질문 (열린 질문·우리 결론 없음)

   실측을 근거로 평가하라. 별도 경로가 필요한가?

   A4 견적 결과는 어떻게 되는가? A6(196)은?
4. 사이즈축별 교정 경로(A5=059 재바인딩 / A4=반칼 전용 siz + import xlsx verbatim 적재 / A6=바인딩 제거)가

각 질문에 ① 독립 판정(성립/불성립/조건부) ② 근거(엔진 동작·권위 가격표·라이브 분포) ③ 확신도 ④ 발견한 추가 위험을
간결히 답하라. 마지막에 이 스티커 보정 설계 전체에 대한 GO / CONDITIONAL / NO-GO 한 줄 결론을 내려라.

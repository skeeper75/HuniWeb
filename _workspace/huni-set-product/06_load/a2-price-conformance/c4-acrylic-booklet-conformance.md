## 2. 축① 단가행 값 verbatim (GO)

### 2-1. 아크릴 본체 (COMP_ACRYL_CLEAR3T, 165행, 면적매트릭스 합가형 PRICE_TYPE.02)
- `use_dims=[siz_width, siz_height, mat_cd]`. mat_cd=두께 차원: **MAT_000043=3mm(113행)·MAT_000042=1.5mm(52행)**.
- 권위 가격표 「아크릴」시트 B01(투명3T, 가로×세로 14×14)·B02(투명1.5T 10×10). 대칭 매트릭스.
- verbatim 스폿체크(라이브 ↔ 권위 일치):
  - 3T 20×20=2500 / 20×30=2700 / 30×30=3100 → 권위 B01 동일 ✓
  - 1.5T 20×20=2000 / 20×30=2160 / 30×30=2480 → 권위 B02 동일 ✓
- 차원·값 정합. (행수 113/52는 대칭행렬 부분저장 — w≤h 93+38, w>h 20+14. 매칭은 양방향 키 존재로 OK이나 §3-D 참조.)

### 2-2. 제본비 (COMP_BIND_*, PRICE_TYPE.01 단가형)
권위 「제본」 B01 4종×수량밴드를 라이브 4 comp가 verbatim 보유(min_qty 1/4/10/30/50/70/100/1000):

| 4 | 2000/2000 | 2000/2000 | 3000/3000 | 5000/5000 |

### 2-3. 접지비 (COMP_FOLD_CARD_2H, 48행, 단가형 use_dims=[min_qty])

- 6000(일반)·8500(투명) 정확한 verbatim 원천(상품마스터 가격포함 시트 원본 셀)은 본 추출본에 미수록 → "도메인·모델 sanity OK, 셀 단위 verbatim은 원본 확인 권장"으로 표기(날조 회피). 고정세트가 모델 자체는 정합.

## 3. 축②·③ 결함 (NO-GO 원천)

### A. [HIGH] PRF_BIND_SUM 다종-1배선 — 무선/PUR/트윈링 책자 제본비 오류
- 4종 책자(068 중철·069 무선·070 PUR·071 트윈링)가 PRF_BIND_SUM 공유. 그러나 공식은 **COMP_BIND_JUNGCHEOL(중철, PROC_000018만) 단 1개 배선**.
- 상품마스터: 각 책자는 제본(필수)이 **고정 단일값**(중철책자=중철제본, 무선책자=무선제본…). 즉 제본종류=상품 정체이지 주문옵션이 아님.
- 엔진 영향: COMP_BIND_JUNGCHEOL 단가행은 PROC_000018(중철)만 보유. 무선/PUR/트윈링 상품이 자기 proc(PROC_000019/20/21)로 매칭 시도하면 `_row_matches` 실패 → **제본비 0원 silent skip(과소청구)**. proc 필터를 안 거치면 역으로 **중철 단가로 오청구**(PUR은 중철의 ~5배 비싸야 함 → 막대한 과소). 어느 경로든 돈크리티컬.
- 정답 배선: 무선책자→COMP_BIND_MUSEON, PUR→COMP_BIND_PUR, 트윈링→COMP_BIND_TWINRING (단가행 이미 적재됨). 상품별 공식 분리(PRF_BIND_MUSEON 등) 또는 상품별 comp 택일 필요.
- ★셋트 하이브리드 직결(§23): 책자=구성원공식(내지/표지)+셋트 제본공식. 본 배선 단절은 셋트 제본 레이어가 중철로 고정돼 무선/PUR/트윈링 셋트 가격이 틀어지는 근본.

### B. [HIGH] PRF_FOLD_SUM 오배선 — 리플렛에 카드 2단 접지비
- 접지리플렛(PRD_000048)은 **리플렛**인데 PRF_FOLD_SUM이 **COMP_FOLD_CARD_2H(카드 2단)** 배선.
- 리플렛 접지 comp(COMP_FOLD_LEAF_3FOLD/4ACC/4GATE/HALF, 각 48행)가 별도 존재하나 미배선.
- 게다가 PRD_000048은 옵션그룹 0개 → 접지종류 선택 불가. 단일 접지비만 강제 → 3단/4단/반접지 견적 불가, 또는 카드2단 단가 오청구.

### C. [HIGH] 아크릴 23상품 미바인딩 — 견적 불가
- 아크릴 계열 26상품 중 가격공식 바인딩 **3개뿐**(142·143 스티커=POSTER, 146 키링=CLR_ACRYL).
- 미바인딩 23: 마그넷·뱃지·집게·스마트톡·명찰·명찰(골드실버)·머리끈·볼펜·지비츠·네임택·포카키링·코스터·자유형스탠드·판아크릴·포카스탠드·미니파츠·**코롯토(164)·포카코롯토(165)·카라비너·입체코롯토(168)·입체블럭·쉐이커(170)·쉐이커코롯토(226)** → 전부 **견적 불가**(공식 미설계/미바인딩).
- root cause = 공식 미설계 또는 바인딩 누락. 메모리 일치: "아크릴명찰=가격 미구성".

### D. [HIGH] COMP_ACRYL_COROTTO 고아 — 공식·단가 완비, 바인딩 0
- PRF_COROTTO_ACRYL(use_yn=Y) + COMP_ACRYL_COROTTO(21행, 6×6 면적매트릭스) **완비**.
- 그러나 PRF_COROTTO_ACRYL **바인딩 0상품**. 코롯토 상품 4개(164·165·168·226)는 §C대로 전부 미바인딩.
- 즉 그릇은 다 만들어졌는데 상품-공식 연결만 빠짐 → 1줄 바인딩으로 코롯토 견적 가능해지는 "마지막 한 칸" 누락. fix_route 명확.

### E. [HIGH] COMP_ACRYL_MIRROR3T 고아 — 52행 적재·미배선·미바인딩
- 미러아크릴3T 단가(52행, 권위 B03 `=B3*2` 양면) 적재됐으나 **어느 공식에도 미배선·어느 상품에도 미바인딩**.
- 미러아크릴을 쓰는 상품(거울계열)이 있다면 견적 불가. 데이터는 죽은 채 방치.

### F. [HIGH] PRD_000070 PUR책자 = 스텁(Test 옵션만)
- PUR책자(070) 옵션그룹이 **OPT-000011 "Test" 단 1개**(사이즈·내지·표지·제본 전부 부재).
- 정상 책자(069 무선)는 9개 옵션그룹 보유. 070은 미완성 상품 → 견적 자체 불가(옵션 선택 불가). §A 배선결함과 중복 견적불가.

### MED/LOW
- [MED] PRF_CLR_ACRYL 후가공 미배선: 키링은 본체(CLEAR3T)만 배선, 키링고리(COMP_ACRYL_KEYRING 등) addtn 미배선 가능성 — 키링 부속비 누락 의심(별도 템플릿 경로 확인 필요).
- [MED] COMP_BIND_TWINRING 단가행 32행(4 proc 중복 적재): PROC_000018~021 4벌 모두 보유. 트윈링 본래 1벌이면 잉여 3벌(매칭 오선택 위험). 값은 각 proc=해당 제본단가로 정확하나 의미상 중복.
- [MED] 아크릴 165행 대칭행렬 부분저장(w>h 34행 존재): 완전 대칭이 아니라 일부만 양방향 — 특정 w>h 조합 매칭 누락 가능. 스폿 무결성 OK이나 전수 대각 검증 권장.
- [MED] 포토카드 세트 verbatim 원천 미확인: 6000/8500 셀 단위 권위 추출본 미수록(모델만 정합).
- [LOW] PRF_PHOTOCARD_FIXED(use_yn=N) silent-sum 폐기 = 260623 V3 교정의 **정상 결과**(결함 아님, 정합 근거로 기록).
- [LOW] 코팅 COMP_COAT_MATTE(92행)는 C4 배선 아님 — PRF_DGP_A/D/E(디지털인쇄 C1)에 배선. 본 클러스터 책자 표지코팅과는 별개 사슬(책자 표지코팅 가격 미배선 가능성 → §23 추후).

## 4. 종합

- **단가 데이터(축①)는 건강** — 아크릴·제본 4종·접지·포토카드 모두 권위 verbatim, 1원 오차 0. 적재 자체는 충실.
- **사슬(축②③)이 광범위 단절** — 제본/접지 다종-1배선(오청구·과소청구), 아크릴 23상품·코롯토·미러 미바인딩(견적불가), PUR책자 스텁. "데이터는 있는데 연결이 없다"가 C4의 본질.
- 책자(068~071)는 §23 셋트 하이브리드 모델과 직결: 제본 배선결함이 셋트 제본 레이어 오류의 근본 → 셋트 가격작업 **선결조건**.
- 라이브 직접 적재·교정 없음. 교정은 인간 승인 후 dbmap/§23 트랙 위임.

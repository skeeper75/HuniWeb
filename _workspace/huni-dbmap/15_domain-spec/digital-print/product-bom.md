## 1. 엽서 / 프리미엄엽서 (MES 001-0001, prd_typ 디자인)

- **인쇄방식:** 디지털. 사이즈 13종(73x98~), 가장 옵션 풍부한 대표 상품.
- **자재(BOM):** 종이 11종 — `*별도설정`(공통풀) · 아트250 · 아트300 · 투명PET 260g · 반투명PET 260g · 큐리어스스킨(블랙/레드/바이올렛) 270g 등. usage=본체. ✅
- **공정(체인):** `디지털출력 → 별색(화이트/클리어/핑크/금/은, 단·양면) → 코팅(무광/유광·단/양면) → 재단 → 커팅(자유형) → 모서리(직각/둥근=귀돌이) → 오시(0~3줄) → 미싱(0~3줄) → 가변데이터(텍스트/이미지 0~3개) → 포장`. ✅
- **추가상품(addon):** 엽서봉투(사이즈선택) · OPP비접착/접착봉투 50장 · 카드봉투(화이트/블랙) · 트레싱지봉투 → `t_prd_product_addons`(tmpl_cd) + `t_prd_templates`. 🟡
- **variant:** 투명/반투명PET = 색상·투명도 variant → material 분기. 화이트 별색은 투명PET와 캐스케이드(투명 위 화이트 베이스, 07_domain §3-2). ✅
- **확정도:** ✅ (별색 핑크/금/은만 🟡)

## 2. 포토카드 / 포토카드 (MES 001-0012)

- **인쇄방식:** 디지털. 소형(57x88 등).
- **자재:** 종이 3종 — 아트300 · 투명PET 260g · 반투명PET 260g. ✅
- **공정:** `디지털출력 → 화이트 별색(단면) → 코팅(무광/유광 양면) → 재단 → 모서리(직각/둥근) → 포장`. ✅
- **variant:** 투명/반투명 PET = material. 화이트 별색 = 투명 베이스. ✅
- **확정도:** ✅

## 3. 접지카드 / 2단접지카드 (MES 001-0014)

- **인쇄방식:** 디지털. 접지물(202x152 등).
- **자재:** 종이 — `*별도설정` · 몽블랑130g. usage=본체. ✅
- **공정:** `디지털출력 → 재단 → 접지(2단/3단 가로·세로 / 6단오시접지 / 6단미싱접지) → 가변데이터 → 박(있/없)+박칼라(금유광·은유광·동박·먹유광·청박·적박·홀로그램·트윙클) → 포장`. ✅
- **공정 의존:** 6단오시접지 = **오시→접지 복합 leaf**(PROC_000073, process-recipe-tree §2-3). 접지 `★사이즈선택`=접지 시 사이즈 종속 캐스케이드. ✅
- **확정도:** ✅ (박칼라 🟡)

## 4. 명함 / 프리미엄명함 (MES 003-0001)

- **인쇄방식:** 디지털. 표준 명함(90x50 등). 가장 자재 풍부(18종).
- **자재:** 종이 18종 — `*별도설정` · 아트250/300 · 백모조220 · 스노우250/300 · 스타드림·큐리어스 등. usage=본체. ✅
- **공정:** `디지털출력 → 별색(화이트/클리어, 단·양면) → 코팅(무광/유광·단/양면) → 재단 → 커팅(한쪽라운딩·나뭇잎·큰라운딩·클래식·자유형·사각라운딩 등 8종) → 모서리(직각/둥근) → 가변데이터 → 박(있/없)+형압(음각/양각)+박칼라(9종) → 포장`. ✅
- **공정 특이:** 형압(음각/양각)이 박과 함께 = `t_proc_processes`(형압) + prcs_dtl_opt(음각/양각). ✅
- **확정도:** ✅ (커팅 형상·박칼라 🟡)

## 5. 상품권 / 스탠다드 쿠폰·상품권 (MES 003-0013)

- **인쇄방식:** 디지털.
- **자재:** 종이 12종 — 백색모조지100g · 아트지150g · 스노우지150g · 몽블랑130/190 · 스타드림240 등. ✅
- **공정:** `디지털출력 → 재단 → 오시(0~3줄) → 미싱(0~3줄=절취선) → 가변데이터(넘버링) → 박(있/없)+박칼라 → 포장`. ✅
- **공정 도메인:** 미싱=절취선(쿠폰 떼기), 가변텍스트=일련번호 넘버링 = 쿠폰/상품권 핵심 공정. ✅
- **확정도:** ✅

## BOM 횡단 — 자재·공정 인벤토리 (디지털인쇄 7상품 합집합)

**자재(종이):** 아트지(250/300) · 백모조 · 스노우(250/300) · 몽블랑(130/190/240) · 큐리어스스킨(색상별) · 투명/반투명PET · 스타드림 · 백색모조지 · `*별도설정`(공통풀). → `t_mat_materials` MAT_TYPE=종이. 평량이 자재 식별자 일부.

**공정(`t_proc_processes`):**
| 공정 | 상품 | prcs_dtl_opt param |
|------|------|---------------------|
| 별색(화이트/클리어/핑크/금/은) | 엽서·포토카드·명함·접지카드·상품권 | 적용면(단/양면) |
| 코팅(무광/유광) | 엽서·포토카드·명함·전단지 | 광택·면, 조건(180g↑) |
| 커팅(완칼/도무송) | 엽서·명함·배경지 | 형상 enum(25종+) |
| 접지 | 접지카드·배경지·전단지 | 단수·방향·6단오시/미싱 복합 |
| 모서리(귀돌이) | 엽서·포토카드·명함 | 직각/둥근 |
| 오시 | 엽서·상품권 | 줄수 0~3 |
| 미싱 | 엽서·상품권 | 줄수 0~3(절취) |
| 가변데이터 | 대부분 | 개수·텍스트/이미지 |
| 박 | 접지카드·명함·상품권 | 박칼라·크기(면적→등급) |
| 형압 | 명함 | 음각/양각 |

**추가상품(template):** 봉투류(엽서봉투·OPP·카드봉투·트레싱지·사이즈선택형) → `t_prd_product_addons`(tmpl_cd) + `t_prd_templates`.

**미저장(앱 런타임):** 판수(판걸이수, C6) · 박 크기→등급(C36) — 가격엔진 계산, DB는 룩업만(메모리 `compute-in-app-db-stores-lookup`).
